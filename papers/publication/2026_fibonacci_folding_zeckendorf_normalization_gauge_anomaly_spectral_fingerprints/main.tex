\begin{abstract}
We study finite-window Zeckendorf normalization for binary words in Fibonacci weights. Given a word of length \(m\), we evaluate its Fibonacci value, replace it by the Zeckendorf expansion, and retain only the original window. This defines an idempotent map onto the length-\(m\) language of the golden-mean shift, but it is not compatible with naive truncation when the window size varies. We show that there is a unique restriction map on microstates for which normalization commutes with restriction, and that the discrepancy from naive truncation may occupy the whole visible window; in particular, no uniformly bounded tail correction can restore commutation. Our main structural result identifies the bulk process of raw and normalized digits with an explicit four-state synchronized sofic shift. From this presentation we derive the exact discrepancy statistics, including density \(4/9\), asymptotic variance \(118/243\), closed covariance formulas, and a rational power spectrum. The same graph yields a local inverse for the induced sliding block code for every \(m\ge 3\), and it also provides a transfer-matrix description of weighted fibers, from which the free energy is recovered as the Legendre transform of a strictly concave entropy profile.
\end{abstract}
