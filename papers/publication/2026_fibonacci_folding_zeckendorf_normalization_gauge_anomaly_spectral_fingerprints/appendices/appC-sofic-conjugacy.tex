\section{Appendix C: Sofic graph models and the conjugacy threshold}
\label{app:sofic}

\subsection{The labeled graph presentation}
Proposition~\ref{prop:sofic} constructs a labeled directed graph \(G_m=(V_m,E_m,\lambda_m)\) with \(V_m=\set{0,1}^{m-1}\) and an edge for each length-\(m\) block \(vb\) (\(v\in\set{0,1}^{m-1}\), \(b\in\set{0,1}\)), labeled by \(\lambda_m(vb)=\Fold_m(vb)\in X_m\). The image shift \(Y_m\) is precisely the label image of the edge shift of \(G_m\).

\subsection{Why m >= 3 yields a conjugacy}
For \(m\ge 3\), the labeled graph admits a finite synchronization delay: after observing a block of labels of length \(m-1\), the underlying vertex path becomes uniquely determined. This implies that the label map is finite-to-one with a deterministic inverse sliding block code of memory \(m-1\), proving Theorem~\ref{thm:conjugacy}.

\subsection{The m=2 degeneracy}
When \(m=2\), the synchronization delay is insufficient: two distinct microstate sequences can generate the same label sequence, and the ambiguity collapses exactly onto two periodic points (the two fixed points). This yields Theorem~\ref{thm:m2}.
