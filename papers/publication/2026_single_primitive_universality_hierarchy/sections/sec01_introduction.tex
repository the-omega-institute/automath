\section{Introduction}\label{sec:intro}
% ══════════════════════════════════════════════════════════

The idea that a single primitive can generate a rich family of
objects is a recurring structural motif in mathematics and logic.
In Boolean algebra the Sheffer stroke (NAND) suffices to express
every truth function~\cite{Sheffer1913}.
In the $\lambda$-calculus a single variable-binding construct
generates all computable functions~\cite{Church1936}.
In combinatory logic two combinators $S$ and~$K$ are enough;
with Sch\"onfinkel's encoding even a single combinator
suffices~\cite{Schonfinkel1924}.

A recent and striking addition to this lineage is
Odrzywołek's demonstration that \emph{every} elementary function
available on a standard scientific calculator---the 36-element
basis comprising arithmetic, algebraic, trigonometric, hyperbolic,
and logarithmic operations---can be reconstructed from a single
binary operator
\begin{equation}\label{eq:eml-def-intro}
  \eml(x,y) \;=\; e^{x} - \ln y
\end{equation}
together with the constant~$1$, using the minimal context-free
grammar $S \to 1 \mid \eml(S,S)$~\cite{Odrzywołek2026}.
This collapses the \emph{expression library} to a uniform binary
tree syntax---we call this \textbf{expression universality}.

The present note identifies expression universality as the bottom
rung of a three-level hierarchy whose upper levels are qualitatively
stronger.

\begin{enumerate}[label=\textbf{L\arabic*},leftmargin=2.5em]
\item \textbf{Expression universality} (Odrzywołek):
  a diverse function library is collapsed to a single operator and
  a tree grammar.  The operations of composition and evaluation
  remain external to the formalism.
\item \textbf{Protocol universality} (this paper, building on
  \cite{OmegaSPG2026}):
  integer arithmetic operations---addition and multiplication
  themselves---are not primitive but \emph{emerge} from a single
  scan--projection protocol.  The unique primitive is function
  composition~$\circ$, and $+$~and~$\times$ are its level-0 and
  level-1 readouts.  Canonical forms are computable, and the word
  problem for projection-word equivalence is decidable.
\item \textbf{Certificate universality} (this paper, building on
  \cite{OmegaSPG2026}):
  the fold fibers carry collision-moment sequences
  $\{S_q(m)\}_{q \ge 0}$ whose Hankel matrices are positive
  semi-definite.  A finite Prony--Hankel reconstruction recovers
  the full multiplicity spectrum from $2r$ moments, yielding an
  auditable, finite-dimensional spectral certificate at each
  resolution~$m$.
\end{enumerate}

\centering\small
\caption{Three levels of single-primitive universality.}
\label{tab:hierarchy}
\begin{tabular}{@{}llll@{}}
\toprule

\midrule
L0: Expression  & function library  & $\eml + 1$  & none intrinsic \\

\end{tabular}
\end{table}

The relationships between levels are directional rather than
strict class containments (\Cref{sec:strictness}):
\begin{itemize}[nosep]
\item \textbf{L0 vs L1.}  EML and the SPG protocol are
  \emph{incomparable} as classes---EML is expression-universal
  but not protocol-universal (the word problem is undecidable;
  \Cref{thm:separation}), and protocol universality is not
  entailed by expression universality.  When a
  semantically-compatible evaluator exists
  (\Cref{prop:L1-compiles-L0}), L1 strictly strengthens L0
  by providing decidable canonical forms.
\item \textbf{L1 vs L2.}  Clauses~(a) and~(b) of~L2
  (Hankel PSD, finite Prony reconstruction) are generic for
  protocol-universal systems with finite fibers
  (\Cref{rem:hankel-psd-generic}).  Clause~(c) (polynomial-
  dimension transfer-operator computability) is the
  non-trivial structural content of L2, witnessed in the
  SPG setting by \Cref{thm:star-kernel}.  Whether this
  clause fails for some protocol-universal system---hence
  witnessing a strict class containment---is left open
  (\Cref{rem:L1-L2-strictness-open}).
\end{itemize}

\paragraph{Relation to the main work.}
The protocol and certificate layers are developed in full
generality in~\cite{OmegaSPG2026}.  The purpose of this note is
narrower: to make the hierarchy explicit, to state the
level-by-level structural relations cleanly, and to anchor
the comparison with Odrzywołek's EML operator as a concrete
external reference point.
