# Prefix Scan Error Boundary Rates

- Working title: `Scan Error Profiles on Prefix Partitions and Boundary-Controlled Convergence Rates`
- Primary target journal: `Ergodic Theory and Dynamical Systems (ETDS)` (provisional)
- Secondary targets: `Nonlinearity`, `Israel Journal of Mathematics`
- Source section: `sections/body/spg/`

## Positioning

This draft isolates the scan-error part of the SPG chapter as an independent
measurable/symbolic-dynamics paper.  The target contribution is narrow and
self-contained:

1. exact cylinder decomposition of the scan-error profile;
2. Bayes-optimal threshold characterization on prefix partitions;
3. discrete Tanaka--Stokes representation of error decay;
4. boundary-cylinder control of convergence rates.

The paper should remain a focused error-decay / observable-information paper.
It should not absorb the full address-semantics package, logic-expansion
language, or the later POM / zeta consequences.

## Scope Kept

- scan-error profile on prefix filtrations;
- Bayes threshold rule;
- Tanaka--Stokes representation;
- boundary-cylinder dimension and scaling laws;
- finite-horizon and golden-mean illustrations.

## Scope Cut

- recursive addressing as a full standalone paper;
- forcing language;
- POM / zeta forward references;
- heavier geometric or cubical infrastructure.
