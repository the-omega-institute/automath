\section{Preliminaries}\label{sec:preliminaries}

Throughout the paper, $(X,\mathcal{B},\mu,T)$ denotes a probability-preserving
system and $W\in\mathcal{B}$ is a fixed measurable readout window. We write
$\Sigma:=\{0,1\}$ and $\Sigma^{<\infty}:=\bigcup_{m\ge 0}\Sigma^m$.

\subsection{Binary readout and the visible symbolic factor}

For $t\in\NNzero$ define the binary readout $s_t(x):=\ind_W(T^t x)$.
For $m\ge 1$ set $\omega_m(x;t):=s_t(x)\cdots s_{t+m-1}(x)\in\Sigma^m$,
and for $a\in\Sigma^m$ write $\abs{a}=m$ and
\[
  C_a:=\{x\in X:\omega_m(x;0)=a\}
  =\bigcap_{j=0}^{m-1}T^{-j}(W^{a_j}),
\]
where $W^1:=W$ and $W^0:=X\setminus W$.  The finite cylinder partition
$\mathcal{C}_m:=\{C_a:a\in\Sigma^m\}$ generates the $\sigma$-algebra
$\mathcal{F}_m$, and
$\mathcal{F}_\infty:=\sigma(\bigcup_{m\ge 1}\mathcal{F}_m)$.

The infinite coding $\omega_\infty(x):=(s_t(x))_{t\ge 0}$ defines a
measurable semiconjugacy $\omega_\infty\circ T=\sigma\circ\omega_\infty$
from $(X,T)$ to $(\Sigma^{\NNzero},\sigma)$, with
$C_a=\omega_\infty^{-1}([a])$ and
$\mathcal{F}_m=\omega_\infty^{-1}(\mathcal{A}_m)$.

\subsection{Conditional probabilities at finite resolution}

If $\mathsf{P}\in\mathcal{F}_\infty$, then for each $m\ge 1$ we write
$p_m:=\mu(\mathsf{P}\mid\mathcal{F}_m)$.
The process $(p_m)_{m\ge 1}$ is the finite-resolution conditional-probability
martingale associated with $\mathsf{P}$; it governs both the optimal
classification rule and the boundary-decay
identities in Section~\ref{sec:clarity}.

When we want the symbolic factor to recover all measurable information, we
assume that the binary partition $\{W,X\setminus W\}$ is generating, so
$\mathcal{F}_\infty=\mathcal{B}$ modulo $\mu$-null sets.
This hypothesis is used only for full-information remarks; the main
open-system theorems do not require it.
