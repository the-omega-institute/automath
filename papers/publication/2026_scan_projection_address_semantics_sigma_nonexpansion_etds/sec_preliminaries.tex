\section{Preliminaries}\label{sec:preliminaries}

The observation model is the same throughout the paper: one sees a
finite-alphabet symbolic process at finite depth.  Let
\((\Omega,\mathcal B,\mu)\) be a probability space and let
\[
  y=(y_t)_{t\geq 0}:\Omega\to \mathcal A_{\mathrm{obs}}^{\NNzero}
\]
be a measurable process with values in a finite alphabet
\(\mathcal A_{\mathrm{obs}}\).  For \(m\geq 1\), define
\[
  \mathcal F_m:=\sigma(y_0,\ldots,y_{m-1}),
  \qquad
  \mathcal F_\infty:=\sigma\Bigl(\bigcup_{m\geq 1}\mathcal F_m\Bigr).
\]
If \(a=a_0\cdots a_{m-1}\in \mathcal A_{\mathrm{obs}}^m\), write
\[
  C_a:=\{\omega\in\Omega:(y_0,\ldots,y_{m-1})(\omega)=a\}.
\]
Then the non-empty cylinders \(C_a\) form the partition generating
\(\mathcal F_m\).

For an event \(\mathsf P\in\mathcal F_\infty\), let
\[
  p_m:=\mu(\mathsf P\mid \mathcal F_m).
\]
The martingale \((p_m)_{m\geq 1}\) is the finite-resolution conditional
probability associated with \(\mathsf P\); it is the only observation
process used in the Bayes-error formulas below.

In the main symbolic sections, the observation process is the coordinate
process on a one-step subshift of finite type:
\[
  y_t(\omega)=\omega_t,
  \qquad
  \mathcal A_{\mathrm{obs}}=A.
\]
After an \(\ell\)-block recoding, the same definition is used with the
block alphabet \(A^{(\ell)}\).  No separate binary-readout model is used
in the proofs.
