\section{Recursive visible refinements}\label{sec:recursive-addressing}

Recursive visible refinements---where one re-codes the visible process
level by level through cylinder partitions of the previous level's
symbolic shift---remain inside ordinary factor theory.  The only
ingredient we need later is that every finite refinement is already a
sliding-block code of the base visible process, so no genuinely new
information is created.

\begin{lemma}[Finite-depth collapse]
\label{thm:finite-depth-collapse}
For every \(L\geq 0\) there exists a sliding-block code
\(\Psi_L\) of window length \(R_L\) such that
\(\pi^{(L)}=\Psi_L\circ\pi^{(0)}\) \(\mu\)-almost everywhere, where
\(R_0:=1\) and \(R_L:=1+\sum_{q=1}^{L}(s_q-1)\) with \(s_q\) the
maximum cylinder length at level \(q\).
Consequently \(\mathcal{F}_m^{(L)}\subseteq \mathcal{F}_{m+R_L-1}^{(0)}\)
and the Bayes error can only decrease after recoding:
\(\varepsilon_{m+R_L-1}^{(0)}(P;\mu)\leq \varepsilon_m^{(L)}(P;\mu)\).
\end{lemma}

\begin{proof}[Sketch]
This is the standard composition of sliding-block codes: each level adds
at most \(s_q-1\) symbols of memory, giving the stated window length
\cite[Chapter~1]{LindMarcus1995}.  Pulling back \(m\)-cylinders through
the composed code yields the \(\sigma\)-algebra inclusion and the error
monotonicity.
\end{proof}
