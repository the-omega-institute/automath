\subsection{Forcing necessity of the enrichment}

The three null modes show that the local-object enrichment distinguishes situations invisible to reference-aware state forcing. This subsection proves the separation at the exact $\mathbb L_1$--$\mathbb L_2$ boundary. The type discipline is the one fixed in \Cref{prop:theorem-a-boundary-typing,def:reference-aware-boundary,prop:layer-typing-comparison-boundary}: a free variable $x:\mathsf{Ref}_s$ and the atom $\Adm_s(x)$ are $\Sigma_1$ data, while $\LocSec_s$, $\CompSec_s$, and $\Sec_s$ are newly adjoined $\Sigma_2$ predicates. The formulas on the lower side are parameter-free formulas of the reference-aware fragment
\[
\Frm_1^{\mathrm{ref}}(s)\subseteq \mathrm{Form}_1
\]
from \Cref{def:reference-aware-comparison-fragment}. This fragment is neither the original pointwise language nor a reference-free reduct: it is the $\Sigma_1$ language with the reference sort and admissibility predicate retained, and with all local-object symbols absent.

The separation argument fixes one $\Sigma_1$ reduct, one information state, and one admitted reference element. In the explicit separating family it fixes even the finite site and its covers, and varies only the local object functor whose induced predicates are $\LocSec_s$, $\CompSec_s$, and $\Sec_s$. Hence indistinguishability is literal reduct invariance for the same pointed lower structure. The proof never compares two different reference elements and never adds a constant symbol naming the chosen reference; the same semantic element is evaluated in all local-object expansions. The parameter-free restriction is the chosen syntactic comparison class, but the invariance argument below uses only the stronger fact that all lower-layer data, including any fixed lower parameters one might name, are unchanged across the comparison.

Thus every comparison in this subsection has three simultaneous coordinates:
\[
\begin{gathered}
\text{same }\Sigma_1\text{ reduct }N,\\
\text{same pointed state }p[r/x],\\
\text{variable local object functor over a fixed }\Sigma_2\text{ site}.
\end{gathered}
\]
The first two coordinates are what lower formulas can inspect; the third coordinate is exactly what the target predicates $\CompSec_s$, $\Sec_s$, and $\Null_s^{\mathrm{glue}}$ inspect.

\begin{corollary}[Well-typed lower comparison at admitted references]\label{cor:well-typed-lower-comparison}
Let $s$ be referenceable. If
\[
\varphi(x)\in \Frm_1^{\mathrm{ref}}(s),
\qquad
x:\mathsf{Ref}_s,
\]
and $r\in D_{\mathsf{Ref}_s}^M$, then the pointed assertion
\[
M,p\Vdash\varphi[r]
\]
is a well-typed $\Sigma_1$ assertion. If moreover $M,p\Vdash\Adm_s(r)$, that admissibility hypothesis is also a $\Sigma_1$ assertion. By contrast,
\[
\LocSec_s(r),\qquad \CompSec_s(r),\qquad \Sec_s(r),
\qquad \Null_s^{\mathrm{glue}}(r)
\]
are $\Sigma_2$ target conditions and are not formulas of $\Frm_1^{\mathrm{ref}}(s)$.
\end{corollary}

\begin{proof}
This is \Cref{prop:layer-typing-comparison-boundary,cor:boundary-correct-reference-formulas} restated at a displayed variable, together with the reduct statement in \Cref{lem:reference-data-survive-reduct}.

The sort $\mathsf{Ref}_s$ and predicate $\Adm_s$ are in $\Sigma_1$, so substitution of a reference element into a $\Sigma_1$ formula is type-correct. The sectionability predicates first occur at the $\mathbb L_1\leadsto\mathbb L_2$ interface, and $\Null_s^{\mathrm{glue}}$ contains $\CompSec_s$ and $\Sec_s$ by \Cref{def:null-failure-modes}.
\end{proof}

We use the pointed-reference evaluation convention of \Cref{def:pointed-reference-evaluation} throughout this subsection. Thus $M,p\Vdash\chi[r]$ means that the displayed free variable of $\chi(x)$ is interpreted by the semantic reference element $r$ in the pointed state; it is not a hidden parameter or an added constant of the lower language.

\begin{lemma}[No type shift in the comparison fragment]\label{lem:no-type-shift-comparison-fragment}
For a referenceable sort $s$, membership
\[
\varphi(x)\in\Frm_1^{\mathrm{ref}}(s)
\]
means exactly the lower-layer type judgment
\[
x:\mathsf{Ref}_s\vdash \varphi(x)\in\mathrm{Form}_1
\quad\text{over }\Sigma_1^{\mathrm{ref}}(s),
\]
where $\mathsf{Ref}_s$ and $\Adm_s$ are already in $\Sigma_1$. It does not mean a formula of the original pointwise language $\mathrm{Form}_0$, and it does not mean a formula obtained by first adding $\Sigma_2$ local-object data and then forgetting them. The predicates
\[
\CompSec_s,\qquad \Sec_s,\qquad \Null_s^{\mathrm{glue}}
\]
are instead $\Sigma_2$ target predicates on the same semantic reference element.
\end{lemma}

\begin{proof}
This is the boundary contract of \Cref{def:reference-aware-boundary,def:reference-aware-comparison-fragment}. The symbols $\mathsf{Ref}_s$ and $\Adm_s$ belong to $\Sigma_1^{\mathrm{ref}}(s)$, while \Cref{prop:layer-typing-comparison-boundary} puts $\LocSec_s,\CompSec_s,\Sec_s$ outside $\Sigma_1$. Since $\Null_s^{\mathrm{glue}}(x)$ contains $\CompSec_s(x)$ and $\Sec_s(x)$, it is outside the lower fragment as well. The target predicates are therefore evaluated only as external $\Sigma_2$ conditions on local-object expansions.
\end{proof}

\begin{definition}[Lower-fragment definability over local-object expansions]\label{def:lower-fragment-definability}
Let $P_s(x)$ be a $\Sigma_2$ condition on admitted references of sort $\mathsf{Ref}_s$, for example $\CompSec_s(x)$, $\Sec_s(x)$, or $\Null^{\mathrm{glue}}_s(x)$. A parameter-free formula
\[
\chi(x)\in \Frm_1^{\mathrm{ref}}(s)
\]
\emph{defines $P_s$ on admitted references over local-object expansions} if for every $\Sigma_2$ local-object model $M$, every state $p$, and every element $r\in D_{\mathsf{Ref}_s}^M$ with
\[
M,p\Vdash \Adm_s(r),
\]
one has
\[
M,p\Vdash \chi[r]
\quad\Longleftrightarrow\quad
M,p\Vdash P_s[r].
\]
The left-hand side is shorthand for evaluation in the retained lower reduct:
\[
M,p\Vdash \chi[r]
\quad\text{means}\quad
U_{2,1}(M),\pi^M_{2,1}(p)\Vdash_1 \chi[r].
\]
The right-hand side is evaluated in the $\Sigma_2$ local-object expansion. This is the only definability notion used in Theorem~A: the defining formula lives in the $\Sigma_1$ reduct, while the target condition may use the added $\Sigma_2$ local-object data.
The parameter-free restriction is part of the comparison class. It keeps the statement focused on formulas built from the symbols present at the $\mathbb L_1$ layer. The proof does not rely on varying the chosen reference or hiding it from the lower language; it fixes the semantic element $r$ and varies only the $\Sigma_2$ local-object expansion.
The target predicates are not assigned arbitrary truth values. In every local-object expansion they are interpreted from the site, covers, and functor by \Cref{def:existence-levels}, and $\Null_s^{\mathrm{glue}}$ is then the derived predicate of \Cref{def:null-failure-modes}.
\end{definition}

\begin{definition}[Fixed-reduct comparison class]\label{def:fixed-reduct-comparison-class}
Let $N$ be a $\Sigma_1$ model, let $p$ be a $\Sigma_1$ information state of $N$, and let
\[
r\in D_{\mathsf{Ref}_s}^{N}
\]
with $N,p\Vdash\Adm_s(r)$. Write
\[
\mathrm{Exp}_2(N,p,r)
\]
for the class of all $\Sigma_2$ local-object expansions $M$ such that:
\begin{enumerate}[label=(\roman*),nosep]
\item $U_{2,1}(M)=N$;
\item the state $p$ and the element $r$ are the same state and element after applying the reduct functor;
\item the interpretation of $\mathsf{Ref}_s$, $\Adm_s$, and every other $\Sigma_1$ symbol is exactly the one fixed in $N$.
\end{enumerate}
A formula $\chi(x)\in\Frm_1^{\mathrm{ref}}(s)$ \emph{defines} a $\Sigma_2$ condition $P_s(x)$ on this fixed reduct if
\[
M,p\Vdash\chi[r]
\quad\Longleftrightarrow\quad
M,p\Vdash P_s[r]
\]
for every $M\in\mathrm{Exp}_2(N,p,r)$. Global definability in the sense of \Cref{def:lower-fragment-definability} implies fixed-reduct definability for every such triple $(N,p,r)$. Therefore a single fixed reduct on which no lower formula defines $P_s$ is already enough to refute uniform definability over local-object expansions.
The chosen element $r$ is not a parameter of the lower formula; it is part of the pointed semantic evaluation of \Cref{def:pointed-reference-evaluation}. Thus the syntactic comparison class remains parameter-free even though the semantic test is made at a fixed admitted reference. Since the same $r$ is used in every expansion of the class, even naming lower-layer elements from $N$ would not create sensitivity to the changed site, cover, or functor data.
The local-object symbols in $\Sigma_2\setminus\Sigma_1$ may vary freely only subject to the axioms of a local-object expansion and the defining clauses for $\LocSec_s$, $\CompSec_s$, and $\Sec_s$; the lower reduct $N$ is kept fixed literally.
\end{definition}

\begin{lemma}[Rigid lower reduct in fixed comparisons]\label{lem:fixed-comparison-lower-rigidity}
Fix a triple $(N,p,r)$ as in \Cref{def:fixed-reduct-comparison-class}. If
\[
M,M'\in \mathrm{Exp}_2(N,p,r),
\]
then the pointed $\Sigma_1$ structures used to evaluate lower formulas are identical:
\[
U_{2,1}(M)=U_{2,1}(M')=N,
\qquad
p[r/x]\text{ is the same pointed information state in both reducts}.
\]
In particular, the reference domain, equality on reference elements, the interpretation of $\Adm_s$, all other $\Sigma_1$ symbols, and the realization set of the pointed information state are the same in $M$ and $M'$. Only the symbols in
\[
\Delta_{1,2}=\{\mathcal C_{p,s},\Cov_{p,s},F_{p,s},\LocSec_s,\CompSec_s,\Sec_s\}
\]
may differ.
\end{lemma}

\begin{proof}
This is the definition of $\mathrm{Exp}_2(N,p,r)$ together with \Cref{prop:no-reference-increment,prop:chain-instantiation}. The reduct $U_{2,1}$ retains the whole $\Sigma_1$ interpretation, including $\mathsf{Ref}_s$ and $\Adm_s$, and forgets exactly the local-object increment $\Delta_{1,2}$. The pointed state $p[r/x]$ is formed from the fixed state $p$ and the fixed semantic element $r$, so it is also the same on both sides.
\end{proof}

\begin{lemma}[Typed lower comparison formulas]\label{lem:typed-lower-comparison-formulas}
Let $s$ be referenceable. If
\[
\varphi(x)\in\Frm_1^{\mathrm{ref}}(s),
\qquad
x:\mathsf{Ref}_s,
\]
then $\varphi$ is interpreted in the $\Sigma_1$-reduct of any local-object model. Its truth value at a realization depends only on that $\Sigma_1$-reduct and never on the $\Sigma_2$-data consisting of sites on admitted reference classes, covers, local object functors, or the sectionability predicates.
\end{lemma}

\begin{proof}
By \Cref{prop:layer-typing-comparison-boundary,def:reference-aware-comparison-fragment,cor:boundary-correct-reference-formulas,lem:reference-data-survive-reduct}, $\mathsf{Ref}_s$ and $\Adm_s$ are already symbols of $\Sigma_1$, while $\LocSec_s$, $\CompSec_s$, and $\Sec_s$ are not. The forgetful functor $U_{2,1}$ in \Cref{prop:chain-instantiation}(ii) retains the reference domains and the interpretation of $\Adm_s$, and forgets exactly the site-on-reference-classes and local-object structure. A structural induction on $\varphi$ now gives the claim. Atomic formulas in $\varphi$ use only $\Sigma_1$ symbols; Boolean connectives and quantifiers are evaluated inside the same retained $\Sigma_1$ domains; and the state-forcing clause only quantifies over the realizations in the pointed state of \Cref{def:pointed-reference-evaluation}, which is formed entirely from the lower state and the retained reference element. Hence evaluating a formula of $\Frm_1^{\mathrm{ref}}(s)$ after passing from $\mathbb L_2$ to the $\mathbb L_1$-reduct loses no symbol or semantic coordinate used by the formula.
\end{proof}

\begin{lemma}[Reduct invariance of lower formulas]\label{lem:lower-formula-reduct-invariance}
Let $M$ and $M'$ be two $\Sigma_2$ local-object expansions with the same $\Sigma_1$ reduct. Let $p$ be a state and let $r\in D_{\mathsf{Ref}_s}$ be the same reference element in the common reduct. Then every
\[
\varphi(x)\in\Frm_1^{\mathrm{ref}}(s)
\]
satisfies
\[
M,p\Vdash\varphi[r]
\quad\Longleftrightarrow\quad
M',p\Vdash\varphi[r].
\]
\end{lemma}

\begin{proof}
By \Cref{lem:typed-lower-comparison-formulas}, the formula $\varphi$ is interpreted entirely in the $\Sigma_1$ reduct. The two expansions have the same reference domain, the same interpretation of $\Adm_s$, the same interpretation of every other lower symbol, and the same state. The pointed states $p[r/x]$ are therefore identical in the common $\Sigma_1$ reduct. Thus each realization in $R_{p[r/x]}$ gives the same pointwise truth value for $\varphi(x)$ in $M$ and in $M'$. The state-forcing equivalence follows by quantifying over exactly the same pointed realizations.
\end{proof}

\medskip
\noindent\emph{Lower parameters do not see the local-object expansion.}
The parameter-free convention is the comparison class used in the theorem, but the fixed-reduct argument is stronger. If a tuple $\bar c$ of elements of lower sorts is held fixed in the common $\Sigma_1$ reduct and $\varphi(x,\bar y)$ is any $\Sigma_1$ formula, then the same proof as \Cref{lem:lower-formula-reduct-invariance} gives
\[
M,p\Vdash \varphi[r,\bar c]
\quad\Longleftrightarrow\quad
M',p\Vdash \varphi[r,\bar c].
\]
Thus the separation below is not obtained by concealing the reference element or other lower-layer data from the language. It is obtained because the site, covers, local object functor, and sectionability predicates are precisely the coordinates forgotten by $U_{2,1}$.
\medskip

\begin{lemma}[Lower truth is a function of the fixed reduct]\label{lem:lower-truth-function-fixed-reduct}
Fix a triple $(N,p,r)$ as in \Cref{def:fixed-reduct-comparison-class}. For every
\[
\varphi(x)\in\Frm_1^{\mathrm{ref}}(s)
\]
there is a truth value
\[
\tau_{N,p,r}(\varphi)\in\{\mathrm{true},\mathrm{false}\}
\]
such that every local-object expansion $M\in\mathrm{Exp}_2(N,p,r)$ satisfies
\[
M,p\Vdash\varphi[r]
\quad\Longleftrightarrow\quad
\tau_{N,p,r}(\varphi)=\mathrm{true}.
\]
The value $\tau_{N,p,r}(\varphi)$ is computed in the $\Sigma_1$ structure $N$ alone.
\end{lemma}

\begin{proof}
Choose any $M\in\mathrm{Exp}_2(N,p,r)$ and set $\tau_{N,p,r}(\varphi)$ to be the truth value of $M,p\Vdash\varphi[r]$. If $M'$ is another expansion in the same class, then $M$ and $M'$ have the same $\Sigma_1$ reduct, the same state, and the same semantic reference element. \Cref{lem:lower-formula-reduct-invariance} gives
\[
M,p\Vdash\varphi[r]\Longleftrightarrow M',p\Vdash\varphi[r],
\]
so the definition is independent of the expansion chosen. The computation uses only $N$, because all symbols and all realizations used by $\varphi$ belong to the retained $\Sigma_1$ reduct.
\end{proof}

\begin{corollary}[Reference-aware lower resources are still fixed]\label{cor:reference-aware-resources-fixed}
In the fixed-reduct comparison class of \Cref{def:fixed-reduct-comparison-class}, lower formulas may use the full reference-aware $\Sigma_1$ apparatus: equality of reference elements, the predicate $\Adm_s$, quantifiers over $\mathsf{Ref}_s$, and any other $\Sigma_1$ relation or function. None of these resources can distinguish two expansions in $\mathrm{Exp}_2(N,p,r)$ unless the expansions already differ in their $\Sigma_1$ reduct. Consequently the separation below is not an automorphism-invariance argument and is not obtained by comparing two different references; it is an invariance statement for the same pointed reference in the same lower structure.
\end{corollary}

\begin{proof}
All listed resources are symbols, domains, or forcing clauses of the common $\Sigma_1$ reduct $N$. By \Cref{lem:fixed-comparison-lower-rigidity}, two expansions in $\mathrm{Exp}_2(N,p,r)$ have the same reference domain, the same equality relation, the same admissibility predicate, the same interpretations of all other $\Sigma_1$ symbols, and the same pointed state $p[r/x]$. The structural induction in \Cref{lem:typed-lower-comparison-formulas} therefore applies unchanged to formulas using any of these lower resources.
\end{proof}

\begin{lemma}[Fixed-reduct variation obstruction]\label{lem:fixed-reduct-variation-obstruction}
Let $P_s(x)$ be a $\Sigma_2$ condition on admitted references of sort $\mathsf{Ref}_s$. Suppose that, for some fixed triple $(N,p,r)$ with
\[
N,p\Vdash\Adm_s(r),
\]
there are two expansions
\[
M,M'\in \mathrm{Exp}_2(N,p,r)
\]
such that
\[
M,p\Vdash P_s[r]
\qquad\text{and}\qquad
M',p\nVdash P_s[r].
\]
Then no parameter-free formula
\[
\chi(x)\in\Frm_1^{\mathrm{ref}}(s)
\]
defines $P_s$ on the fixed-reduct comparison class $\mathrm{Exp}_2(N,p,r)$.
\end{lemma}

\begin{proof}
Assume that such a formula $\chi(x)$ defined $P_s$ on $\mathrm{Exp}_2(N,p,r)$. Since $M$ and $M'$ have the same $\Sigma_1$ reduct, the same state, and the same reference element $r$, \Cref{lem:lower-formula-reduct-invariance} gives
\[
M,p\Vdash\chi[r]
\quad\Longleftrightarrow\quad
M',p\Vdash\chi[r].
\]
The defining equivalence for $P_s$ would identify these two equal lower truth values with the two different truth values of $P_s[r]$, a contradiction.
\end{proof}

\begin{lemma}[Fixed classes test uniform lower definability]\label{lem:fixed-classes-test-uniform-definability}
Let $P_s(x)$ be one of the $\Sigma_2$ target conditions considered in \Cref{def:lower-fragment-definability}. If a formula
\[
\chi(x)\in\Frm_1^{\mathrm{ref}}(s)
\]
defines $P_s$ on admitted references over all local-object expansions in the sense of \Cref{def:lower-fragment-definability}, then for every fixed triple $(N,p,r)$ as in \Cref{def:fixed-reduct-comparison-class}, the same $\chi$ defines $P_s$ on the whole fixed class $\mathrm{Exp}_2(N,p,r)$.
\end{lemma}

\begin{proof}
Let $M\in\mathrm{Exp}_2(N,p,r)$. By definition of the fixed class, $U_{2,1}(M)=N$, the state and reference element are the fixed $p$ and $r$, and
\[
N,p\Vdash\Adm_s(r).
\]
Since $\Adm_s$ belongs to $\Sigma_1$ and is preserved by the reduct, \Cref{lem:reference-data-survive-reduct} gives
\[
M,p\Vdash\Adm_s(r).
\]
The global defining equivalence of \Cref{def:lower-fragment-definability} therefore applies to this particular $M,p,r$ and yields
\[
M,p\Vdash\chi[r]\quad\Longleftrightarrow\quad M,p\Vdash P_s[r].
\]
As $M$ was arbitrary in the fixed class, $\chi$ defines $P_s$ on $\mathrm{Exp}_2(N,p,r)$.
\end{proof}

\begin{proposition}[Fixed lower reduct and fixed site with variable local-object functor]\label{prop:fixed-lower-reduct-three-expansions}
There exist a $\Sigma_1$-reduct $N$, the singleton empty-context state
\[
p=(\varnothing,\{\rho_\varnothing\}),
\]
a referenceable sort $s$, an element
\[
r\in D_{\mathsf{Ref}_s}^{N},
\]
and three $\Sigma_2$ local-object expansions
\[
M_0,\quad M_1,\quad M_2
\]
with
\[
U_{2,1}(M_i)=N\qquad (i=0,1,2),
\]
all with the same state $p$, the same reference element $r$, and the same finite thin site on the admitted reference classes, such that:
\begin{enumerate}[label=(\roman*),nosep]
\item $M_i,p\Vdash \Adm_s(r)$ for $i=0,1,2$;
\item $M_0,p\nVdash \CompSec_s(r)$ and $M_0,p\nVdash \Sec_s(r)$;
\item $M_1,p\Vdash \CompSec_s(r)\land \Sec_s(r)$;
\item $M_2,p\Vdash \CompSec_s(r)$ and $M_2,p\nVdash \Sec_s(r)$;
\item $M_1,p\nVdash \Null^{\mathrm{glue}}_s(r)$, while $M_2,p\Vdash \Null^{\mathrm{glue}}_s(r)$.
\end{enumerate}
\end{proposition}

\begin{proof}
Take one referenceable sort $s$. Let the $\Sigma_1$ reference domain contain four distinct elements
\[
r,u,v,c
\]
and interpret $\Adm_s$ as true exactly on these elements. Put
\[
\Gamma_p=\varnothing,
\qquad
R_p=\{\rho_\varnothing\}.
\]
All remaining $\Sigma_1$ symbols are interpreted once and for all; they are not changed in the three expansions. In particular, equality of reference elements, the interpretation of $\Adm_s$, the information state $p$, and the distinguished element $r$ all belong to the fixed lower reduct. Write
\[
a=[r]_p,\qquad b=[u]_p,\qquad d=[v]_p,\qquad e=[c]_p
\]
for the four admitted reference classes.

The fixed lower structure is now complete for the separation proof: every $\Sigma_1$ formula with displayed variable $x:\mathsf{Ref}_s$ is evaluated against the same domain, the same admissibility predicate, the same equality relation, the same state $p$, and the same semantic value $r$. The next paragraphs alter only the local-object data allowed at $\Sigma_2$.
No truth value of $\LocSec_s$, $\CompSec_s$, or $\Sec_s$ is stipulated independently; those predicates are read off from the displayed site and functor data by \Cref{def:existence-levels}.

Fix, once for all three expansions, the thin category generated by
\[
e\sqsubseteq b\sqsubseteq a,
\qquad
e\sqsubseteq d\sqsubseteq a,
\]
so that $e=b\times_a d$. The pretopology is generated by identity covers and the nonempty binary cover
\[
\{b\to a,\ d\to a\},
\]
then closed under pullback and transitivity. This same category and this same pretopology are used in $M_0$, $M_1$, and $M_2$.

In $M_0$ put
\[
F^0_{p,s}(a)=F^0_{p,s}(b)=F^0_{p,s}(d)=F^0_{p,s}(e)=\varnothing.
\]
Every restriction map is the unique function with empty domain. The displayed binary cover of $a$ has no local section on either member, and the identity cover of $a$ has no section at $a$. Hence neither a compatible local family nor a global section exists over $a$.

In $M_1$ put
\[
F^1_{p,s}(a)=F^1_{p,s}(b)=F^1_{p,s}(d)=F^1_{p,s}(e)=\{*\},
\]
with all restrictions equal to the identity. The displayed binary cover supports the compatible family $(*,*)$, and $*\in F^1_{p,s}(a)$ is a global section.

In $M_2$ put
\[
F^2_{p,s}(b)=F^2_{p,s}(d)=F^2_{p,s}(e)=\{*\},
\qquad
F^2_{p,s}(a)=\varnothing,
\]
with the restrictions among $b,d,e$ equal to the identity and the restrictions out of $a$ the unique maps from the empty set. The same binary cover supports the compatible family $(*,*)$, because the two restrictions to $e$ agree, but $F^2_{p,s}(a)=\varnothing$ gives no global section.

These are valid local-object expansions. The displayed finite poset has the meet $e=b\times_a d$, and the pretopology is the smallest one containing identity covers and the binary cover $\{b\to a,d\to a\}$ and closed under pullback and transitivity. Pulling this binary cover back along $b\to a$ and $d\to a$ produces finite nonempty covers by objects already present in the poset, and all pairwise pullbacks required by \Cref{def:local-reference-frame} exist. The assignments $F^0_{p,s}$, $F^1_{p,s}$, and $F^2_{p,s}$ respect the unique arrows of the thin category, so they are contravariant functors.

The only data that change from $M_i$ to $M_j$ are now even narrower than the whole $\Sigma_2$ increment: the local object functor and the induced truth values of $\LocSec_s$, $\CompSec_s$, and $\Sec_s$. The site, covers, lower reference domain, interpretation of $\Adm_s$, equality of the four reference elements, state $p$, and distinguished semantic element $r$ are identical in the three expansions. Applying the forgetful functor $U_{2,1}$ of \Cref{prop:chain-instantiation}(ii) therefore returns the same $\Sigma_1$ structure $N$ for all three expansions.

The admissibility assertion is the same in all three expansions because $\Adm_s$ belongs to the fixed lower reduct $N$. The sectionability assertions are exactly the computations above, using \Cref{def:existence-levels}. In $M_0$ no cover of $a$ has sections on all of its members, so $\CompSec_s(r)$ fails; also $F^0_{p,s}(a)=\varnothing$, so $\Sec_s(r)$ fails. In $M_1$ the singleton section over $a$ gives both a global section and the induced compatible local family. In $M_2$ the binary cover gives a compatible local family, but $F^2_{p,s}(a)=\varnothing$ prevents a global section. Since the local-object predicates are bivalent by the forcing clauses in \Cref{def:existence-levels}, $M_1$ does not satisfy $\Null^{\mathrm{glue}}_s(r)$ because it has a global section, while $M_2$ does satisfy it because $r$ is admitted, the displayed compatible family exists, and $F^2_{p,s}(a)=\varnothing$.
\end{proof}

\begin{lemma}[Same-reference fixed-site separation scheme]\label{lem:same-reference-fixed-site-separation}
For the data constructed in \Cref{prop:fixed-lower-reduct-three-expansions}, the three expansions
\[
M_0,\quad M_1,\quad M_2
\]
belong to one fixed comparison class
\[
\mathrm{Exp}_2(N,p,r).
\]
They have the same $\Sigma_1$ reduct, the same pointed state $p[r/x]$, the same admitted semantic reference element $r$, and the same finite site and covers. Hence, for every
\[
\varphi(x)\in \Frm_1^{\mathrm{ref}}(s),
\]
there is a single lower truth value independent of $i$:
\[
M_i,p\Vdash\varphi[r]
\quad\Longleftrightarrow\quad
\tau_{N,p,r}(\varphi)=\mathrm{true}
\qquad (i=0,1,2).
\]
At the same time the induced $\Sigma_2$ target predicates have the pattern
\[
\begin{array}{c|ccc}
 & \CompSec_s(r) & \Sec_s(r) & \Null^{\mathrm{glue}}_s(r)\\ \hline
M_0 & \text{false} & \text{false} & \text{false}\\
M_1 & \text{true} & \text{true} & \text{false}\\
M_2 & \text{true} & \text{false} & \text{true}.
\end{array}
\]
Thus the separation scheme changes neither the reference element nor its admissibility status; it changes only the local object functor, whose induced sectionability predicates are outside the lower fragment.
\end{lemma}

\begin{proof}
The equality $U_{2,1}(M_i)=N$, the fixed state $p$, and the fixed element $r\in D_{\mathsf{Ref}_s}^N$ are part of \Cref{prop:fixed-lower-reduct-three-expansions}. The same proposition also fixes the site and covers for all three expansions and varies only $F_{p,s}$. Hence the three expansions lie in $\mathrm{Exp}_2(N,p,r)$ and have the same pointed $\Sigma_1$ structure by \Cref{lem:fixed-comparison-lower-rigidity}.

Now apply \Cref{lem:lower-truth-function-fixed-reduct}: each lower formula $\varphi(x)\in\Frm_1^{\mathrm{ref}}(s)$ is evaluated wholly in the fixed reduct $N$ at the fixed pointed state $p[r/x]$. This gives the displayed lower truth value $\tau_{N,p,r}(\varphi)$, independent of the expansion.

The sectionability table is the calculation in \Cref{prop:fixed-lower-reduct-three-expansions}. In $M_0$ all values of the local object functor on the displayed site are empty, so no compatible family and no global section exists. In $M_1$ the singleton value at $a$ gives both a global section and a compatible family. In $M_2$ the binary cover carries the compatible family but $F^2_{p,s}(a)=\varnothing$, so no global section exists. Since $r$ is admitted in all three expansions, \Cref{lem:admitted-gluing-null-reduction} gives the displayed values of $\Null^{\mathrm{glue}}_s(r)$.
\end{proof}

\noindent\emph{Boundary audit for the separation theorem.}
In the separating family of \Cref{prop:fixed-lower-reduct-three-expansions,lem:same-reference-fixed-site-separation}, the lower and upper sides of the comparison have the following exact status.
\begin{enumerate}[label=(\roman*),nosep]
\item The quantified lower formulas are precisely
\[
\varphi(x)\in\Frm_1^{\mathrm{ref}}(s),
\qquad
x:\mathsf{Ref}_s\vdash\varphi(x)\in\mathrm{Form}_1
\]
over the $\Sigma_1$ reference-aware signature. The atom $\Adm_s(x)$ is allowed in such formulas.
\item The symbols $\LocSec_s$, $\CompSec_s$, $\Sec_s$, and the derived predicate $\Null_s^{\mathrm{glue}}$ are not in $\Frm_1^{\mathrm{ref}}(s)$; they are $\Sigma_2$ target predicates computed from the local-object functor.
\item The element used in the evaluation is one fixed semantic element
\[
r\in D_{\mathsf{Ref}_s}^{N}
=D_{\mathsf{Ref}_s}^{M_i}
=D_{\mathsf{Ref}_s}^{U_{2,1}(M_i)}
\]
for all $i=0,1,2$. No comparison is made between different reference elements, and no constant naming $r$ is added to the lower language.
\item The site and covers are also fixed across $M_0,M_1,M_2$; only the local object functor $F_{p,s}$ varies. Therefore every lower formula sees the same pointed $\Sigma_1$ structure, while the sectionability predicates may change.
\end{enumerate}
Consequently Theorem~A is a typed lower-fragment separation at the $\mathbb L_1$--$\mathbb L_2$ boundary. It is not a statement about a reference-free $\mathrm{Form}_1$ language, and it is not a statement in which references or admissibility are supplied by the local-object layer.

\emph{Justification.}
Clause \textup{(i)} is the definition of $\Frm_1^{\mathrm{ref}}(s)$ from \Cref{def:reference-aware-comparison-fragment} together with the placement $\mathsf{Ref}_s,\Adm_s\in\Sigma_1$ in \Cref{prop:theorem-a-boundary-typing,prop:no-reference-increment}. Clause \textup{(ii)} is \Cref{cor:well-typed-lower-comparison,lem:no-type-shift-comparison-fragment}. Clause \textup{(iii)} is the domain preservation and fixed semantic element recorded in \Cref{prop:fixed-lower-reduct-three-expansions,lem:fixed-comparison-lower-rigidity}. Clause \textup{(iv)} is the fixed-site part of \Cref{prop:fixed-lower-reduct-three-expansions} and the sectionability calculation in \Cref{lem:same-reference-fixed-site-separation}. The final sentence is just these four clauses combined.

\begin{theorem}[Same-reference lower-fragment undefinability of gluing predicates]\label{thm:pointwise-irreducibility}
Let the $\mathbb L_1$--$\mathbb L_2$ boundary be the reference-aware boundary fixed in \Cref{prop:theorem-a-boundary-typing,prop:no-reference-increment,def:reference-aware-boundary} and summarized in the preceding boundary audit: variables
\[
x:\mathsf{Ref}_s
\]
and atoms $\Adm_s(x)$ are $\Sigma_1$ data, and the local-object predicates are the new $\Sigma_2$ data. At this boundary, the predicates
\[
\CompSec_s,\qquad \Sec_s,\qquad \Null_s^{\mathrm{glue}}
\]
are not definable, on admitted references and uniformly over local-object expansions, by any formula of the lower comparison fragment $\Frm_1^{\mathrm{ref}}(s)$. Here a lower formula is a parameter-free $\Sigma_1$ formula with displayed free variable $x:\mathsf{Ref}_s$, as fixed in \Cref{def:reference-aware-comparison-fragment}; it may mention $\Adm_s(x)$, but it contains none of the $\Sigma_2$ local-object predicates. The target predicate is evaluated externally in the $\Sigma_2$ expansion at the same semantic reference element.

The type judgment in the preceding sentence is part of the theorem, not a convention imported from the local-object layer. The quantifier over lower formulas is not a quantifier over a reference-free version of $\mathrm{Form}_1$, nor over the whole $\mathrm{Form}_2$ language with its local-object symbols forgotten afterwards. It is the following $\Sigma_1$ judgment fixed in \Cref{prop:theorem-a-boundary-typing}:
\[
\varphi(x)\in\Frm_1^{\mathrm{ref}}(s)
\quad\text{means}\quad
x:\mathsf{Ref}_s\vdash\varphi(x)\in\mathrm{Form}_1
\ \text{over}\ \Sigma_1^{\mathrm{ref}}(s)
\ \text{with no parameters},
\]
with $\mathsf{Ref}_s,\Adm_s\in\Sigma_1$ and
\[
\{\LocSec_s,\CompSec_s,\Sec_s\}
\cap \Sigma_1^{\mathrm{ref}}(s)=\varnothing .
\]
Consequently $\Null_s^{\mathrm{glue}}(x)$ is also outside $\Frm_1^{\mathrm{ref}}(s)$, because it is the Boolean combination $\Adm_s(x)\land\CompSec_s(x)\land\neg\Sec_s(x)$. Thus the theorem compares a well-typed reference-aware lower formula with a local-object target predicate; it does not quantify over a reference-free $\mathrm{Form}_1$ language and it does not obtain reference variables by taking a reduct of $\mathbb L_2$.
The statement is a theorem about this specific signature boundary. If references or admissibility were instead placed in $\Sigma_2\setminus\Sigma_1$, the lower fragment would be a different language and the theorem below would not be the comparison problem studied in this paper.
Every occurrence of the reference element in the comparison is an occurrence of the same semantic element in the retained $\Sigma_1$ domain:
\[
r\in D_{\mathsf{Ref}_s}^{N}
=D_{\mathsf{Ref}_s}^{M_i}
=D_{\mathsf{Ref}_s}^{U_{2,1}(M_i)}.
\]
Equivalently, throughout the theorem the lower assertion
\[
M_i,p\Vdash\varphi[r]
\]
is the conservative-reduct assertion
\[
U_{2,1}(M_i),\pi^{M_i}_{2,1}(p)\Vdash_1\varphi[r].
\]
The target assertions involving $\CompSec_s$, $\Sec_s$, and $\Null_s^{\mathrm{glue}}$ are the only assertions evaluated in the $\Sigma_2$ expansion.
The local-object expansions change the functor $F_{p,s}$ and hence the $\Sigma_2$ sectionability predicates; they do not change the type of $x$, the interpretation of $\Adm_s$, the state, or the semantic reference at which $\varphi(x)$ is evaluated.

More strongly, each target predicate fails to be definable even on one fixed-reduct comparison class in which the finite site and its covers are held fixed. Thus the theorem does not compare two lower reference elements, and it does not rely on hiding the reference from the lower language. The lower formulas are allowed to use the reference-aware resources actually present in $\Sigma_1$, including equality and $\Adm_s$; those resources are simply identical in all compared expansions. The proof fixes the lower reduct, the state, the admitted reference element, and the pretopological site, and varies only the local object functor forgotten by $U_{2,1}$. This fixed-class failure is enough to refute the global uniform definability notion because any global definition restricts to every fixed class by \Cref{lem:fixed-classes-test-uniform-definability}.

More precisely, there exist a $\Sigma_1$-reduct $N$, a singleton empty-context state $p$, a reference element
\[
r\in D_{\mathsf{Ref}_s}^{N},
\]
admitted in each expansion, and three $\Sigma_2$ local-object expansions $M_0,M_1,M_2$ of $N$ with the same site and covers such that, for every $i,j\in\{0,1,2\}$ and every parameter-free formula
\[
\varphi(x)\in\Frm_1^{\mathrm{ref}}(s),
\qquad x:\mathsf{Ref}_s,
\]
one has
\[
M_i,p\Vdash\varphi[r]
\quad\Longleftrightarrow\quad
M_j,p\Vdash\varphi[r],
\]
while the three expansions realize the sectionability pattern
\[
\begin{array}{c|cccc}
 & \Adm_s(r) & \CompSec_s(r) & \Sec_s(r) & \Null^{\mathrm{glue}}_s(r)\\ \hline
M_0 & \text{true} & \text{false} & \text{false} & \text{false}\\
M_1 & \text{true} & \text{true} & \text{true} & \text{false}\\
M_2 & \text{true} & \text{true} & \text{false} & \text{true}.
\end{array}
\]
The quantified formulas are well typed by \Cref{prop:theorem-a-boundary-typing,prop:no-reference-increment,prop:layer-typing-comparison-boundary,cor:boundary-correct-reference-formulas,lem:typed-comparison-boundary,lem:reference-data-survive-reduct,lem:no-type-shift-comparison-fragment}: the free variable $x:\mathsf{Ref}_s$ and the atom $\Adm_s(x)$ belong to $\Sigma_1$, while $\LocSec_s$, $\CompSec_s$, and $\Sec_s$ belong only to $\Sigma_2$. Consequently no parameter-free lower formula in $\Frm_1^{\mathrm{ref}}(s)$ can define any of the predicates
\[
\CompSec_s,
\qquad
\Sec_s,
\qquad
\Null^{\mathrm{glue}}_s
\]
on this fixed comparison class. A fortiori, by \Cref{lem:fixed-classes-test-uniform-definability}, no such formula defines any of them uniformly on admitted references over all local-object expansions in the sense of \Cref{def:lower-fragment-definability}.
\end{theorem}

\begin{proof}
We use the lower reduct, state, reference, and three expansions from \Cref{prop:fixed-lower-reduct-three-expansions}. They satisfy $U_{2,1}(M_i)=N$, use the same state $p$, and evaluate every formula at the same element $r\in D_{\mathsf{Ref}_s}^N$. Thus
\[
M_0,M_1,M_2\in\mathrm{Exp}_2(N,p,r)
\]
in the sense of \Cref{def:fixed-reduct-comparison-class}.

First check typing. By \Cref{prop:theorem-a-boundary-typing,prop:no-reference-increment,prop:layer-typing-comparison-boundary,cor:well-typed-lower-comparison,lem:reference-data-survive-reduct,lem:no-type-shift-comparison-fragment}, the displayed variable $x:\mathsf{Ref}_s$, the semantic evaluation at the reference element $r$, and the admissibility hypothesis $M_i,p\Vdash\Adm_s(r)$ all live at $\Sigma_1$ and are preserved by the $\mathbb L_2\to\mathbb L_1$ reduct. The target predicates being tested are $\Sigma_2$ predicates. Thus the assertion
\[
M_i,p\Vdash\varphi[r]
\]
is evaluated in the lower reduct, while
\[
M_i,p\Vdash P_s[r]
\]
for $P_s\in\{\CompSec_s,\Sec_s,\Null_s^{\mathrm{glue}}\}$ is evaluated in the local-object expansion. This is the intended cross-layer definability question; it is not an attempt to place $P_s$ inside $\Frm_1^{\mathrm{ref}}(s)$.
Concretely, the domain equality
\[
D_{\mathsf{Ref}_s}^{N}
=D_{\mathsf{Ref}_s}^{M_i}
=D_{\mathsf{Ref}_s}^{U_{2,1}(M_i)}
\]
and the equality of the $\Adm_s$ interpretations are part of the fixed reduct. Hence the pointed state $p[r/x]$ is a $\Sigma_1$ pointed state common to all three expansions.

The expansions have exactly the same $\Sigma_1$ reduct, the same state, the same reference element $r$, and even the same site and covers; the same-reference fixed-site package is \Cref{lem:same-reference-fixed-site-separation}. By \Cref{cor:reference-aware-resources-fixed}, this remains true even for lower formulas that use reference equality, $\Adm_s$, and quantification over the reference sort. In the concrete construction only the local object functor, and hence the induced $\Sigma_2$ sectionability predicates, varies. Thus
\[
M_i,p\Vdash\varphi[r]
\quad\Longleftrightarrow\quad
M_j,p\Vdash\varphi[r]
\]
for every lower formula $\varphi(x)\in\Frm_1^{\mathrm{ref}}(s)$ and every $i,j$. The displayed sectionability table is the target-predicate part of \Cref{lem:same-reference-fixed-site-separation}.

Apply \Cref{lem:fixed-reduct-variation-obstruction} three times on the same fixed comparison class. The pair $(M_0,M_1)$ rules out a definition of $\CompSec_s$, because $\CompSec_s(r)$ changes from false to true while every lower formula keeps the same truth value. The pair $(M_1,M_2)$ rules out a definition of $\Sec_s$, because $\Sec_s(r)$ changes from true to false while every lower formula again stays fixed. Finally, $r$ is admitted in both $M_1$ and $M_2$; \Cref{lem:admitted-gluing-null-reduction} identifies
\[
\Null^{\mathrm{glue}}_s(r)
\quad\text{with}\quad
\CompSec_s(r)\land \neg\Sec_s(r)
\]
under that fixed admission hypothesis. Hence $\Null^{\mathrm{glue}}_s(r)$ is false in $M_1$ and true in $M_2$, and the same obstruction lemma rules out a lower-fragment definition of $\Null^{\mathrm{glue}}_s$.

These applications prove non-definability in the fixed-reduct comparison class. By \Cref{lem:fixed-classes-test-uniform-definability}, any formula satisfying the global uniform definability condition of \Cref{def:lower-fragment-definability} would restrict to a definition on this fixed class $\mathrm{Exp}_2(N,p,r)$. The fixed-reduct obstructions therefore rule out uniform definability over all local-object expansions.
\end{proof}
