\section{Computational certification and audited numerical data}
\label{app:certification}

Section~\ref{sec:chebotarev} uses two kinds of input in the range
\(q=9,\dots,17\): exact integer recurrences for the collision moments
\(S_q(m)\), and finite certificate tables extracted from those recurrence
polynomials.  We also record, separately and with explicit status labels,
audited high-precision secondary spectral fingerprints.  Since all of these
data enter the audited arithmetic window of Section~\ref{sec:chebotarev}, we
make their provenance explicit here.  The present file bundle contains the
exact tables used in the text, but not yet a standalone verification archive
with source code and raw exported certificates; accordingly
Section~\ref{sec:chebotarev} treats this material as audited computation rather
than as part of the formal theorem chain.

For each \(m\ge 0\), let \(c_m(r)\) denote the number of binary words
\(\omega\in\{0,1\}^m\) whose Fibonacci-weighted sum is congruent to
\(r\bmod F_{m+2}\).  Then
\[
  S_q(m)=\sum_{r\bmod F_{m+2}} c_m(r)^q.
\]
Thus \(S_q(m)\) can be computed exactly by modular dynamic programming on
\(\ZZ/F_{m+2}\ZZ\), without enumerating all \(2^m\) words.  The recurrence
polynomials \(\Pi_q\) used in Section~\ref{sec:chebotarev} are defined from the
exact coefficients listed in Table~\ref{tab:appendix-recurrences-q9-17}.

\medskip
\noindent\textbf{Provenance of the certificate tables.}
The tables above summarize an exact arithmetic audit of the sequences
\((S_q(m))_{m\ge 0}\) in the finite window \(q=9,\dots,17\).  That audit
computes the recurrences in Table~\ref{tab:appendix-recurrences-q9-17},
integral bases of the null modules \(\mathrm{NullModes}(q)\), and the
basis-change matrices whose Smith normal forms are recorded in
\Cref{tab:appendix-principalization-q9-17}.  The underlying code and raw
machine-readable exports are not bundled with the present submission, so the
appendix records only the exact data extracted from that audit.

\medskip
\noindent\textbf{Role in the paper.}
Section~\ref{sec:chebotarev} uses only the exact polynomials \(\Pi_q\)
determined by Table~\ref{tab:appendix-recurrences-q9-17}, the
Smith-normal-form certificates in
\Cref{tab:appendix-principalization-q9-17}, and the finite mod-\(p\)
factorization and discriminant-squareclass patterns in
\Cref{tab:galois-certificates-q9-17,tab:disc-squareclass-q12-15}.  More
precisely, \Cref{obs:principalization-window} uses
Tables~\ref{tab:appendix-recurrences-q9-17}
and~\ref{tab:appendix-principalization-q9-17},
\Cref{obs:galois-window} uses
Tables~\ref{tab:appendix-recurrences-q9-17}
and~\ref{tab:galois-certificates-q9-17}, and
\Cref{obs:linear-disjointness-window} uses
Table~\ref{tab:disc-squareclass-q12-15} together with
\Cref{obs:galois-window}.  The high-precision root fingerprints recorded in
\Cref{tab:secondary-spectral-q9-17} are included only as descriptive audited
data.

\begin{remark}[Status of the secondary spectral pattern]
\label{rem:secondary-spectral-pattern}
Table~\ref{tab:secondary-spectral-q9-17} records high-precision approximations
to the dominant root \(\lambda_q\), the outer negative real root
\(\lambda_q^-\), and the largest remaining modulus \(\Theta_q\) for
\(q=9,\dots,17\).  Throughout the displayed window the numerical fingerprints
are consistent with a unique negative real secondary root satisfying
\[
  \Theta_q<|\lambda_q^-|<\lambda_q.
\]
If this ordering is upgraded to a symbolic root-isolation certificate, then the
usual linear-recurrence expansion would yield an eventual alternating
second-order term for \(S_q(m)\).  At present, however, these data remain
audited descriptive fingerprints rather than symbolic isolation intervals, and
we therefore keep them outside the formal theorem chain.
\end{remark}

\begin{remark}[What remains open]
\label{rem:q16q17}
The full symmetric-group statement already closes for \(q=16\) and \(q=17\).
The next arithmetic target is to extend squareclass independence and linear
disjointness beyond the block \(q=12,13,14,15\).  On the spectral side, the
table above strongly suggests a persistent negative second root throughout the
audited window, but an entirely symbolic certification of that pattern would
be preferable.
\end{remark}
