\section{Fixed-response locus theorem for one calibrated response function}\label{sec:kms-shells}

The record theory contributes two selected rates,
\[
\gamma_\varphi=\frac{\log\varphi}{\Delta\tau},
\qquad
\gamma_{\mathrm{mem}}=\Gamma_c(\kappa_{\mathrm r},\Delta\tau),
\]
and this section inserts them into one fixed calibrated response function
$R_\Omega$. Sections~2--3 contain the main classical-counter theorem chain;
this section is only the shorter conditional response-side lift. Its output is
one fixed-response locus theorem for that prescribed $R_\Omega$. The
threshold-specific response-locus corollaries and the later homogeneous
toy-model specializations are collected in the appendices. Shell language is
used only when the target lies in the range of $R_\Omega$ and the regularity
condition $dR_\Omega\neq0$ is verified. Appendix
Table~\ref{tab:hypothesis-ledger} collects the standing response-side
hypotheses, so the prose here records only the response-theoretic input not
already fixed by the stochastic analysis. From this point onward we also follow
\Cref{rem:notation-layers-main}: the record shorthands $p$, $c$, and
$\lambda_{\mathrm{hid}}$ stay in \Cref{sec:records}, while the response layer
uses $\Gamma_\pm$, $\beta_\infty$, $\beta_{\mathrm{loc}}$, $\lambda_c$, and
$\lambda_{\mathrm{sw}}$. Tables~\ref{tab:notation-response}
and~\ref{tab:notation-inverse} collect the response-side and later inverse-side
symbols reused most often from this point on.
The response-side contract is fixed once and for all by the boxed Model~A
assumption below. The response calculations in this section use that contract
only to insert the record-selected target rates into one chosen stationary
response profile; they do not derive the sampled instrument from UDW
measurement theory.

\paragraph{Logical order for the response-side material.}
The response-side material is a conditional corollary chain rather than a new
detector-dynamics theory.
\begin{enumerate}[label=\textup{(\roman*)},leftmargin=2.2em,itemsep=0.25em]
\item \Cref{prop:stationary-udw-response-main,lem:local-kms-spectral-main}
record the standard stationary UDW input at one fixed gap and extract one fixed
calibrated response profile $R_\Omega$.
\item \Cref{lem:one-gap-kms-underdetermination-main} shows that Tolman scaling
and one-gap KMS detailed balance do not determine the positive excitation
profile, so the calibrated $R_\Omega$ must be specified independently.
\item \Cref{thm:response-locus-main} is the central response-side theorem: once
$R_\Omega$ is fixed, the record-selected targets $\gamma_\varphi$ and
$\gamma_{\mathrm{mem}}$ become admissible response targets, and shell language
for the corresponding level sets is justified only after the separate range and
regularity checks.
\item Appendix~\ref{app:abstract-homogeneous-main} then packages the actual
target substitutions as conditional corollaries of that theorem.
\item Every later homogeneous, near-horizon, and fixed-radius inverse
statement is an additional algebraic specialization of that fixed-response
theorem, or of the pair transport law
\Cref{cor:spectral-transport-main}; none of those later statements is a new
exact nonconstant-lapse detector theorem. They are collected separately in the
abstract calibrated toy-model Appendices~\ref{app:abstract-homogeneous-main}
and~\ref{sec:inverse}, together with Appendix~\ref{app:schwarzschild-toy}.
\end{enumerate}

\paragraph{Standing convention for this response section.}
Unless a later statement explicitly adds further input, the response-side

\paragraph{Comparator boundary for the response-locus theorem.}
The theorem family proved in this section is narrower than the recent
stationary-trajectory classification theorem of Bunney
\cite{Bunney2024StationaryTraj}. Bunney classifies stationary worldlines. Here
the stationary worldline, the stationary detector-response formula, and the
gap-regular response input are all fixed first, and the mathematical step is
then only target-rate level-set algebra for one calibrated response function
$R_\Omega$. The increment is therefore a conditional response-locus theorem,
with shell language added only after the range and regularity checks of
\Cref{thm:response-locus-main}.

Let $N(x)$ be the lapse function of a static spacetime and assume the field is in a global $\beta_\infty$-KMS state for Killing time. Along a static worldline at position $x$, Tolman scaling gives \cite{Tolman1930,TolmanEhrenfest1930}
\begin{equation}\label{eq:tolman-main}
\beta_{\mathrm{loc}}(x)=\beta_\infty N(x).
\end{equation}
This relation fixes the proper-time KMS parameter seen by a static detector. It does not say that every local response observable is a function of $N(x)$ alone; mode density, boundary conditions, detector profile, and greybody transmission can still enter the absolute transition rate.

For a static worldline, the weak-coupling transition rates are Fourier transforms of the pulled-back two-point function. We use this standard UDW response fact explicitly rather than absorbing it into a response ansatz. Let $\Phi_x(\tau)$ denote the field operator smeared with the detector profile and transported along the static orbit through $x$, and set
\[
W_x(s)=\omega\!\left(\Phi_x(\tau)\Phi_x(\tau-s)\right),
\qquad
\widehat W_x(\nu)=\int_{\RR}e^{-i\nu s}W_x(s)\,ds,
\]
with the Fourier transform understood distributionally whenever $W_x$ is not an $L^1$ function.
Because $W_x$ is of positive type, its Fourier transform is represented by a
positive tempered measure $\mu_x$:
\[
\langle \widehat W_x,\phi\rangle=\int_{\RR}\phi(\nu)\,d\mu_x(\nu)
\qquad (\phi\in\mathcal S(\RR)).
\]
The response results below use only local gap regularity: there are open
neighborhoods of $\pm\Omega$ on which
\[
d\mu_x(\nu)=\widehat W_x(\nu)\,d\nu
\]
with $\widehat W_x$ finite and continuous. Equivalently, the spectral measure
has no atomic or singular component at either gap frequency.
The response-side input is therefore the standard stationary detector setup
rather than a new regularization scheme: stationary trajectories in the sense of
Letaw \cite{Letaw1981}, finite-size or smooth-switching regularizations in the
sense of Schlicht, Satz, and Louko--Satz
\cite{Schlicht2004,Satz2007,LoukoSatz2006,LoukoSatz2008}, and then the local
gap evaluation needed for the fixed calibrated response profile used below.

\begin{assumption}[Switched stationary response input, gap-regular spectrum, and KMS strip regularity]\label{ass:long-time-switch-main}
Fix a static worldline through $x$, a detector gap $\Omega>0$, and a smooth nonnegative switching profile $\chi\in C_c^\infty(\mathbb R)$ with $\int_{\RR}\chi=1$. Let $\chi_T(t)=\chi(t/T)$ and let $H_{g\to e}^{(T)}(x,\Omega)$ and $H_{e\to g}^{(T)}(x,\Omega)$ be the corresponding first-order transition probabilities for a detector with fixed spatial profile and bare coupling $\lambda_c$. Assume:
\begin{enumerate}[label=\textup{(\roman*)},leftmargin=2.2em,itemsep=0.25em]
\item The pullback $W_x$ is a tempered distribution on $\RR$, and there is an open interval $I_x\subset\RR$ containing $\pm\Omega$ and a continuous finite function, still denoted $\widehat W_x:I_x\to[0,\infty)$, such that the restriction of the spectral distribution to $I_x$ is the absolutely continuous measure $\widehat W_x(\nu)\,d\nu$. Equivalently, near $\pm\Omega$ the pulled-back spectrum is absolutely continuous with continuous density $\widehat W_x$, so there is no singular part and in particular no spectral atom at either $\pm\Omega$.
\item The switched first-order transition probabilities are given by the standard stationary UDW response pairing
\begin{equation}\label{eq:switched-pairing-main}
H_{g\to e}^{(T)}(x,\Omega)
=\lambda_c^2\langle W_x,K_{T,\Omega}\rangle,
\qquad
H_{e\to g}^{(T)}(x,\Omega)
=\lambda_c^2\langle W_x,K_{T,-\Omega}\rangle,
\end{equation}
where
\[
K_{T,\omega}(s)=e^{-i\omega s}\int_{\RR}\chi_T(u)\chi_T(u-s)\,du.
\]
Equivalently, whenever $W_x$ is represented by a function, this is the usual double integral
\begin{equation}\label{eq:switched-double-integral-main}
H_{g\to e}^{(T)}(x,\Omega)
=\lambda_c^2\int_{\RR^2}
\chi_T(\tau)\chi_T(\tau')e^{-i\Omega(\tau-\tau')}W_x(\tau-\tau')\,d\tau\,d\tau',
\end{equation}
with the de-excitation formula obtained by replacing $\Omega$ by $-\Omega$.
\item If the state is $\beta_\infty$-KMS with respect to static Killing time, then with $\beta_{\mathrm{loc}}(x)=\beta_\infty N(x)$ there exists a function $F_x$ holomorphic on the strip
\[
S_{\beta_{\mathrm{loc}}}^-=\{z\in\mathbb C:-\beta_{\mathrm{loc}}(x)<\Im z<0\}
\]
such that for every closed substrip $-\beta_{\mathrm{loc}}(x)+\delta\le \Im z\le -\delta$ there exist $m_\delta\in\mathbb N$ and $C_\delta>0$ with
\[
|F_x(z)|\le C_\delta(1+|\Re z|)^{m_\delta},
\]
and the boundary values satisfy
\[
\lim_{\epsilon\to0^+}F_x(s-i\epsilon)=W_x(s),\qquad
\lim_{\epsilon\to0^+}F_x(s-i\beta_{\mathrm{loc}}(x)+i\epsilon)=W_x(-s)
\]
in $\mathcal S'(\RR)$.
\end{enumerate}
Define the switched effective coupling\footnote{The original bare coupling is $\lambda_c$; the switching convention contributes the fixed factor $\|\chi\|_2^2:=\int_{\mathbb R}\chi(u)^2\,du>0$.} by
\[
\lambda_{\mathrm{sw}}^2:=\lambda_c^2\,\|\chi\|_2^2
\]
and use $\lambda_{\mathrm{sw}}$ in the long-time rate formulas below.
\end{assumption}

\begin{lemma}[Approximate-identity limit at a gap-regular absolutely continuous spectral point]\label{lem:gap-regular-approx-identity-main}
Fix $\omega\in\RR$ and define
\begin{equation}\label{eq:approx-identity-main}
\Phi_{T,\omega}(\nu)=
\frac{T}{2\pi\|\chi\|_2^2}\,
\bigl|\widehat\chi\bigl(T(\omega-\nu)\bigr)\bigr|^2 .
\end{equation}
Then $\Phi_{T,\omega}$ is a nonnegative approximate identity centered at
$\omega$. If $\mu$ is a positive tempered measure on $\RR$ and there is an open
interval $I\ni\omega$ together with a continuous finite density $f:I\to[0,\infty)$
such that
\[
\mu|_I=f(\nu)\,d\nu,
\]
then
\[
\langle \mu,\Phi_{T,\omega}\rangle\longrightarrow f(\omega)
\qquad (T\to\infty).
\]
Equivalently, the switched kernel extracts the absolutely continuous density at
the gap provided there is no spectral atom or singular part in a neighborhood of
$\omega$.
\end{lemma}

\begin{proof}
The functions $\Phi_{T,\omega}$ are nonnegative and
\[
\int_{\RR}\Phi_{T,\omega}(\nu)\,d\nu
=\frac{1}{2\pi\|\chi\|_2^2}\int_{\RR}|\widehat\chi(\zeta)|^2\,d\zeta
=1
\]
by the change of variables $\zeta=T(\omega-\nu)$ and Plancherel. Thus
$\Phi_{T,\omega}$ is an approximate identity centered at $\omega$. Choose
$\eta\in C_c^\infty(I)$ with $\eta\equiv1$ on a smaller neighborhood of
$\omega$. Then
\[
\langle \mu,\Phi_{T,\omega}\rangle
=\int_{\RR}\eta(\nu)f(\nu)\Phi_{T,\omega}(\nu)\,d\nu
+\langle \mu,(1-\eta)\Phi_{T,\omega}\rangle.
\]
Because $\eta f$ is continuous near $\omega$, the first term tends to
$f(\omega)$. Because $\widehat\chi$ is Schwartz and $(1-\eta)$ vanishes near
$\omega$, the second term tends to $0$ in $\mathcal S(\RR)$. More explicitly,
fix the Schwartz seminorms
\[
p_{m,k}(\psi)=\sup_{\nu\in\RR}(1+|\nu|)^m\bigl|\partial_\nu^k\psi(\nu)\bigr|,
\qquad m,k\in\mathbb N_0.
\]
Choose $\delta>0$ so that $(1-\eta)(\nu)=0$ for $|\nu-\omega|<\delta$. For each
$k$, differentiating $\Phi_{T,\omega}$ gives a finite linear combination of
terms of the form
\[
T^{j+1}
\widehat\chi^{(\ell)}\!\bigl(T(\omega-\nu)\bigr)\,
\overline{\widehat\chi^{(\ell')}\!\bigl(T(\omega-\nu)\bigr)},
\qquad 0\le j,\ell,\ell'\le k.
\]
Hence, on $\operatorname{supp}(1-\eta)$ where $|T(\omega-\nu)|\ge \delta T$,
the Schwartz decay of $\widehat\chi$ implies that for every $L\in\mathbb N$
there is a constant $C_{m,k,L,\eta}$ such that
\[
p_{m,k}\bigl((1-\eta)\Phi_{T,\omega}\bigr)\le C_{m,k,L,\eta}\,T^{-L}.
\]
Thus $(1-\eta)\Phi_{T,\omega}\to0$ in every Schwartz seminorm. Since $\mu$ is a
tempered measure, there exist $C>0$ and finitely many seminorm indices
$(m_r,k_r)$ such that
\[
\bigl|\langle \mu,\psi\rangle\bigr|
\le C\sum_r p_{m_r,k_r}(\psi)
\qquad (\psi\in\mathcal S(\RR)).
\]
Applying this to $\psi=(1-\eta)\Phi_{T,\omega}$ gives
\[
\langle \mu,(1-\eta)\Phi_{T,\omega}\rangle\longrightarrow0.
\]
Therefore
\[
\langle \mu,\Phi_{T,\omega}\rangle\longrightarrow f(\omega).
\]
\end{proof}

\begin{proposition}[Stationary UDW response at the detector gap]\label{prop:stationary-udw-response-main}
Fix a static worldline through $x$ and a two-level detector with gap $\Omega>0$, coupling scale $\lambda_c$, and fixed spatial profile. Assume \Cref{ass:long-time-switch-main}. In particular, on some open interval containing $\pm\Omega$ the pulled-back spectrum is absolutely continuous with finite continuous density $\widehat W_x$ and has no atom or singular part at $\pm\Omega$. Then the stationary weak-coupling long-time excitation and de-excitation rates are
\begin{equation}\label{eq:long-time-kernel-main}
\frac{1}{T}H_{g\to e}^{(T)}(x,\Omega)
=\lambda_c^2\|\chi\|_2^2\,\langle \widehat W_x,\Phi_{T,\Omega}\rangle,
\qquad
\frac{1}{T}H_{e\to g}^{(T)}(x,\Omega)
=\lambda_c^2\|\chi\|_2^2\,\langle \widehat W_x,\Phi_{T,-\Omega}\rangle,
\end{equation}
where $\Phi_{T,\omega}$ is the approximate-identity family from
\eqref{eq:approx-identity-main}. Since the spectral measure is gap-regular and
absolutely continuous at $\pm\Omega$, with no atom or singular part there,
\Cref{lem:gap-regular-approx-identity-main} gives
\begin{equation}\label{eq:long-time-limit-main}
\begin{aligned}
\Gamma_+(x,\Omega)
&:=\lim_{T\to\infty}\frac{1}{T}H_{g\to e}^{(T)}(x,\Omega)
=\lambda_c^2\|\chi\|_2^2\,\widehat W_x(\Omega),\\
\Gamma_-(x,\Omega)
&:=\lim_{T\to\infty}\frac{1}{T}H_{e\to g}^{(T)}(x,\Omega)
=\lambda_c^2\|\chi\|_2^2\,\widehat W_x(-\Omega).
\end{aligned}
\end{equation}
Hence, with
\[
\lambda_{\mathrm{sw}}^2=\lambda_c^2\|\chi\|_2^2,
\]
one obtains
\begin{equation}\label{eq:fourier-rates-main}
\Gamma_+(x,\Omega)=\lambda_{\mathrm{sw}}^2\widehat W_x(\Omega),
\qquad
\Gamma_-(x,\Omega)=\lambda_{\mathrm{sw}}^2\widehat W_x(-\Omega).
\end{equation}
Thus the long-time rate is obtained by extracting the absolutely continuous
spectral density at the detector gap; it is not a formal evaluation of a
distribution at $\nu=\pm\Omega$.
\end{proposition}

\begin{proof}
Write
\[
g_T(s)=\int_{\RR}\chi_T(u)\chi_T(u-s)\,du.
\]
Then \Cref{ass:long-time-switch-main}(ii) gives
\[
H_{g\to e}^{(T)}(x,\Omega)=\lambda_c^2\langle W_x,e^{-i\Omega(\cdot)}g_T\rangle.
\]
Because $g_T$ is the autocorrelation of $\chi_T$, its Fourier transform is
\[
\widehat g_T(\nu)=|\widehat{\chi_T}(\nu)|^2
=T^2\bigl|\widehat\chi(T\nu)\bigr|^2.
\]
Using the Fourier inversion convention
\[
g(s)=\frac{1}{2\pi}\int_{\RR}e^{i\nu s}\widehat g(\nu)\,d\nu,
\]
we obtain the exact spectral representation
\[
\frac{1}{T}H_{g\to e}^{(T)}(x,\Omega)
=\frac{\lambda_c^2}{2\pi T}
\left\langle \widehat W_x,\bigl|\widehat{\chi_T}(\Omega-\cdot)\bigr|^2\right\rangle
=\lambda_c^2\|\chi\|_2^2\,\langle \widehat W_x,\Phi_{T,\Omega}\rangle,
\]
which is \eqref{eq:long-time-kernel-main}; the de-excitation formula is the same
with $\Omega$ replaced by $-\Omega$.
\Cref{lem:gap-regular-approx-identity-main} applied to the positive tempered
spectral measure $\mu_x$ at $\omega=\Omega$ and $\omega=-\Omega$ yields
\eqref{eq:long-time-limit-main}, and therefore
\eqref{eq:fourier-rates-main}. This is the exact long-time normalization used in
the standard smooth-switching stationary detector theorem
\cite{LoukoSatz2008,FewsterJuarezLouko2016}.
\end{proof}

\begin{lemma}[Local KMS spectral relation near the detector gap]\label{lem:local-kms-spectral-main}
Assume \Cref{ass:long-time-switch-main}, let the state be $\beta_\infty$-KMS
with respect to static Killing time, and abbreviate
\[
\beta_{\mathrm{loc}}=\beta_\infty N(x).
\]
Let $I\subset I_x$ be any open interval containing $\pm\Omega$. Then for every
$\phi\in C_c^\infty(I)$,
\begin{equation}\label{eq:kms-spectral-distribution-main}
\langle \widehat W_x,\phi\rangle
=\left\langle \widehat W_x(-\cdot),
e^{-\beta_{\mathrm{loc}}(\cdot)}\phi(\cdot)\right\rangle.
\end{equation}
If, moreover, $\widehat W_x$ is represented by a continuous density on a
neighborhood of $\pm\Omega$, then
\begin{equation}\label{eq:kms-detailed-balance-main}
\widehat W_x(-\Omega)=e^{\beta_{\mathrm{loc}}\Omega}\widehat W_x(\Omega).
\end{equation}
\end{lemma}

\begin{proof}
Let $\phi\in C_c^\infty(I)$. Define the entire function
\[
h_\phi(z)=\int_{\RR}e^{-i\nu z}\phi(\nu)\,d\nu.
\]
Because $\phi$ has compact support, $h_\phi$ decays faster than any inverse power uniformly on closed horizontal strips. Together with the polynomial bounds in \Cref{ass:long-time-switch-main}(iii), Cauchy's theorem on rectangles in $S_{\beta_{\mathrm{loc}}}^-$ shows that
\[
\int_{\RR}F_x(s-i\eta)\,h_\phi(s-i\eta)\,ds
\]
is independent of $\eta\in(0,\beta_{\mathrm{loc}})$. Taking $\eta\downarrow0$ and $\eta\uparrow\beta_{\mathrm{loc}}$, and using the distributional boundary values from \Cref{ass:long-time-switch-main}(iii), yields
\[
\langle W_x,h_\phi\rangle
=
\langle W_x(-\cdot),h_\phi(\cdot-i\beta_{\mathrm{loc}})\rangle.
\]
Since
\[
h_\phi(s-i\beta_{\mathrm{loc}})
=\int_{\RR}e^{-i\nu s}e^{-\beta_{\mathrm{loc}}\nu}\phi(\nu)\,d\nu
=h_{e^{-\beta_{\mathrm{loc}}(\cdot)}\phi}(s),
\]
the preceding identity becomes
\[
\langle \widehat W_x,\phi\rangle
=
\langle \widehat{W_x(-\cdot)},e^{-\beta_{\mathrm{loc}}(\cdot)}\phi\rangle.
\]
Using $\widehat{W_x(-\cdot)}(\nu)=\widehat W_x(-\nu)$ distributionally, we obtain
\[
\langle \widehat W_x,\phi\rangle
=
\langle e^{-\beta_{\mathrm{loc}}(\cdot)}\widehat W_x(-\cdot),\phi\rangle
\]
for all $\phi\in C_c^\infty(I)$, which is exactly
\eqref{eq:kms-spectral-distribution-main}. If $\widehat W_x$ is represented by
continuous densities on a neighborhood of $\pm\Omega$, choose a standard
approximate delta family $(\phi_\epsilon)_{\epsilon>0}\subset C_c^\infty(I)$
with $\operatorname{supp}\phi_\epsilon\to\{\Omega\}$. Then
\eqref{eq:kms-spectral-distribution-main} gives
\[
\int \widehat W_x(\nu)\phi_\epsilon(\nu)\,d\nu
=
\int \widehat W_x(-\nu)e^{-\beta_{\mathrm{loc}}\nu}\phi_\epsilon(\nu)\,d\nu.
\]
Letting $\epsilon\downarrow0$ and using continuity at $\pm\Omega$ yields
\eqref{eq:kms-detailed-balance-main}. For operator-algebraic formulations of
the same KMS strip and spectral relation, see
\cite{HaagHugenholtzWinnink1967,KayWald1991,BratteliRobinson1997}.
\end{proof}

Combining \eqref{eq:fourier-rates-main} with
\Cref{lem:local-kms-spectral-main} gives the one-gap detailed-balance ratio
\[
\frac{\Gamma_-(x,\Omega)}{\Gamma_+(x,\Omega)}
=e^{\beta_{\mathrm{loc}}\Omega},
\]
with our convention that $\Gamma_+$ is excitation and $\Gamma_-$ is
de-excitation. If, moreover, the one-gap density is strictly positive,
$\widehat W_x(\Omega)>0$, define the positive spectral weight
\begin{equation}\label{eq:spectral-weight-def-main}
\mathcal S_x(\Omega)=
\bigl(e^{\beta_\infty N(x)\Omega}-1\bigr)\widehat W_x(\Omega).
\end{equation}
Then
\begin{equation}\label{eq:kms-rate-main}
\Gamma_+(x,\Omega)=
\lambda_{\mathrm{sw}}^2\,
\frac{\mathcal S_x(\Omega)}{e^{\beta_\infty N(x)\Omega}-1},
\qquad
\Gamma_-(x,\Omega)=
\lambda_{\mathrm{sw}}^2\,
\frac{e^{\beta_\infty N(x)\Omega}\mathcal S_x(\Omega)}
{e^{\beta_\infty N(x)\Omega}-1}.
\end{equation}

Thus the KMS condition fixes only the detailed-balance ratio above, with our
convention that $\Gamma_+$ is excitation and $\Gamma_-$ is de-excitation. It
does not, by itself, fix the absolute magnitude of the excitation rate. The
remaining positive factor is a local response spectral weight. We therefore
keep a separate regularity and calibration hypothesis for this factor rather
than treating it as a geometric identity.

\begin{assumption}[Static response regularity and calibration]\label{ass:kms-response-main}
Fix a static region $U$ and a detector with positive gap $\Omega$ and fixed spatial profile. For each static worldline $x\in U$, use the stationary weak-coupling rates supplied by \Cref{prop:stationary-udw-response-main}. Assume explicitly that the pulled-back spectral measure is gap-regular at $\pm\Omega$: on some open interval containing $\pm\Omega$ it is absolutely continuous with continuous finite density $\widehat W_x$, hence has no atom at $\pm\Omega$, and this one-gap density satisfies
\[
\widehat W_x(\Omega)>0.
\]
Define $\mathcal S_x(\Omega)$ by \eqref{eq:spectral-weight-def-main}. The
response model fixes the positive spectral-weight profile
$x\mapsto\mathcal S_x(\Omega)$ on $U$. Whenever the response-level theorem
chain below is invoked, this calibrated profile is assumed to be $C^1$ on the
relevant region, so that the corresponding fixed response function
\[
R_\Omega(x)=
\lambda_{\mathrm{sw}}^2\,
\frac{\mathcal S_x(\Omega)}{e^{\beta_\infty N(x)\Omega}-1}
\]
is a positive $C^1$ function there.
\end{assumption}

\begin{proposition}[Exact homogeneous model from spatial symmetry]\label{prop:symmetry-homogeneous-main}
Assume, in addition to \Cref{ass:long-time-switch-main}, that the static region is an ultrastatic spacetime $(\mathbb R\times\Sigma,-dt^2+h)$ and that a group $G$ of spatial isometries acts transitively on a region $U\subset\Sigma$. Suppose the $\beta_\infty$-KMS state $\omega$ is invariant under the induced automorphisms $\alpha_g$ of the field algebra, and that the detector profile at $x=gx_0$ is obtained from one reference profile by rigid transport under $g$. Then
\[
W_x(s)=W_{x_0}(s),\qquad
\widehat W_x(\nu)=\widehat W_{x_0}(\nu),\qquad
\mathcal S_x(\Omega)=\mathcal S_{x_0}(\Omega)
\]
for all $x\in U$. Hence the homogeneous spectral-weight regime holds exactly on $U$.

In particular, for a thermal Klein--Gordon KMS state in Minkowski space with identical translated compactly supported smearing profiles, the one-gap spectral weight is exactly independent of the detector position.
\end{proposition}

\begin{proof}
Let $x=gx_0$ with $g\in G$. By construction of the transported detector profile, the smeared detector observable satisfies
\[
\Phi_x(\tau)=\alpha_g\!\bigl(\Phi_{x_0}(\tau)\bigr).
\]
Using the $G$-invariance of the state,
\[
\begin{aligned}
W_x(s)
&=\omega\!\left(\Phi_x(\tau)\Phi_x(\tau-s)\right) \\
&=\omega\!\left(\alpha_g(\Phi_{x_0}(\tau))\,\alpha_g(\Phi_{x_0}(\tau-s))\right) \\
&=\omega\!\left(\alpha_g(\Phi_{x_0}(\tau)\Phi_{x_0}(\tau-s))\right) \\
&=\omega\!\left(\Phi_{x_0}(\tau)\Phi_{x_0}(\tau-s)\right)
=W_{x_0}(s).
\end{aligned}
\]
Taking Fourier transforms gives $\widehat W_x(\nu)=\widehat W_{x_0}(\nu)$, and then \eqref{eq:spectral-weight-def-main} gives $\mathcal S_x(\Omega)=\mathcal S_{x_0}(\Omega)$. In Minkowski thermal equilibrium the spatial translation group is transitive, the thermal state is translation invariant, and rigidly translated detector profiles satisfy the stated covariance, so the same conclusion applies.
\end{proof}

In this exact symmetry example the ultrastatic lapse satisfies $N\equiv 1$ on
$U$, so the fixed calibrated response function $R_\Omega$ is constant on $U$ as
well. The corresponding level sets are therefore either empty or all of $U$.
Proposition~\ref{prop:symmetry-homogeneous-main} verifies only the homogeneous
amplitude statement $\mathcal S_x(\Omega)\equiv\mathcal S(\Omega)$; nontrivial
homogeneous shell geometry still requires a separate calibrated model with
nonconstant lapse.

Under \Cref{ass:kms-response-main}, \eqref{eq:kms-rate-main} is used only in the
forward direction needed for shell analysis: a response model fixes the
positive-frequency pullback, then \eqref{eq:spectral-weight-def-main} defines
the spectral weight, and only afterwards are the record-selected target rates
imposed. Read backwards, the same identity merely parametrizes an arbitrary
positive excitation profile with the same detailed-balance ratio:
\[
\mathcal S_x(\Omega)=
\lambda_{\mathrm{sw}}^{-2}\bigl(e^{\beta_\infty N(x)\Omega}-1\bigr)\Gamma_+(x,\Omega).
\]
Changing that calibrated amplitude changes the response loci while preserving the
one-gap KMS ratio, so from here on we simply freeze the response function
\begin{equation}\label{eq:response-function-main}
R_\Omega(x)=
\lambda_{\mathrm{sw}}^2\,
\frac{\mathcal S_x(\Omega)}{e^{\beta_\infty N(x)\Omega}-1}.
\end{equation}
Proposition~\ref{prop:symmetry-homogeneous-main} supplies one exact
detector-field model where the homogeneous specialization
$\mathcal S_x(\Omega)\equiv\mathcal S(\Omega)$ holds, although there the lapse
is constant. The nonconstant-lapse homogeneous case is used only in the
appendices, as abstract calibrated toy-model algebra for this fixed response
function.

\begin{lemma}[One-gap KMS underdetermination]\label{lem:one-gap-kms-underdetermination-main}
At the fixed-gap rate level used in this section, Tolman scaling and KMS detailed balance do not determine the positive excitation-rate profile. Let $G:U\to(0,\infty)$ be any $C^1$ function. The rates
\[
\Gamma_+^G(x,\Omega)=G(x),
\qquad
\Gamma_-^G(x,\Omega)=e^{\beta_\infty N(x)\Omega}G(x)
\]
satisfy the one-gap detailed-balance relation
\[
\Gamma_-^G(x,\Omega)/\Gamma_+^G(x,\Omega)
=e^{\beta_\infty N(x)\Omega},
\]
and can be written in the form \eqref{eq:kms-rate-main} with
\[
\mathcal S_x^G(\Omega)=
\lambda_{\mathrm{sw}}^{-2}\bigl(e^{\beta_\infty N(x)\Omega}-1\bigr)G(x)>0 .
\]
Consequently, changing the positive spectral weight changes the target-rate loci while preserving the same one-gap detailed-balance ratio. The shell statements below are therefore conditional statements for a fixed response function $R_\Omega$, or for an additional calibration such as the homogeneous spectral-weight regime. This is a fixed-frequency underdetermination statement, not an existence theorem asserting that every such $G$ extends to a global QFT two-point function.
\end{lemma}

\begin{proof}
The displayed ratio of $\Gamma_-^G$ and $\Gamma_+^G$ is immediate. Substituting the displayed $\mathcal S_x^G(\Omega)$ into \eqref{eq:kms-rate-main} gives $\Gamma_+^G(x,\Omega)=G(x)$. Since $G$ was arbitrary among positive $C^1$ functions, the equation $G(x)=\gamma$ has whatever regular level-set structure the chosen response profile supplies, subject only to the ordinary range and regularity conditions. Thus detailed balance alone cannot select the shell.
\end{proof}

The underdetermination lemma fixes the logical status of the response profile
before any shell interpretation. Once \eqref{eq:response-function-main} is
fixed, the record analysis supplies two target rates,
\begin{equation}\label{eq:target-rates-main}
\gamma_\varphi=\frac{\log\varphi}{\Delta\tau},
\qquad
\gamma_{\mathrm{mem}}=\Gamma_c(\kappa_{\mathrm r},\Delta\tau).
\end{equation}
From this point onward these two symbols are standing shorthand:
$\gamma_\varphi$ and $\gamma_{\mathrm{mem}}$ are not redefined in the later
corollaries, and for $\star\in\{\varphi,\mathrm{mem}\}$ we write
\[
\Sigma_\star^{\mathrm{loc}}:=R_\Omega^{-1}(\gamma_\star)
\]
for the corresponding response locus of the fixed calibrated response function.
The theorem chain from this point onward has three separate layers and we keep
them separate. The classical sampled-counter theorem is
\Cref{prop:recovery-main}; the next assumption fixes one stationary response
profile and the sole Model~A calibration bridge; the corollary after the
level-set theorem then turns the two classical thresholds into response loci.
This is the point of the separation: the regular-level-set algebra below is
not a new detector-dynamics theorem.

\begin{assumption}[Model A: sampled counter with one fixed response closure]\label{ass:udw-rate-calibration-main}
\leavevmode\par\smallskip
\noindent\fbox{%
\parbox{\dimexpr\linewidth-2\fboxsep-2\fboxrule\relax}{%
\textbf{Model A.}
Sections~\ref{sec:operational}--\ref{sec:records} have already fixed one
classical sampled counter law. In the response layer we do not derive that
counter law from KMS thermality or from general UDW measurement dynamics.
Instead, we fix one stationary detector response profile and impose only the
closure
\[
\Gamma=R_\Omega(x)=\Gamma_+(x,\Omega).
\]
All response loci, shell upgrades, transport identities, and inverse
corollaries below are conditional consequences of that prescribed calibration
step.
}}

Work under \Cref{ass:kms-response-main}. Fix a detector gap $\Omega>0$ and a
static region $U$. The stationary detector response model supplies one
positive $C^1$ function
\[
R_\Omega:U\to(0,\infty),
\qquad
R_\Omega(x)=\Gamma_+(x,\Omega)
=\lambda_{\mathrm{sw}}^2\,
\frac{\mathcal S_x(\Omega)}{e^{\beta_\infty N(x)\Omega}-1}.
\]
Once this function is fixed, the remaining analysis in this section is level-
set algebra for that one prescribed response profile, exactly as stipulated in
Model~A.
\end{assumption}

\begin{assumption}[Standing response-side hypotheses for the fixed-response theorem chain]\label{ass:response-standing-main}
Work under \Cref{ass:udw-rate-calibration-main}. Thus one fixed static region
$U$, one detector gap $\Omega>0$, and one calibrated response profile
$R_\Omega:U\to(0,\infty)$ have already been chosen. In particular:
\begin{enumerate}[label=\textup{(\roman*)},leftmargin=2.2em,itemsep=0.25em]
\item for every static worldline $x\in U$, the pulled-back spectral measure is
gap-regular at $\pm\Omega$, absolutely continuous with continuous finite
density $\widehat W_x$ on a neighborhood of $\pm\Omega$, and has no atom or
singular part there;
\item the positive-frequency density satisfies $\widehat W_x(\Omega)>0$ on $U$;
\item the calibrated spectral weight
$x\mapsto\mathcal S_x(\Omega)$ is a positive $C^1$ function on $U$;
\item equivalently, the fixed response profile
\[
R_\Omega(x)=
\lambda_{\mathrm{sw}}^2\,
\frac{\mathcal S_x(\Omega)}{e^{\beta_\infty N(x)\Omega}-1}
\]
is a positive $C^1$ function on $U$.
\end{enumerate}
Every response-locus, shell, transport, and fixed-radius theorem below is read
inside this standing hypothesis package unless it explicitly adds a stronger
specialization, such as homogeneous spectral weight, a small-locus horizon
family, or externally supplied ratio-data asymptotics.
\end{assumption}

The next theorem is therefore only about level sets of one fixed calibrated
response function.

\begin{theorem}[Fixed-response locus theorem for one calibrated response function]\label{thm:response-locus-main}
\ResponseStandingClause let $\gamma>0$ be any target
rate, and define
\[
L_\gamma=\{x\in U:R_\Omega(x)=\gamma\}.
\]
Then $L_\gamma$ is empty exactly when $\gamma$ lies outside the range of
$R_\Omega$ on $U$. If $x_0\in L_\gamma$ and $dR_\Omega(x_0)\neq0$, then
$L_\gamma$ is, near $x_0$, a $C^1$ codimension-one hypersurface with normal
covector $dR_\Omega(x_0)$.

In the calibrated homogeneous spectral-weight regime,
$\mathcal S_x(\Omega)\equiv\mathcal S(\Omega)$, the equation $R_\Omega(x)=\gamma$
is equivalent to
\[
\beta_\infty N(x)=\beta_\gamma,
\qquad
\beta_\gamma=\frac{1}{\Omega}
\log\!\left(1+\frac{\lambda_{\mathrm{sw}}^2\mathcal S(\Omega)}{\gamma}\right).
\]
In that regime $dR_\Omega(x_0)\neq0$ if and only if $dN(x_0)\neq0$. Thus shell
language is justified only after the separate range check
$\gamma\in\operatorname{range}(R_\Omega)$ and the regularity check
$dR_\Omega(x_0)\neq0$.
\end{theorem}

\begin{proof}
The first assertion is the definition of the range. The local hypersurface
statement is the regular level-set theorem applied to the $C^1$ function
$R_\Omega$.

If $\mathcal S_x(\Omega)$ is constant, the equation
\[
\lambda_{\mathrm{sw}}^2\frac{\mathcal S(\Omega)}{e^{\beta_\infty N(x)\Omega}-1}=\gamma
\]
is equivalent to
\[
e^{\beta_\infty N(x)\Omega}
=1+\frac{\lambda_{\mathrm{sw}}^2\mathcal S(\Omega)}{\gamma},
\]
which gives the displayed value of $\beta_\gamma$. Differentiating the
homogeneous response function gives
\[
dR_\Omega
=
-\lambda_{\mathrm{sw}}^2\mathcal S(\Omega)
\frac{\beta_\infty\Omega e^{\beta_\infty N\Omega}}
{(e^{\beta_\infty N\Omega}-1)^2}\,dN .
\]
The scalar prefactor is strictly negative on $U$, so $dR_\Omega(x_0)$
vanishes exactly when $dN(x_0)$ vanishes.
\end{proof}

Under \Cref{ass:udw-rate-calibration-main}, the two classical target rates from
\Cref{sec:records} now select the response-level equations
\[
\Gamma=R_\Omega(x)=\gamma_\varphi,
\qquad
\Gamma=R_\Omega(x)=\gamma_{\mathrm{mem}}.
\]
Those equations remain only target substitutions inside the fixed calibrated
response model until the range and regularity checks from
\Cref{thm:response-locus-main} are verified. Appendix
~\ref{app:abstract-homogeneous-main} records the corresponding threshold
substitutions as conditional corollaries: first the two response loci
selected by $\gamma_\varphi$ and $\gamma_{\mathrm{mem}}$, then the later

\Cref{lem:one-gap-kms-underdetermination-main,ass:udw-rate-calibration-main,thm:response-locus-main}
remain the entire bridge from record thresholds to response-locus language:
calibration dependence first, fixed-response closure second, regular level

specializations, quotient laws, near-horizon limits, fixed-radius
