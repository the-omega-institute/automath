# Golden-Mean Folding in Binary Scan--Projection Systems

This directory contains the English Paper I manuscript extracted from the larger scan--projection / audit-certificate monograph.

## Positioning

Working title:

**Golden-Mean Folding in Binary Scan--Projection Systems: Stable Types, Normal Forms, and Auditable Addressing**

Main claim:

> Binary scan--projection readouts admit a golden-mean folding that produces computable, cross-scale compatible, and statistically auditable stable types.

## Scope

The manuscript intentionally covers only:

- scan--projection systems;
- golden / Zeckendorf syntax;
- finite-window folding and normal forms;
- fold-aware cross-resolution compatibility;
- recursive addressing without visible sigma-algebra expansion;
- scan error profiles and boundary clarity estimates;
- finite certificate and reproducible audit interfaces.

It does **not** claim to derive quantum mechanics, spacetime, Einstein equations, the Riemann hypothesis, or a unified physical theory. Those are later bridge papers.

## Files

- `main.tex`: manuscript entry point.
- `preamble.tex`: LaTeX definitions.
- `sections/`: twelve-section submission structure.
- `appendices/`: A--F submission appendices for local normalization, finite-state Markov covers, statistical fold-histogram audits, deterministic fiber ledgers, reproducibility, and examples.
- `references.bib`: local references drawn from existing publication tracks.
- `SOURCE_MAP.md`: mapping from the monograph plan to this paper.
- `THEOREM_LIST.md`: main theorem package.
