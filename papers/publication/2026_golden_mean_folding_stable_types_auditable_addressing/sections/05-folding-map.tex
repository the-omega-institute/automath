\section{The folding map}
\label{sec:folding}

The folding map can be computed by local Fibonacci carry elimination.  In a most-significant-digit convention the basic rewrite is
\[
011\longrightarrow 100,
\]
which expresses
\[
F_k+F_{k+1}=F_{k+2}.
\]
In the low-to-high convention used in \Cref{sec:golden-syntax}, the same relation is read with the word reversed.  The rewrite is value preserving: it changes the representation, not the represented Fibonacci value.

\begin{definition}[Golden carry-elimination system]
\label{def:golden-carry-elimination}
The depth-\(m\) carry-elimination system consists of the finite local rewrites generated by the Fibonacci carry relation, together with a deterministic boundary convention for carries that leave the visible window.  A word is normal if it has no adjacent ones.
\end{definition}

\begin{theorem}[Golden folding normal form]
\label{thm:golden-fold-normal-form}
For every \(m\ge1\), the depth-\(m\) carry-elimination system is terminating and confluent.  Hence every \(\omega\in\Om_m\) reaches a unique legal word in \(X_m\), independent of rewriting order.  This legal word is \(\Fold_m(\omega)\), and
\[
\Fold_m^2=\Fold_m.
\]
\end{theorem}

\begin{proof}
Each rewrite preserves the Fibonacci value \(N_m(\omega)\).  Termination follows from a well-founded descent measure that first orders unresolved adjacent-one patterns by their carry height and then by the finite lexicographic profile of illegal pairs.  Local confluence is checked on the finite list of critical overlaps of adjacent carry rules.  Since the system is terminating and locally confluent, Newman's lemma gives global confluence.  The unique terminal word is legal, value preserving, and therefore agrees with the Zeckendorf prefix of \(N_m(\omega)\).  If the input is already legal, no rewrite applies, which gives \(\Fold_m^2=\Fold_m\).
\end{proof}

\begin{corollary}[Stable type quotient]
\label{cor:stable-type-quotient}
The fibers of \(\Fold_m\) define an equivalence relation on \(\Om_m\),
\[
\omega\sim_m\omega'
\quad\Longleftrightarrow\quad
\Fold_m(\omega)=\Fold_m(\omega').
\]
The quotient is canonically represented by \(X_m\).
\end{corollary}

\begin{corollary}[Stable state-count reduction]
\label{cor:state-count-reduction}
With the indexing of \Cref{sec:golden-syntax},
\[
\frac{|X_m|}{|\Om_m|}
=\frac{F_{m+1}}{2^m}
=\Theta\!\left(\left(\frac{\varphi}{2}\right)^m\right).
\]
\end{corollary}

\begin{remark}
\Cref{thm:golden-fold-normal-form} is the core finite-combinatorial theorem.  Stable types are not introduced by an empirical clustering choice; they are normal forms of a typed, value-preserving rewriting system.
\end{remark}
