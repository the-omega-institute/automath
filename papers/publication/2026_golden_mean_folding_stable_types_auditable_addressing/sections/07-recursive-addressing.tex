\section{Recursive addressing and visible sigma-algebras}
\label{sec:recursive-addressing}

Folding produces stable representatives.  Addressing gives a way to name, query, and organize those representatives without claiming that new externally visible information has been added.

\begin{definition}[Address object]
\label{def:address-object}
An address at depth \(m\) is a finite object
\[
a_m=(m,x_m,\Cert_m),
\]
where \(x_m\in X_m\) is a stable type and \(\Cert_m\) is the certificate data used to verify its construction, restriction history, or statistical status.
\end{definition}

\begin{definition}[Partial readout]
\label{def:partial-readout}
The readout operation is a partial map
\[
\Read(a_m)\in \mathrm{Value}\cup\{\mathrm{NULL}\}.
\]
It returns \(\mathrm{NULL}\) when the requested value is not determined by the address, when the certificate budget is insufficient, or when compatible completions remain indistinguishable at the stated depth.
\end{definition}

Let \(\mathcal G_L\) denote the visible sigma-algebra generated by all readout objects available up to address layer \(L\).

\begin{theorem}[Recursive addressing and non-expansion]
\label{thm:recursive-address-nonexpansion}
Suppose layer \(L+1\) addresses are constructed by finite readout words, folding certificates, restriction maps, and deterministic bookkeeping from layer \(L\).  Then
\[
\mathcal G_{L+1}\subseteq\mathcal G_L.
\]
Thus recursive addressing does not add new externally visible information.  It reorganizes, names, and selects events already measurable at the previous visible layer.
\end{theorem}

\begin{proof}
Each layer \(L+1\) generator is a measurable function of layer \(L\) data.  Therefore every event determined by level \(L+1\) addresses is measurable with respect to \(\mathcal G_L\).
\end{proof}

\begin{remark}
The theorem is a scope-control statement.  It prevents address semantics from being mistaken for an information leak.  Addresses may improve organization, queryability, and reproducibility while remaining internal to the prior visible layer.
\end{remark}
