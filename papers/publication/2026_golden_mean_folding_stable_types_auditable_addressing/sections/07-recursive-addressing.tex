\section{Recursive addressing and visible sigma-algebras}
\label{sec:recursive-addressing}

The folding construction produces stable types.  Addressing gives a way to name, query, and organize those types without claiming that new externally visible information has been added.

\begin{definition}[Address object]
\label{def:address-object}
An address at depth \(m\) is a finite object
\[
a_m=(m,x_m,\Cert_m),
\]
where \(x_m\in X_m\) is a stable type and \(\Cert_m\) is certificate data used to verify its construction, restriction history, or statistical status.
\end{definition}

\begin{definition}[Partial readout]
\label{def:partial-readout}
The readout operation is a partial map
\[
\Read(a_m)\in \mathrm{Value}\cup\{\mathrm{NULL}\}.
\]
It returns \(\mathrm{NULL}\) when the requested value is not determined by the address, when the certificate budget is insufficient, or when compatible completions remain indistinguishable at the stated depth.
\end{definition}

Let \(\mathcal G_{\le L}\) denote the cumulative visible sigma-algebra up to address layer \(L\).  Let \(\mathcal H_{L+1}\) denote the sigma-algebra generated only by the new layer-\((L+1)\) address events.

\begin{theorem}[Recursive addressing is visibility-conservative]
\label{thm:recursive-address-nonexpansion}
Suppose every new layer-\((L+1)\) address event is measurable with respect to \(\mathcal G_{\le L}\).  Then
\[
\mathcal H_{L+1}\subseteq\mathcal G_{\le L}.
\]
Consequently,
\[
\mathcal G_{\le L+1}=\mathcal G_{\le L}.
\]
\end{theorem}

\begin{proof}
Each generator of \(\mathcal H_{L+1}\) is \(\mathcal G_{\le L}\)-measurable by assumption.  Since \(\mathcal G_{\le L}\) is a sigma-algebra, the sigma-algebra generated by those events is contained in \(\mathcal G_{\le L}\).  Hence
\[
\mathcal H_{L+1}\subseteq\mathcal G_{\le L}.
\]
The cumulative algebra at the next layer is
\[
\mathcal G_{\le L+1}=\sigma(\mathcal G_{\le L},\mathcal H_{L+1}).
\]
Since \(\mathcal H_{L+1}\subseteq\mathcal G_{\le L}\), this equals \(\mathcal G_{\le L}\).
\end{proof}

\begin{remark}
The theorem distinguishes the new address layer \(\mathcal H_{L+1}\) from the cumulative visible layer \(\mathcal G_{\le L+1}\).  Recursive addressing may improve naming, queryability, and reproducibility, but under the stated measurability condition it does not add externally new measurable information.
\end{remark}
