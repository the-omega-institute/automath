\begin{theorem}\label{thm:kernel-chebotarev-second-main-term-witness}
Fix a finite group \(G\) and \(u\in(0,1]\) in the setting of Assumption \ref{thm:kernel-chebotarev-exp}. Let \(\lambda_1(u)=\spr(M_{K,\ind}(u))\) be the Perron root in the trivial channel, and write \(M_{K,\rho}(u)\) for the twisted matrix attached to a nontrivial irreducible representation \(\rho\neq \ind\). Assume that there exist a nontrivial channel \(\rho_\star\neq \ind\) and a complex number \(\mu_\star\in\mathrm{spec}(M_{K,\rho_\star}(u))\) such that
\begin{enumerate}
  \item \(\abs{\mu_\star}=\eta(u)>0\);
  \item if \(\mu_\star\notin\RR\), then the only eigenvalues of modulus \(\eta(u)\) occurring in the family \(\{M_{K,\rho}(u):\rho\neq \ind\}\) are \(\mu_\star\) and \(\overline{\mu_\star}\), and each occurs exactly once, with algebraic multiplicity \(1\); if \(\mu_\star\in\RR\), then \(\mu_\star\) is the only eigenvalue of modulus \(\eta(u)\) occurring in the family \(\{M_{K,\rho}(u):\rho\neq \ind\}\), and it occurs exactly once, with algebraic multiplicity \(1\);
  \item
  \[
  \Gamma(u):=\max\bigl\{\abs{\nu}:\nu\in\mathrm{spec}(M_{K,\rho}(u)),\ \rho\neq \ind,\ \nu\notin\{\mu_\star,\overline{\mu_\star}\}\bigr\}
  \]
  satisfies \(\Gamma(u)<\eta(u)\).
\end{enumerate}
Then, for every real number \(\Gamma_+(u)\) with
\[
\Gamma(u)<\Gamma_+(u)<\eta(u)
\]
and every conjugacy-invariant subset \(A\subseteq G\),
\[
B_{n,A}(u):=\sum_{c\in A}B_{n,c}(u)
\]
admits an expansion
\[
B_{n,A}(u)
=
\mathrm{unif}_G(A)\frac{\lambda_1(u)^n}{n}
\;+\;
\Re\!\bigl(c_A\,\mu_\star^{\,n}\bigr)\frac{1}{n}
\;+\;
O\!\left(\frac{\Gamma_+(u)^n}{n}\right)
\]
for some coefficient \(c_A\in\CC\). If \(c_A\neq 0\), then there is a constant \(c(A)>0\) such that
\[
\bigl|\Re(c_A\mu_\star^{\,n})\bigr|\ge c(A)\,\eta(u)^n
\]
for infinitely many integers \(n\).
\end{theorem}

\begin{proof}
By Assumption \ref{thm:kernel-chebotarev-exp}, for each conjugacy-invariant subset \(A\subseteq G\) there are linear functionals \(L_{A,\rho}\) such that
\[
B_{n,A}(u)
=
\mathrm{unif}_G(A)\frac{\lambda_1(u)^n}{n}
\;+\;
\frac{1}{n}\sum_{\rho\in\widehat G\setminus\{\ind\}}L_{A,\rho}\!\bigl(M_{K,\rho}(u)^n\bigr).
\]
Fix \(\Gamma_+(u)\in(\Gamma(u),\eta(u))\). For \(\rho\neq \ind\), write \(T_\rho:=M_{K,\rho}(u)\), and set
\[
\Sigma_\rho:=\mathrm{spec}(T_\rho)\cap\{\mu_\star,\overline{\mu_\star}\}.
\]
For each \(\omega\in\Sigma_\rho\), let \(P_{\rho,\omega}\) be the corresponding Riesz projection, and set
\[
R_\rho:=T_\rho-\sum_{\omega\in\Sigma_\rho}\omega\,P_{\rho,\omega}.
\]
Since the eigenvalues in \(\Sigma_\rho\) are simple, the projections \(P_{\rho,\omega}\) have rank one, commute with \(T_\rho\), and satisfy \(P_{\rho,\omega}P_{\rho,\omega'}=0\) for \(\omega\neq \omega'\). Therefore
\[
T_\rho^n
=
\sum_{\omega\in\Sigma_\rho}\omega^n P_{\rho,\omega}+R_\rho^{\,n}
\qquad(n\ge 1).
\]
Moreover,
\[
\mathrm{spec}(R_\rho)=\mathrm{spec}(T_\rho)\setminus \Sigma_\rho,
\]
so \(\spr(R_\rho)\le \Gamma(u)<\Gamma_+(u)\). Fix any matrix norm. By equivalence of norms in finite dimension and Gelfand's formula, there exists \(C_\rho>0\) such that
\[
\|R_\rho^{\,n}\|\le C_\rho\,\Gamma_+(u)^n
\qquad(n\ge 1).
\]
Applying \(L_{A,\rho}\) and summing over the finitely many nontrivial channels yields coefficients \(a_{A,\omega}\in\CC\) for which
\[
S_n:=\sum_{\rho\in\widehat G\setminus\{\ind\}}L_{A,\rho}\!\bigl(M_{K,\rho}(u)^n\bigr)
=
\sum_{\omega\in\{\mu_\star,\overline{\mu_\star}\}}a_{A,\omega}\,\omega^n
+O\!\bigl(\Gamma_+(u)^n\bigr).
\]
Because \(B_{n,A}(u)\) and \(\mathrm{unif}_G(A)\lambda_1(u)^n/n\) are real, each \(S_n\) is real. It remains only to identify the symmetry between the coefficients of \(\mu_\star^n\) and \(\overline{\mu_\star}^{\,n}\).

If \(\mu_\star\notin\RR\), then conjugating the previous expansion and subtracting gives
\[
\bigl(a_{A,\mu_\star}-\overline{a_{A,\overline{\mu_\star}}}\bigr)\mu_\star^n
+
\bigl(a_{A,\overline{\mu_\star}}-\overline{a_{A,\mu_\star}}\bigr)\overline{\mu_\star}^{\,n}
=
O\!\bigl(\Gamma_+(u)^n\bigr).
\]
After division by \(\overline{\mu_\star}^{\,n}\), this becomes
\[
\bigl(a_{A,\mu_\star}-\overline{a_{A,\overline{\mu_\star}}}\bigr)
\left(\frac{\mu_\star}{\overline{\mu_\star}}\right)^n
+
\bigl(a_{A,\overline{\mu_\star}}-\overline{a_{A,\mu_\star}}\bigr)
=
O\!\left(\left(\frac{\Gamma_+(u)}{\eta(u)}\right)^n\right).
\]
Set
\[
z:=\frac{\mu_\star}{\overline{\mu_\star}}\in \TT\setminus\{1\},
\qquad
c_1:=a_{A,\mu_\star}-\overline{a_{A,\overline{\mu_\star}}},
\qquad
c_2:=a_{A,\overline{\mu_\star}}-\overline{a_{A,\mu_\star}}.
\]
Since \(\Gamma_+(u)<\eta(u)\), the right-hand side tends to \(0\). If \(z\) has finite order \(q\), then the left-hand side is constant along each residue class modulo \(q\). Evaluating along the classes \(n\equiv 0\pmod q\) and \(n\equiv 1\pmod q\) gives
\[
c_1+c_2=0,
\qquad
c_1z+c_2=0.
\]
Because \(z\neq 1\), this forces \(c_1=c_2=0\). If \(z\) has infinite order, then \((z^n)_{n\ge 1}\) is dense in \(\TT\). Choosing subsequences with limits \(1\) and \(z\), and passing to the limit in the displayed relation, yields the same two equations and again forces \(c_1=c_2=0\). Therefore
\[
a_{A,\overline{\mu_\star}}=\overline{a_{A,\mu_\star}}.
\]
Setting \(c_A:=2a_{A,\mu_\star}\) gives
\[
S_n
=
\Re\!\bigl(c_A\mu_\star^n\bigr)+O\!\bigl(\Gamma_+(u)^n\bigr).
\]
If \(\mu_\star\in\RR\), then the dominant set consists only of \(\mu_\star\), and the reality of \(S_n\) shows that the corresponding coefficient is real. The same formula therefore holds with \(c_A:=a_{A,\mu_\star}\in\RR\). Substituting into the expression for \(B_{n,A}(u)\) proves the claimed expansion.

Finally, write \(c_A=\abs{c_A}e^{i\phi}\) and \(\mu_\star=\eta(u)e^{i\theta}\). Then
\[
\Re(c_A\mu_\star^n)=\abs{c_A}\,\eta(u)^n\cos(\phi+n\theta).
\]
If \(\theta/\pi\in\QQ\), the sequence \((\cos(\phi+n\theta))_{n\ge 1}\) is periodic. It cannot vanish identically. Indeed, when \(\mu_\star\in\RR\) one has \(c_A\in\RR\), so \(\Re(c_A\mu_\star^n)=c_A\mu_\star^n\), which is not identically zero because \(c_A\neq 0\) and \(\eta(u)>0\). When \(\mu_\star\notin\RR\), the identity \(\Re(c_A\mu_\star^n)=0\) for all \(n\) would imply
\[
a_{A,\mu_\star}\left(\frac{\mu_\star}{\overline{\mu_\star}}\right)^n+\overline{a_{A,\mu_\star}}=0
\qquad(n\ge 1),
\]
and comparing \(n=1\) with \(n=2\) would force \(a_{A,\mu_\star}=0\) because \(\mu_\star/\overline{\mu_\star}\neq 1\), hence \(c_A=0\), again a contradiction. Therefore the periodic sequence attains a positive absolute value infinitely often. If \(\theta/\pi\notin\QQ\), the sequence \((\phi+n\theta)\bmod 2\pi\) is dense in \(\TT\), so there are infinitely many \(n\) with \(\abs{\cos(\phi+n\theta)}\ge 1/2\). In either case there exists \(c(A)>0\) such that
\[
\abs{\Re(c_A\mu_\star^n)}\ge c(A)\,\eta(u)^n
\]
for infinitely many \(n\), as claimed.
\end{proof}
