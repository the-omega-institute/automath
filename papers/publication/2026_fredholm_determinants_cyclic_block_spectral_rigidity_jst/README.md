# Split-out manuscript

This folder contains the standalone operator-focused manuscript split
out from
`2026_dynamical_zeta_finite_part_spectral_fingerprint_etds`.

Focus of this paper:

- the Fredholm--Witt bridge for trace-class operators,
- cyclic-block converse realisations of Euler products,
- spectral rigidity of Fredholm determinants,
- analytic variation of finite-part constants,
- spectral-gap limit laws for locally constant observables.

The layout and tone follow the ETDS-facing publication draft, but the
narrative has been compressed to a single operator-centered line.
