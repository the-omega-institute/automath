# Split-out manuscript

This folder contains the standalone manuscript split out from
`2026_dynamical_zeta_finite_part_spectral_fingerprint_etds`.

Focus of this paper:

- the explicit ten-state weighted synchronisation kernel,
- the self-dual determinant identity,
- the completed determinant and its curve geometry,
- cyclotomic resultants and twisted Perron roots.

The style, theorem layout, and section structure follow the ETDS-facing
publication draft.
