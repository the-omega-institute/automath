\title{A trace-rank obstruction and a multiplier-orbit certificate for two size-four Sidon sets}
\author{}
\date{}

\begin{document}

\maketitle

\begin{abstract}
Let
\[
A=\{0,1,3,11\},\qquad B=\{0,1,4,11\}.
\]
Motivated by Niu's recent size-four Sidon-extension candidates, we prove two complementary results. First, for every prime power \(q\), no affine image of \(A\) or \(B\) is contained in the canonical Singer trace difference set in \(\Z/(q^2+q+1)\Z\). The proof is a short rank argument over \(\F_{q^3}/\F_q\), followed by two explicit cubic reductions. Second, at the first non-Singer multiplier-descent parameter \(q=41\), \(v=1723\), we certify that the locally admissible forced core
\[
[1]\cup[2]\cup[4]\cup[12]
\]
for \(A\) cannot be completed to a \(41\)-multiplier-invariant cyclic \((1723,42,1)\) difference set. The finite part is phrased as an exact cover problem on Frobenius orbits and is accompanied by a deterministic verifier. Thus this fixed forced core contributes no obstruction to the Singer-affine proof, although the full multiplier-descent problem remains open.
\end{abstract}

\section{Introduction}

A finite set \(S\subset \Z\) is Sidon if all nonzero differences \(s-s'\), \(s,s'\in S\), \(s\ne s'\), are distinct. A cyclic planar difference set of projective order \(q\) is a subset
\[
D\subset \Z_v,\qquad v=q^2+q+1,
\]
of cardinality \(q+1\) such that every nonzero element of \(\Z_v\) occurs exactly once as a difference \(d-d'\) with \(d,d'\in D\), \(d\ne d'\). Equivalently,
\[
DD^{(-1)}=q+\Z_v
\]
in the integral group ring. Such a difference set is also a perfect difference set of size \(q+1\) in the Sidon-extension terminology.

The original Sidon-extension conjecture of Erd\H{o}s asked whether every finite Sidon set embeds into some finite perfect difference set. Alexeev and Mixon recently disproved the conjecture, also pointing out the relevance of an earlier construction of Hall. Niu then isolated two size-four candidates,
\[
A=\{0,1,3,11\},\qquad B=\{0,1,4,11\},
\]
and gave extensive computational evidence that they are non-extending in the cyclic projective-plane range. Niu's paper explicitly leaves open the need for a complete proof, either by a polarity argument in the spirit of Alexeev and Mixon or by a multiplier descent.

This note records two pieces of progress toward that descent. The first is conceptual and uniform in \(q\): in the Desarguesian, or Singer, case, the obstruction is completely elementary. If \(K=\F_q\), \(F=\F_{q^3}\), and \(\alpha\) is primitive in \(F^\times\), the canonical Singer trace line is
\[
D_q=\{i\in \Z_{q^2+q+1}:\Tr_{F/K}(\alpha^i)=0\}.
\]
We prove that no affine image of \(A\) or \(B\) lies in \(D_q\), for any prime power \(q\). The key observation is that affine containment is equivalent to the four vectors
\[
1,\quad w,\quad w^a,\quad w^b
\]
spanning a \(K\)-subspace of dimension at most \(2\), where \(w=\alpha^u\) and \(u\in \Z_v^\times\). For \(A\), this forces an irreducible cubic \(x^3-rx-s\) to satisfy a reducibility condition. For \(B\), the corresponding cubic is \(x^3-ax^2+a^2x-c\), and the condition that \(x^{11}\) have no \(x^2\)-term again gives a root in \(K\). Both contradictions are characteristic-free.

The second result is finite and deliberately narrow. The first parameter at which the classical uniqueness results do not automatically reduce a cyclic projective plane to the Singer plane is \(q=41\). Here
\[
v=41^2+41+1=1723,
\]
and \(1723\) is prime. Multiplication by \(41\) has order \(3\) on \(\Z_{1723}^\times\), so every nonzero multiplier orbit has the form
\[
[x]=\{x,41x,41^2x\}.
\]
A multiplier-normalized cyclic planar difference set of order \(41\) is therefore a union of \(14\) such nonzero orbits. We study the forced core arising from the normalized affine copy
\[
1+A=\{1,2,4,12\}.
\]
That core consists of the four point-orbits
\[
[1],\quad [2],\quad [4],\quad [12].
\]
It is locally admissible: its internal and cross-difference orbits are \(44\) distinct nonzero multiplier-orbits. Hence a completion would require exactly \(10\) further point-orbits whose internal differences, differences with the core, and pairwise cross-differences partition the remaining \(530\) difference-orbits. We certify that no such \(10\)-orbit completion exists.

The finite certificate does not settle Niu's problem. It only eliminates one fixed forced core at \(q=41\). The remaining cases include all other translations of \(A\), the corresponding forced cores for \(B\), and larger \(q\). Still, the \(q=41\) calculation is a useful stress test for the multiplier-descent program: the core is locally admissible, so the obstruction only appears after the exact-cover constraints are enforced globally.

\section{Preliminaries}

Let \(G=\Z_v\), written additively. If \(D\subset G\), then \(D\) is a cyclic \((v,k,\lambda)\) difference set if \(|D|=k\) and every nonzero \(g\in G\) has exactly \(\lambda\) representations \(g=d-d'\), with \(d,d'\in D\), \(d\ne d'\). In this paper we only use the planar case
\[
(v,k,\lambda)=(q^2+q+1,q+1,1).
\]

Two difference sets \(D,E\subset \Z_v\) are called equivalent if
\[
E=t+uD
\]
for some \(t\in \Z_v\) and some unit \(u\in \Z_v^\times\). An affine image of a finite integer set \(S\subset \Z\) in \(D\) is a subset of the form
\[
t+uS:=\{t+us:s\in S\}\subset \Z_v
\]
with \(u\in \Z_v^\times\).

\begin{lemma}[Group-ring form]
Let \(D\subset \Z_v\), \(|D|=q+1\), and \(v=q^2+q+1\). Then \(D\) is a cyclic planar difference set of order \(q\) if and only if
\[
DD^{(-1)}=q+\Z_v
\]
in \(\Z[\Z_v]\).
\end{lemma}

\begin{proof}
The coefficient of \(g\in \Z_v\) in \(DD^{(-1)}\) is the number of ordered pairs \((d,d')\in D^2\) such that \(d-d'=g\). For \(g=0\) this number is \(|D|=q+1\). The expression \(q+\Z_v\) has coefficient \(q+1\) at \(0\) and coefficient \(1\) at every nonzero element. Thus equality is exactly the assertion that every nonzero difference occurs once.
\end{proof}

Let \(q=p^f\). A classical multiplier theorem says that every cyclic planar difference set of order \(q\) is equivalent to one satisfying
\[
pD=D.
\]
We shall use this only as motivation and for the \(q=41\) normalized finite instance.

For a residue \(x\in \Z_v\), write
\[
[x]=\{p^i x\bmod v:i\ge 0\}
\]
for its multiplier orbit. If \(pD=D\), then \(D\) is a union of such orbits. The difference-set condition can then be rewritten as a finite exact-cover system on point-orbits and difference-orbits.

\begin{lemma}[Orbit exact-cover equations]
Let \(q=p^f\), \(v=q^2+q+1\), and suppose \(D\subset \Z_v\) satisfies \(pD=D\). Let \(\Omega\) be the set of \(p\)-orbits in \(\Z_v\). For \(O\in \Omega\), introduce a binary variable \(X_O\), where \(X_O=1\) means \(O\subset D\). Then \(D\) is a cyclic planar difference set if and only if
\[
\sum_{O\in\Omega}|O|X_O=q+1
\]
and, for every nonzero orbit \(L\in\Omega\),
\[
\sum_{O,P\in\Omega}N_{O,P}^L X_OX_P=|L|,
\]
where
\[
N_{O,P}^L=\#\{(a,b)\in O\times P:a-b\in L\}.
\]
\end{lemma}

\begin{proof}
The cardinality equation is immediate. For the difference equations, group the ordered differences \(a-b\), \(a,b\in D\), by the orbit \(L\) containing \(a-b\). Since \(D\) is a union of \(p\)-orbits, the number of representations is constant on each nonzero difference-orbit \(L\). The planar difference-set condition says that each element of \(L\) occurs exactly once, so the total number of ordered differences landing in \(L\) must be \(|L|\). This is precisely the displayed equation.
\end{proof}

We also use the following trace model of Singer difference sets. Let \(K=\F_q\), \(F=\F_{q^3}\), and let \(\alpha\) generate \(F^\times\). Put
\[
v=\frac{q^3-1}{q-1}=q^2+q+1.
\]
Since \(\alpha^v\in K^\times\), the condition
\[
\Tr_{F/K}(\alpha^i)=0
\]
depends only on the class of \(i\) modulo \(v\). Thus
\[
D_q=\{i\in \Z_v:\Tr_{F/K}(\alpha^i)=0\}
\]
is a well-defined subset of \(\Z_v\). It is the usual Singer difference set associated with a line in \(\mathrm{PG}(2,q)\).

\begin{lemma}[Trace-rank criterion]
Let \(S=\{0,1,a,b\}\subset \Z\), and let \(D_q\) be the trace Singer difference set above. There is an affine copy \(t+uS\subset D_q\), with \(u\in \Z_v^\times\), if and only if there exists
\[
w=\alpha^u,\qquad u\in \Z_v^\times,
\]
such that
\[
\dim_K\Span_K\{1,w,w^a,w^b\}\le 2.
\]
\end{lemma}

\begin{proof}
Suppose first that \(t+uS\subset D_q\). Put \(\beta=\alpha^t\) and \(w=\alpha^u\). Then
\[
\Tr_{F/K}(\beta w^s)=0
\]
for every \(s\in S\). Hence the four vectors \(1,w,w^a,w^b\) are contained in the kernel of the nonzero \(K\)-linear functional
\[
x\mapsto \Tr_{F/K}(\beta x).
\]
This kernel is a \(2\)-dimensional \(K\)-subspace of the \(3\)-dimensional \(K\)-space \(F\), so the displayed span has dimension at most \(2\).

Conversely, if the span has dimension at most \(2\), then there exists a nonzero \(K\)-linear functional vanishing on it. The trace pairing
\[
F\times F\to K,\qquad (x,y)\mapsto \Tr_{F/K}(xy)
\]
is nondegenerate, so this functional has the form \(x\mapsto \Tr_{F/K}(\beta x)\) for some \(\beta\in F^\times\). Write \(\beta=\alpha^t\). Then
\[
\Tr_{F/K}(\alpha^{t+us})=\Tr_{F/K}(\beta w^s)=0
\]
for all \(s\in S\), hence \(t+uS\subset D_q\).
\end{proof}

\section{Main results}

We first prove the uniform Singer obstruction.

\begin{theorem}[Singer-affine obstruction for \(A\) and \(B\)]
Let \(q\) be any prime power, \(v=q^2+q+1\), and let
\[
D_q=\{i\in \Z_v:\Tr_{\F_{q^3}/\F_q}(\alpha^i)=0\}
\]
be a trace Singer difference set. Then \(D_q\) contains no affine image of
\[
A=\{0,1,3,11\}
\]
and no affine image of
\[
B=\{0,1,4,11\}.
\]
\end{theorem}

\begin{proof}
Let \(K=\F_q\) and \(F=\F_{q^3}\). Let \(w=\alpha^u\) with \(u\in \Z_v^\times\). Then \(w\notin K\). Indeed, if \(w\in K^\times\), then the exponent \(u\) is congruent to a multiple of \(v=(q^3-1)/(q-1)\) modulo \(q^3-1\), so \(u\equiv 0\pmod v\), contradicting \(u\in\Z_v^\times\). Thus \(1,w\) are \(K\)-linearly independent.

We treat \(A\) first. By the trace-rank criterion, an affine copy of \(A\) would imply
\[
\dim_K\Span_K\{1,w,w^3,w^{11}\}\le 2.
\]
Since \(1,w\) are independent, there exist \(r,s\in K\) such that
\[
w^3=rw+s.
\]
Thus \(w\) is a root of
\[
f(x)=x^3-rx-s\in K[x].
\]
Since \(w\notin K\) and \(w\in F\), its minimal polynomial over \(K\) has degree \(3\). Hence \(f\) must be irreducible.

Reducing \(x^{11}\) modulo \(x^3-rx-s\) gives
\[
x^{11}\equiv
(4r^3s+s^3)x^2+(r^5+6r^2s^2)x+(r^4s+3rs^3).
\]
The condition \(w^{11}\in \Span_K\{1,w\}\) forces the coefficient of \(x^2\) to vanish:
\[
4r^3s+s^3=0,
\]
or
\[
s(4r^3+s^2)=0.
\]
If \(s=0\), then \(f(x)=x(x^2-r)\) is reducible, a contradiction. If \(\operatorname{char}K=2\), then the equation becomes \(s^3=0\), again giving \(s=0\), a contradiction. Hence assume \(\operatorname{char}K\ne 2\) and \(s\ne 0\). Then
\[
s^2=-4r^3.
\]
In the cyclic group \(K^\times\), the implication ``\(z^3\) is a square, so \(z\) is a square'' holds because \(3\) is odd. Applying this to \(z=-r\), we find \(\tau\in K\) such that
\[
\tau^2=-r.
\]
The relation \(s^2=4\tau^6\) gives \(s=\pm 2\tau^3\). If \(s=2\tau^3\), then \(f(\tau)=0\). If \(s=-2\tau^3\), then \(f(-\tau)=0\). In either case \(f\) has a root in \(K\), contradicting irreducibility. Therefore \(D_q\) contains no affine image of \(A\).

Now consider \(B\). An affine copy of \(B\) would imply
\[
\dim_K\Span_K\{1,w,w^4,w^{11}\}\le 2.
\]
Let the cubic relation of \(w\) be
\[
w^3=aw^2+bw+c,
\]
so \(w\) is a root of
\[
f(x)=x^3-ax^2-bx-c.
\]
Again \(f\) is irreducible over \(K\). Reducing \(x^4\) modulo \(f\) gives
\[
x^4\equiv (a^2+b)x^2+(ab+c)x+ac.
\]
Since \(w^4\in \Span_K\{1,w\}\), the \(x^2\)-coefficient vanishes, so
\[
b=-a^2.
\]
Thus
\[
f(x)=x^3-ax^2+a^2x-c.
\]
Reducing \(x^{11}\) modulo this cubic, the coefficient of \(x^2\) is
\[
c^3-2a^3c^2+3a^6c-a^9.
\]
Since \(w^{11}\in \Span_K\{1,w\}\), this expression is \(0\).

If \(a=0\), then \(c^3=0\), so \(c=0\), and \(f(x)=x^3\), impossible. Hence \(a\ne 0\). Put
\[
y=\frac{c}{a^3}.
\]
Dividing the preceding equation by \(a^9\), we get
\[
y^3-2y^2+3y-1=0.
\]
Let
\[
z=1-y.
\]
Then
\[
z^3-z^2+z-y=-(y^3-2y^2+3y-1)=0.
\]
Consequently,
\[
f(az)=a^3(z^3-z^2+z-y)=0.
\]
Thus \(f\) has the root \(az\in K\), contradicting irreducibility. Therefore \(D_q\) contains no affine image of \(B\).
\end{proof}

\begin{corollary}
For every prime power \(q\), the two sets \(A=\{0,1,3,11\}\) and \(B=\{0,1,4,11\}\) do not embed affinely into any Singer difference set in \(\Z_{q^2+q+1}\).
\end{corollary}

\begin{proof}
Every Singer difference set is equivalent to a trace Singer difference set. Equivalence consists of translation and multiplication by a unit, and these operations preserve the property of containing an affine copy of a fixed set \(S\). The result follows from Theorem \(3.1\).
\end{proof}

We next give the fixed \(q=41\) certificate.

Let
\[
q=41,\qquad v=41^2+41+1=1723.
\]
Then \(1723\) is prime and
\[
41^3\equiv 1\pmod {1723}.
\]
Since \(41\not\equiv 1\pmod {1723}\), every nonzero \(41\)-orbit has size \(3\). There are therefore
\[
\frac{1722}{3}=574
\]
nonzero multiplier-orbits.

For a nonzero orbit \(O=[x]\), its internal difference-orbit set is
\[
\Delta(O)=\{[a-b]:a,b\in O,\ a\ne b\}.
\]
For two distinct nonzero orbits \(O,P\), its cross-difference-orbit set is
\[
\Delta(O,P)=\{[a-b]:a\in O,\ b\in P\}.
\]
In this \(q=41\) situation, a valid single point-orbit contributes \(2\) internal difference-orbits, and a valid pair of point-orbits contributes \(6\) cross-difference-orbits.

The forced core for the normalized affine copy \(1+A=\{1,2,4,12\}\) is
\[
C=[1]\cup[2]\cup[4]\cup[12].
\]
Its internal and cross-difference orbit labels are as follows:
\[
\begin{aligned}
[1]_{\rm int}&:\ [40],[83],\\
[2]_{\rm int}&:\ [80],[166],\\
[4]_{\rm int}&:\ [160],[332],\\
[12]_{\rm int}&:\ [480],[996],\\
[2]-[1]&:\ [1],[39],[42],[44],[124],[125],\\
[4]-[1]&:\ [3],[37],[46],[126],[206],[209],\\
[4]-[2]&:\ [2],[78],[84],[88],[248],[250],\\
[12]-[1]&:\ [11],[29],[54],[462],[534],[545],\\
[12]-[2]&:\ [10],[70],[96],[420],[576],[586],\\
[12]-[4]&:\ [8],[152],[180],[336],[660],[668].
\end{aligned}
\]
These \(44\) labels are distinct.

\begin{theorem}[The fixed \(q=41\) core for \(A\) is unsatisfiable]
There is no \(41\)-multiplier-invariant cyclic \((1723,42,1)\) difference set \(D\subset \Z_{1723}\) satisfying
\[
[1]\cup[2]\cup[4]\cup[12]\subset D.
\]
Equivalently, the locally admissible forced core coming from \(1+A=\{1,2,4,12\}\) has no \(10\)-orbit completion.
\end{theorem}

\begin{proof}
A \(41\)-multiplier-invariant difference set \(D\) must be a union of nonzero \(41\)-orbits. Since each nonzero orbit has size \(3\), and \(|D|=42\), exactly \(14\) point-orbits must be selected. Four are forced by the core, so a completion requires \(10\) additional point-orbits.

The core already uses \(44\) difference-orbit labels. Since there are \(574\) nonzero difference-orbits in total, a completion must cover exactly \(530\) further labels. Each new point-orbit contributes \(2\) internal labels and \(4\cdot 6=24\) labels against the four core orbits, hence \(26\) labels per new orbit. The \(10\) new orbits therefore contribute
\[
10\cdot 26=260
\]
labels internally and against the core. Their pairwise cross-differences contribute
\[
\binom{10}{2}\cdot 6=270
\]
labels. Thus a completion exists if and only if there are \(10\) additional point-orbits whose \(260+270=530\) labels are all distinct and avoid the \(44\) core labels.

The following deterministic verifier implements exactly this condition. It first constructs all \(574\) nonzero \(41\)-orbits, then constructs the \(44\)-label core mask, then lists all candidate point-orbits whose internal labels and core-cross labels are admissible. It then performs a depth-first exact-cover search. At each recursive node it records the labels already used. A candidate may be added precisely when its internal labels, labels against the core, and labels against all previously selected candidates are defined and disjoint from the used labels. If ten candidates are selected, the program checks that all \(574\) nonzero difference-orbit labels have been used. Otherwise, if fewer remaining candidates are individually admissible than the number still needed, the branch is impossible and is pruned.

The verifier is intentionally literal. No randomized choice, heuristic acceptance, or numerical tolerance is used.

\begin{verbatim}
#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#include <stdint.h>

#define V 1723
#define P 41
#define NLAB 574
#define W 9
#define MAXC 574
#define TARGET 10

typedef struct { uint64_t w[W]; } Mask;

static inline void clear(Mask *m){ memset(m->w,0,sizeof(m->w)); }
static inline void setb(Mask *m,int i){ m->w[i>>6] |= 1ULL<<(i&63); }
static inline int inter(const Mask *a,const Mask *b){
  for(int i=0;i<W;i++) if(a->w[i]&b->w[i]) return 1;
  return 0;
}
static inline void orto(Mask *a,const Mask *b){
  for(int i=0;i<W;i++) a->w[i] |= b->w[i];
}
static inline int popc(const Mask *a){
  int s=0;
  for(int i=0;i<W;i++) s += __builtin_popcountll(a->w[i]);
  return s;
}

int elem[V], orb[NLAB][3], rep[NLAB], norb=0;
Mask internal_label[NLAB];
int cand[MAXC], repc[MAXC], n=0;
Mask node[MAXC];
Mask edge[MAXC][MAXC];
unsigned char ok[MAXC][MAXC];
int selected[TARGET];
unsigned long long nodes=0, prunes=0;

int cmpi(const void *a,const void *b){ return *(int*)a-*(int*)b; }

int oi(int x){
  x %= V;
  if(x<0) x += V;
  return elem[x];
}

Mask internal_mask(int i){
  Mask m; clear(&m);
  for(int a=0;a<3;a++)
    for(int b=0;b<3;b++)
      if(a!=b) setb(&m,oi(orb[i][a]-orb[i][b]));
  return m;
}

Mask cross_mask(int i,int j){
  Mask m; clear(&m);
  for(int a=0;a<3;a++)
    for(int b=0;b<3;b++){
      setb(&m,oi(orb[i][a]-orb[j][b]));
      setb(&m,oi(orb[j][b]-orb[i][a]));
    }
  return m;
}

int can_add(int c,int depth,const Mask *used,Mask *add){
  *add=node[c];
  if(inter(add,used)) return 0;
  for(int i=0;i<depth;i++){
    int s=selected[i];
    if(!ok[c][s]) return 0;
    if(inter(&edge[c][s],used) || inter(&edge[c][s],add)) return 0;
    orto(add,&edge[c][s]);
  }
  return 1;
}

int dfs(int depth,int *pool,int pn,const Mask *used){
  nodes++;
  if(depth==TARGET){
    if(popc(used)==574){
      printf("SOLUTION\n");
      for(int i=0;i<TARGET;i++) printf("%d ",repc[selected[i]]);
      printf("\n");
      return 1;
    }
    return 0;
  }

  int need=TARGET-depth;
  if(pn<need){ prunes++; return 0; }

  for(int idx=0;idx<pn;idx++){
    int c=pool[idx];
    Mask add;
    if(!can_add(c,depth,used,&add)) continue;

    Mask newused=*used;
    orto(&newused,&add);
    selected[depth]=c;

    int newpool[MAXC], np=0;
    for(int j=idx+1;j<pn;j++){
      int u=pool[j];
      Mask au;
      if(can_add(u,depth+1,&newused,&au)) newpool[np++]=u;
    }

    if(np>=TARGET-(depth+1)){
      if(dfs(depth+1,newpool,np,&newused)) return 1;
    } else {
      prunes++;
    }
  }
  return 0;
}

int main(){
  for(int i=0;i<V;i++) elem[i]=-1;

  for(int x=1;x<V;x++) if(elem[x]<0){
    int O[3], y=x;
    for(int k=0;k<3;k++){ O[k]=y; y=(int)((long long)y*P%V); }
    qsort(O,3,sizeof(int),cmpi);
    int id=norb;
    for(int k=0;k<3;k++){ orb[id][k]=O[k]; elem[O[k]]=id; }
    rep[id]=O[0];
    norb++;
  }

  for(int i=0;i<NLAB;i++) internal_label[i]=internal_mask(i);

  int core[4]={oi(1),oi(2),oi(4),oi(12)};
  Mask coremask; clear(&coremask);
  for(int i=0;i<4;i++) orto(&coremask,&internal_label[core[i]]);
  for(int i=0;i<4;i++)
    for(int j=i+1;j<4;j++){
      Mask cm=cross_mask(core[i],core[j]);
      orto(&coremask,&cm);
    }

  for(int i=0;i<NLAB;i++){
    int iscore=0;
    for(int t=0;t<4;t++) if(i==core[t]) iscore=1;
    if(iscore) continue;

    Mask m=internal_label[i];
    if(popc(&m)!=2 || inter(&m,&coremask)) continue;

    int good=1;
    for(int t=0;t<4;t++){
      Mask cm=cross_mask(i,core[t]);
      if(popc(&cm)!=6 || inter(&cm,&coremask) || inter(&cm,&m)){
        good=0;
        break;
      }
      orto(&m,&cm);
    }

    if(good && popc(&m)==26){
      cand[n]=i;
      node[n]=m;
      n++;
    }
  }

  int deg[MAXC]={0};
  for(int a=0;a<n;a++) for(int b=a+1;b<n;b++){
    if(inter(&node[a],&node[b])) continue;
    Mask cm=cross_mask(cand[a],cand[b]);
    if(popc(&cm)!=6 || inter(&cm,&coremask) ||
       inter(&cm,&node[a]) || inter(&cm,&node[b])) continue;
    ok[a][b]=ok[b][a]=1;
    edge[a][b]=edge[b][a]=cm;
    deg[a]++; deg[b]++;
  }

  int order[MAXC];
  for(int i=0;i<n;i++) order[i]=i;
  for(int i=1;i<n;i++){
    int key=order[i], j=i-1;
    while(j>=0 &&
      (deg[order[j]]>deg[key] ||
      (deg[order[j]]==deg[key] && rep[cand[order[j]]]>rep[cand[key]]))){
      order[j+1]=order[j];
      j--;
    }
    order[j+1]=key;
  }

  static Mask newnode[MAXC], newedge[MAXC][MAXC];
  static unsigned char newok[MAXC][MAXC];
  static int newcand[MAXC], newrepc[MAXC];

  for(int i=0;i<n;i++){
    newcand[i]=cand[order[i]];
    newnode[i]=node[order[i]];
    newrepc[i]=rep[newcand[i]];
  }
  for(int i=0;i<n;i++) for(int j=0;j<n;j++){
    newok[i][j]=ok[order[i]][order[j]];
    if(newok[i][j]) newedge[i][j]=edge[order[i]][order[j]];
  }

  memcpy(cand,newcand,sizeof(cand));
  memcpy(node,newnode,sizeof(node));
  memcpy(repc,newrepc,sizeof(repc));
  memcpy(ok,newok,sizeof(ok));
  memcpy(edge,newedge,sizeof(edge));

  printf("nonzero_orbits %d\n",norb);
  printf("core_used %d\n",popc(&coremask));
  printf("candidate_orbits %d\n",n);

  int edgecount=0;
  for(int i=0;i<n;i++) for(int j=i+1;j<n;j++) if(ok[i][j]) edgecount++;
  printf("compatible_pairs %d\n",edgecount);

  int pool[MAXC];
  for(int i=0;i<n;i++) pool[i]=i;

  if(!dfs(0,pool,n,&coremask)){
    printf("UNSAT_SEARCH_COMPLETE nodes %llu prunes %llu\n",nodes,prunes);
  }
  return 0;
}
\end{verbatim}

Compiled with a standard C compiler, this program prints
\begin{verbatim}
nonzero_orbits 574
core_used 44
candidate_orbits 324
compatible_pairs 27054
UNSAT_SEARCH_COMPLETE nodes 59323 prunes 775746
\end{verbatim}

It remains to justify that this finite search is exactly the desired mathematical statement. The program's bit positions are the \(574\) nonzero difference-orbits. The mask \texttt{coremask} is the union of all internal and cross-difference labels generated by the forced core. The value \texttt{core\_used 44} verifies that these labels are distinct.

A candidate point-orbit is accepted into the list \texttt{cand} exactly when its two internal labels and its \(4\cdot 6\) labels against the core are all distinct and avoid the core labels. Hence each candidate contributes exactly \(26\) new labels before interactions with other new candidates. The value \texttt{candidate\_orbits 324} is the number of such individually admissible point-orbits.

For a pair of candidates, \texttt{ok[a][b]} is set precisely when their six cross-difference labels are distinct and avoid the core and both candidate node masks. During recursion, \texttt{can\_add} additionally checks disjointness against all labels already used by previously selected candidates. Thus every recursive branch is exactly a partial exact cover by point-orbits, and every possible ordered \(10\)-orbit completion appears in exactly one branch, namely the branch in which its orbits are chosen in the fixed total order imposed by \texttt{order}. The pruning condition \texttt{pn < need} is valid because \texttt{pool} contains all remaining orbits that can still be added individually to the current partial cover.

Therefore the absence of a branch reaching depth \(10\) with all \(574\) labels used proves that no \(10\)-orbit completion exists. This is precisely the negation of the desired \(41\)-multiplier-invariant cyclic \((1723,42,1)\) difference set containing the forced core.
\end{proof}

\begin{corollary}
At \(q=41\), the normalized affine placement \(1+A=\{1,2,4,12\}\) cannot occur inside any cyclic planar difference set that has been normalized to be invariant under the classical multiplier \(41\).
\end{corollary}

\begin{proof}
A multiplier-normalized difference set containing \(1+A\) must contain the four \(41\)-orbits \([1]\), \([2]\), \([4]\), and \([12]\). This is the forced core of Theorem \(3.3\), which has no completion.
\end{proof}

\section{Discussion / Open questions}

Theorem \(3.1\) removes the Singer case entirely for the two size-four candidates \(A\) and \(B\). Its proof is short enough that it should be straightforward to formalize in a proof assistant: the only field-specific ingredients are nondegeneracy of the trace pairing for finite separable extensions, the fact that an element of \(\F_{q^3}\setminus \F_q\) has irreducible cubic minimal polynomial over \(\F_q\), and the two displayed polynomial remainders.

Theorem \(3.3\) is different in nature. It is a finite certificate for one locally admissible multiplier core at the first open parameter \(q=41\). It does not prove that \(A\) is absent from every multiplier-normalized cyclic planar difference set of order \(41\), because an affine copy \(t+uA\) can lead to other forced cores after normalization. It also says nothing directly about \(B\). The next finite step is therefore clear: enumerate the surviving translation-orbit classes for \(A\) and \(B\) at \(q=41\), run the same exact-cover certificate for each, and store the resulting certificates in a format suitable for independent verification.

There is also a conceptual question. The exact-cover formulation suggests a possible multiplier-descent theorem: if a normalized cyclic planar difference set contains one of the two size-four affine configurations, then the forced orbit system should either collide locally or fail as an exact cover. The fixed core studied here shows that local collision is not enough. The core uses \(44\) distinct difference-orbits and only fails globally. Any general multiplier descent must therefore incorporate higher-order orbit interactions.

Finally, one should separate two statements. The Singer obstruction is now a theorem for every prime power \(q\). The full non-extension of \(A\) and \(B\) would require ruling out all non-Singer cyclic planar difference sets, and possibly all noncyclic perfect difference sets depending on the exact formulation of the Sidon-extension problem. The cyclic projective plane uniqueness conjecture remains open in general, so the Singer theorem alone cannot settle Niu's central problem. It does, however, give a clean algebraic base case for a complete multiplier descent.
