\begin{theorem}[平方根门槛失效完全来自 core 因子]\label{thm:killo-real-input-40-dirichlet-threshold-core-origin}
沿用命题 \ref{prop:real-input-40-output-dirichlet-nonrh} 的记号。对任意整数 \(m\ge 2\) 与 \(1\le j\le m-1\)，若取
\[
u=\omega_m^j=e^{2\pi i j/m},
\]
则因子
\[
1-uz^2
\]
的全部零点都满足 \(|z|=1\)。因此原子因子在单位根扭曲上所能贡献的谱半径恒等于 \(1\)，不可能产生任何满足
\[
\rho_{m,j}>1
\]
的模态。特别地，只要
\[
\theta_{m,j}>\frac12,
\]
等价地
\[
\rho_{m,j}>\lambda^{1/2}=\varphi>1,
\]
\leanverified{paper\_killo\_real\_input\_40\_dirichlet\_threshold\_core\_origin}
则决定 \(\rho_{m,j}\) 的最小模零点必属于 core 因子 \(F(z,\omega_m^j)\)。换言之，全部平方根门槛失效现象都只能由 \(F\) 的零点引起，而与原子因子 \((1-uz^2)\) 无关。
\end{theorem}
\begin{proof}
本小节已给出精确分解
\[
\Delta(z,u)=(1-uz^2)F(z,u).
\]
当 \(|u|=1\) 时，原子方程
\[
1-uz^2=0
\]
等价于
\[
z=\pm u^{-1/2},
\]
故其全部零点都满足
\[
|z|=|u|^{-1/2}=1.
\]
因此原子因子单独对应的最小零点模恒为 \(1\)，所能贡献的谱半径也恒为
\[
1.
\]

另一方面，由命题 \ref{prop:real-input-40-output-dirichlet-nonrh}，
\[
\rho_{m,j}=\frac{1}{\min\{|z|:\Delta(z,\omega_m^j)=0\}},
\qquad
\theta_{m,j}=\frac{\log\rho_{m,j}}{\log\lambda},
\qquad
\lambda=\varphi^2.
\]
若 \(\theta_{m,j}>\frac12\)，则
\[
\rho_{m,j}>\lambda^{1/2}=\varphi>1.
\]
这意味着决定 \(\rho_{m,j}\) 的最小模零点必严格落在单位圆内。由于原子因子的全部零点都在单位圆上，这个最小模零点不可能来自 \(1-uz^2\)，只能来自 \(F(z,\omega_m^j)\)。证毕。
\end{proof}
