\begin{theorem}[饱和端点大偏差率函数的对数尖点]\label{thm:killo-real-input-40-rate-logarithmic-cusp}
沿用命题 \ref{prop:real-input-40-pressure-freezing} 的记号，设
\[
\alpha_{\max}:=\lim_{\theta\to+\infty}P'(\theta)=\frac12,
\qquad
I_{\mathrm{sat}}:=I(\alpha_{\max})=\log\frac{\varphi^2}{c}.
\]
则当
\[
\delta:=\alpha_{\max}-\alpha\downarrow 0
\]
时，大偏差率函数满足精确展开
\[
I\!\left(\frac12-\delta\right)
=
I_{\mathrm{sat}}
+2\delta\log\delta
+2\delta\log\frac{2c}{d}
-2\delta
+O(\delta^2).
\]
特别地，\(I(\alpha)\) 在饱和端点 \(\alpha\uparrow \alpha_{\max}\) 处呈现对数尖点，而非解析二次边缘。
\end{theorem}
\begin{proof}
记
\[
A:=\frac{d}{c},
\qquad
B:=\frac{e}{c}-\frac{d^2}{2c^2}.
\]
命题 \ref{prop:real-input-40-pressure-freezing} 给出
\[
P(\theta)
=
\frac{\theta}{2}+\log c
+Ae^{-\theta/2}
+Be^{-\theta}
+O(e^{-3\theta/2})
\qquad (\theta\to+\infty),
\]
从而
\[
P'(\theta)
=
\frac12-\frac{A}{2}e^{-\theta/2}-Be^{-\theta}+O(e^{-3\theta/2}).
\]
令 \(y:=e^{-\theta/2}\)。则
\[
\delta
=
\frac12-P'(\theta)
=
\frac{A}{2}y+By^2+O(y^3).
\]
由于 \(A=d/c>0\)，该关系可在 \(y=0\) 邻域反解为
\[
y=\frac{2\delta}{A}+O(\delta^2),
\qquad
\theta=-2\log y=-2\log\frac{2\delta}{A}+O(\delta).
\]

另一方面，对 \(\alpha(\theta):=P'(\theta)\) 的 Legendre 对偶有
\[
I(\alpha(\theta))
=
\alpha(\theta)\theta-P(\theta)+P(0).
\]
代入 \(\alpha(\theta)=\frac12-\delta\) 及上式展开，得到
\[
I(\alpha)
=
\left(\frac12-\delta\right)\theta
-\left(
\frac{\theta}{2}+\log c+Ay+By^2+O(y^3)-P(0)
\right),
\]
即
\[
I(\alpha)
=
P(0)-\log c-\delta\theta-Ay+O(y^2).
\]
由推论 \ref{cor:real-input-40-rate-endpoints}，
\(
P(0)-\log c=I_{\mathrm{sat}}
\)。
再代入
\(
\theta=-2\log(2\delta/A)+O(\delta)
\)
与
\(
Ay=2\delta+O(\delta^2)
\)，
即得
\[
I(\alpha)
=
I_{\mathrm{sat}}
+2\delta\log\delta
+2\delta\log\frac{2}{A}
-2\delta
+O(\delta^2).
\]
最后用 \(A=d/c\) 化简，得到
\[
I\!\left(\frac12-\delta\right)
=
I_{\mathrm{sat}}
+2\delta\log\delta
+2\delta\log\frac{2c}{d}
-2\delta
+O(\delta^2).
\]
证毕。
\end{proof}

\begin{theorem}[饱和端点由正熵地基层支配而非有限周期轨道]\label{thm:killo-real-input-40-positive-entropy-freezing}
沿用命题 \ref{prop:real-input-40-pressure-freezing} 及其后零温缩放极限的记号。令 \(B_\infty\) 为满足
\[
\widehat F(w)=\det(I-wB_\infty)
\]
的 \(4\times4\) 非负整数矩阵，则
\[
h_{\mathrm{ground}}
=
\log c
=
\frac12\log\rho(B_\infty)
>
0.
\]
特别地，饱和端点 \(\alpha_{\max}=1/2\) 不可能仅由有限个最大平均能量的周期轨道支撑。并且
\[
I(\alpha_{\max})
=
\log\lambda(1)-h_{\mathrm{ground}}
=
\log\frac{\varphi^2}{c}.
\]
\end{theorem}
\begin{proof}
由零温缩放极限的显式矩阵表示，
\(
c=\sqrt{\rho(B_\infty)}
\)，
故
\[
\frac12\log\rho(B_\infty)=\log c.
\]
再由推论 \ref{cor:real-input-40-ground-entropy}，
\(
h_{\mathrm{ground}}=\log c>0
\)。

若饱和端点仅由有限个最大平均能量的周期轨道支撑，则相应地基层至多是有限个周期轨道的并，其拓扑熵必为 \(0\)。
这与 \(h_{\mathrm{ground}}>0\) 矛盾，故该端点不能由有限个最大平均能量周期轨道实现。

最后，由推论 \ref{cor:real-input-40-rate-endpoints}，
\[
I(\alpha_{\max})
=
I\!\left(\frac12\right)
=
\log\frac{\varphi^2}{c}
=
\log\lambda(1)-\log c
=
\log\lambda(1)-h_{\mathrm{ground}}.
\]
证毕。
\end{proof}
