\begin{theorem}[Lee--Yang 分支图的全谱闭式]\label{thm:killo-leyang-branch-graph-full-spectrum}
\leanverified{paper\_killo\_leyang\_branch\_graph\_full\_spectrum}
沿用命题 \ref{prop:killo-leyang-branchset-hypercube-families} 的记号。
对每个 \(n\ge 1\)，在每个分支平行族 \(\Pi_{i,n}\) 上传递 \(Q_{2n}\) 的 Hamming 邻接关系，并令 \(\mathcal R_n\) 为所得图的离散并。
则 \(\mathcal R_n\) 的图同构类与基点选择无关，且
\[
\mathcal R_n\cong \bigsqcup_{i=1}^{5}Q_{2n}.
\]
若 \(A_{\mathcal R_n}\) 与 \(L_{\mathcal R_n}:=2nI-A_{\mathcal R_n}\) 分别表示其邻接矩阵与 Laplacian，则有：
\begin{enumerate}
  \item 邻接谱为
  \[
  \operatorname{Spec}(A_{\mathcal R_n})
  =
  \left\{
  2n-2k\ \text{with multiplicity }5\binom{2n}{k}
  :\ 0\le k\le 2n
  \right\};
  \]
  \item Laplacian 谱为
  \[
  \operatorname{Spec}(L_{\mathcal R_n})
  =
  \left\{
  2k\ \text{with multiplicity }5\binom{2n}{k}
  :\ 0\le k\le 2n
  \right\};
  \]
  \item 对任意 \(t\ge 0\)，热迹满足
  \[
  \operatorname{Tr}(e^{-tL_{\mathcal R_n}})
  =
  5\sum_{k=0}^{2n}\binom{2n}{k}e^{-2kt}
  =
  5(1+e^{-2t})^{2n};
  \]
  \item 对任意整数 \(\ell\ge 0\)，\(\ell\)-步闭路总数满足
  \[
  \operatorname{Tr}(A_{\mathcal R_n}^{\ell})
  =
  5\sum_{k=0}^{2n}\binom{2n}{k}(2n-2k)^\ell.
  \]
\end{enumerate}
\end{theorem}
\begin{proof}
由命题 \ref{prop:killo-leyang-branchset-hypercube-families}，每个 \(\Pi_{i,n}\) 在选取基点后可识别为 \(Q_{2n}\) 的顶点集，且五个分量彼此不交。
不同基点选择只会将该识别预复合一个超立方体的平移自同构，故得到的图同构类不变。
于是
\[
\mathcal R_n\cong \bigsqcup_{i=1}^{5}Q_{2n}.
\]

而 \(Q_d\) 的经典邻接谱为
\[
\left\{
d-2k\ \text{with multiplicity }\binom{d}{k}
:\ 0\le k\le d
\right\},
\]
故取 \(d=2n\) 并乘上五个连通分量的重数，即得邻接谱公式。
Laplacian 谱由
\[
L_{Q_{2n}}=2nI-A_{Q_{2n}}
\]
立即化为 \(2k\) 同样的重数。

热迹与闭路计数分别是谱指数和与谱幂和，因此
\[
\operatorname{Tr}(e^{-tL_{\mathcal R_n}})
=
5\sum_{k=0}^{2n}\binom{2n}{k}e^{-2kt}
=
5(1+e^{-2t})^{2n},
\]
以及
\[
\operatorname{Tr}(A_{\mathcal R_n}^{\ell})
=
5\sum_{k=0}^{2n}\binom{2n}{k}(2n-2k)^\ell.
\]
证毕。
\end{proof}

\begin{proposition}[Lee--Yang 分支图的全自同构群为超立方体花环积]\label{prop:killo-leyang-branch-graph-automorphism-wreath}
\leanverified{paper\_killo\_leyang\_branch\_graph\_automorphism\_wreath}
在定理 \ref{thm:killo-leyang-branch-graph-full-spectrum} 的图 \(\mathcal R_n\) 上，有自然群同构
\[
\operatorname{Aut}(\mathcal R_n)
\cong
\operatorname{Aut}(Q_{2n})\wr S_5
\cong
\bigl((\ZZ/2\ZZ)^{2n}\rtimes S_{2n}\bigr)\wr S_5.
\]
特别地，
\[
\card{\operatorname{Aut}(\mathcal R_n)}
=
\bigl(2^{2n}(2n)!\bigr)^5\cdot 5!.
\]
\end{proposition}
\begin{proof}
由定理 \ref{thm:killo-leyang-branch-graph-full-spectrum}，
\[
\mathcal R_n\cong \bigsqcup_{i=1}^{5}Q_{2n}.
\]
图自同构必须将连通分量映到连通分量，故首先诱导一个 \(S_5\) 的分量置换；
在每个分量内部，则给出 \(Q_{2n}\) 的图自同构。
因此
\[
\operatorname{Aut}(\mathcal R_n)\cong \operatorname{Aut}(Q_{2n})\wr S_5.
\]
另一方面，\(Q_d\) 的经典自同构群为
\[
\operatorname{Aut}(Q_d)\cong (\ZZ/2\ZZ)^d\rtimes S_d,
\]
其中 \((\ZZ/2\ZZ)^d\) 对应逐坐标翻转，\(S_d\) 对应坐标置换。
取 \(d=2n\) 即得所述结构。
群阶公式则由花环积的标准计数
\[
\card{G\wr S_5}=\card{G}^5\cdot 5!
\]
与
\[
\card{\operatorname{Aut}(Q_{2n})}=2^{2n}(2n)!
\]
直接推出。
\end{proof}
