\subsubsection{物理化主张、量子/时空桥接、Born 规则、Hilbert 闭包、因果锥、曲率极限、Einstein 域与物理解释边界审计}\label{subsubsec:typed-address-biaxial-completion-physicalization-quantum-spacetime-interpretation-boundary-closure}
统计审计给出的是可复核数据证据；物理化审计处理更强的防火墙：
\[
\boxed{
\mathsf{MathematicalCertificate}
\not\Rightarrow
\mathsf{PhysicalLaw}.
}
\]
即使已经构造出 Hilbert 载体、Born 型配对、幺正演化、因果前序、离散 holonomy、Lorentz 结构或 Einstein 型闭包，也只能说明某个形式结构或 admissible 扇区通过了内部审计。要成为物理主张，还必须给出操作协议、单位标定、经验锚点与可证伪拒绝域。

\[
\boxed{
\mathsf{FormalStructure}
\to
\mathsf{OperationalProtocol}
\to
\mathsf{AdmissibleSector}
\to
\mathsf{UnitCalibration}
\to
\mathsf{EmpiricalPrediction}
\to
\mathsf{PhysicalClaim}.
}
\]

\paragraph{三层口径。}
物理化审计首先区分三层。形式层包含
\[
\mathcal H,\quad
\mathcal A,\quad
\rho,\quad
E,\quad
U_t,\quad
\langle\cdot,\cdot\rangle,\quad
g_{\mu\nu},\quad
\nabla,\quad
R_{\mu\nu\rho\sigma}.
\]
操作层包含
\[
\mathsf{Preparation},\quad
\mathsf{Measurement},\quad
\mathsf{Clock},\quad
\mathsf{Rod},\quad
\mathsf{Signal},\quad
\mathsf{ObserverProtocol}.
\]
物理层包含
\[
\mathsf{Mass},\quad
\mathsf{Charge},\quad
\mathsf{Energy},\quad
\mathsf{Momentum},\quad
\mathsf{LightCone},\quad
\mathsf{StressEnergy},\quad
\mathsf{ExperimentalPrediction}.
\]
把形式对象直接当成物理对象时输出 \(\mathsf{FormalAsPhysicalFail}\)。

\begin{definition}[物理化证书]\label{def:typed-address-biaxial-completion-physicalization-certificate}
称
\[
\mathsf{PhysCert}
=
(
\mathsf{FormalObject},
\mathsf{OperationalProtocol},
\mathsf{UnitDictionary},
\mathsf{CalibrationMap},
\mathsf{ObservableMap},
\mathsf{ErrorBudget},
\mathsf{EmpiricalAnchor},
\mathsf{FalsificationRule}

\]
为物理化证书。若缺少核心字段，只能输出 \(\mathsf{MathematicalStructureCandidate}\)、\(\mathsf{PhysicalAnalogy}\)、\(\mathsf{PhysicalSkeleton}\) 或 \(\mathsf{AdmissibleSector}\)，不能输出 \(\mathsf{PhysicalLaw}\)。
\end{definition}

总防火墙为
\[
\boxed{
\mathsf{FormalDerivation}
+
\mathsf{InternalAudit}
\not\Rightarrow
\mathsf{EmpiricalPhysics}.
}
\]
只有 \(\mathsf{PhysCert}\) 与经验审计同时通过后，才允许输出 \(\mathsf{PhysicalModelSupported}\)。

\paragraph{正定核到 Hilbert 载体。}
设 \(\mathcal T\) 为可读时间字或可见态类集合，且
\[
K:\mathcal T\times\mathcal T\to\mathbb C
\]
满足
\[
\sum_{i,j=1}^n\overline{c_i}c_jK(t_i,t_j)\ge0.
\]
令
\[
\mathcal N=\{v:\langle v,v\rangle_K=0\}.
\]
则
\[
\boxed{
\mathcal H_K=\overline{\mathbb C[\mathcal T]/\mathcal N}
}
\]
是 Hilbert 型载体。对应证书为
\[
\boxed{
\mathsf{KernelToHilbertCert}
=
(\mathcal T,K,\mathsf{PSDCheck},\mathcal N,\mathcal H_K).
}
\]
若未证明正定性，输出 \(\mathsf{KernelPSDUnverifiedFail}\)。

Hilbert 载体本身不是物理量子理论。还需声明 states、effects、preparations、measurements 与 probabilities。若写
\[
\mathcal H\text{ exists}
\Rightarrow
\text{quantum physics derived},
\]
输出 \(\mathsf{HilbertAsQuantumPhysicsFail}\)。可通过的较弱结论是 \(\mathsf{HilbertRepresentationPass}\) 或 Hilbert 型操作表示接口。

\paragraph{射影相位与 Born 配对。}
射影态空间为
\[
\mathbb P(\mathcal H)=(\mathcal H\setminus\{0\})/\mathbb C^\times,
\qquad
\psi\sim e^{i\theta}\psi.
\]
若相位来自局部 lift 的粘接 residual，应记录
\[
\boxed{
\mathsf{ProjectivePhaseCert}
=
(\mathsf{LocalLifts},\mathsf{GaugeRelation},\mathsf{Residual},\mathsf{PhaseProjection}).
}
\]
把 residual 相位直接解释为电磁相位、规范势或物理相位时，还需
\[
\mathsf{GaugeGroupIdentification},\quad
\mathsf{CouplingConstant},\quad
\mathsf{ExperimentalObservable}.
\]
否则输出 \(\mathsf{MathematicalPhaseAsPhysicalPhaseFail}\)。

在 \(C^*\)-代数或 von Neumann 代数口径中，态是正归一线性泛函
\[
\rho:\mathcal A\to\mathbb C,\qquad
\rho(A^*A)\ge0,\qquad
\rho(1)=1.
\]
事件为投影或 effect。Born 型内部配对为
\[
\boxed{
p(E|\rho)=\rho(E).
}
\]
在 Hilbert 表示中，
\[
\rho(A)=\operatorname{Tr}(\hat\rho A),\qquad
p(E|\hat\rho)=\operatorname{Tr}(\hat\rho E).
\]
证书为
\[
\boxed{
\mathsf{BornPairingCert}
=
(\mathcal A,\rho,E,\rho(E)\in[0,1],\sigma\text{-additivity}).
}
\]
该配对只说明内部概率规则；它不说明 \(\rho(E)\) 已等于实验频率极限。经验连接还需要
\[
\mathsf{PreparationProtocol},\quad
\mathsf{MeasurementProtocol},\quad
\mathsf{Repeatability},\quad
\mathsf{InstrumentModel},\quad
\mathsf{FrequencyConvergence}.
\]
若把形式 Born 配对当实验 Born 规则，输出 \(\mathsf{FormalBornAsEmpiricalBornFail}\)。内部通过项为 \(\mathsf{FormalBornRepresentationPass}\)，经验通过项为 \(\mathsf{EmpiricalBornCalibrationPass}\)。

\paragraph{幺正、连续时间与 Schr\"odinger 扇区。}
离散时间演化若满足
\[
U^*U=UU^*=I,
\]
则给出
\[
\psi_t=U^t\psi_0.
\]
证书为
\[
\boxed{
\mathsf{DiscreteUnitaryCert}
=
(\mathcal H,U,U^*U=UU^*=I,\mathsf{TimeIndex}).
}
\]
若仅有 \(U^*U=I\) 而无 \(UU^*=I\)，只能输出 \(\mathsf{IsometryEvolutionCert}\)。误称完整幺正时输出 \(\mathsf{IsometryAsUnitaryFail}\)。

连续时间群需要
\[
U_{t+s}=U_tU_s,\qquad
U_0=I,\qquad
t\mapsto U_t\psi\text{ strongly continuous}.
\]
由 Stone 型接口得到自伴生成元 \(H\)：
\[
U_t=e^{-itH}.
\]
证书为
\[
\boxed{
\mathsf{StronglyContinuousUnitaryGroupCert}
=
(\mathcal H,U_t,\mathsf{StrongContinuity},H=H^*).
}
\]
仅给离散 \(U^n\) 却声明连续时间动力学时输出 \(\mathsf{DiscreteAsContinuousTimeFail}\)。

标量 Schr\"odinger 扇区还需
\[
\mathcal H\simeq L^2(M,\nu),\qquad
H=-\frac{1}{2m}\Delta_g+V,
\]
以及
\[
M,\quad g,\quad \nu,\quad \Delta_g,\quad V,\quad \operatorname{Dom}(H),\quad H=H^*.
\]
证书为
\[
\boxed{
\mathsf{SchrodingerSectorCert}
=
(L^2(M,\nu),g,\Delta_g,V,H,\operatorname{Dom}(H),H=H^*).
}
\]
从抽象自伴 \(H\) 直接写成 \(-\Delta+V\) 时输出 \(\mathsf{AbstractHamiltonianAsSchrodingerFail}\)。

\paragraph{复合体系、纠缠与 CP 信道。}
若写
\[
\mathcal H_{AB}=\mathcal H_A\otimes\mathcal H_B,
\]
必须给出 subsystem decomposition、operational independence、local event algebras 与 compatibility ledger。证书为
\[
\boxed{
\mathsf{TensorProductCert}
=
(\mathcal A_A,\mathcal A_B,\mathcal A_{AB},\mathsf{Embedding},\mathsf{Commutation},\mathsf{StateExtension}).
}
\]
仅因对象并列存在而使用张量积时输出 \(\mathsf{TensorProductAssumedFail}\)。

若
\[
\rho_{AB}\ne\sum_kp_k\rho_A^{(k)}\otimes\rho_B^{(k)},
\]
则给出形式纠缠证书
\[
\boxed{
\mathsf{EntanglementCert}
=
(\rho_{AB},\mathcal A_A,\mathcal A_B,\mathsf{SeparabilityTest}).
}
\]
物理非定域性还需 spatial separation、local measurement、Bell scenario 与 detection model。缺失这些字段时输出 \(\mathsf{FormalEntanglementAsPhysicalNonlocalityFail}\)。

量子信道 \(\Phi:\mathcal A\to\mathcal B\) 需完全正：
\[
\Phi\otimes\operatorname{id}_{M_n}\ge0\quad\forall n,
\]
并满足单位保持或迹保持条件。证书为
\[
\boxed{
\mathsf{CPCert}
=
(\Phi,\mathsf{Positivity},\mathsf{CompletePositivity},\mathsf{TraceOrUnitCondition}).
}
\]
只证明正性而误称完全正时输出 \(\mathsf{PositiveAsCompletelyPositiveFail}\)。

\paragraph{同步成本、因果锥与单位标定。}
设事件集合为 \(\mathcal E\)，同步成本为 \(C_{\rm sync}(a,b)\)。允许传播或精化关系记为 \(a\preceq b\)。因果前序证书为
\[
\boxed{
\mathsf{CausalPreorderCert}
=
(\mathcal E,\preceq,\mathsf{Reflexive},\mathsf{Transitive},\mathsf{AntisymmetryOrPreorderLedger}).
}
\]
只有距离函数 \(d(a,b)\) 时，不能自动推出因果序；否则输出 \(\mathsf{MetricAsCausalOrderFail}\)。

若定义锥结构 \(\mathcal C_x\subset T_xM\)，需给出 closed、convex、pointed 或更弱结构的 ledger：
\[
\boxed{
\mathsf{ConeStructureCert}
=
(M,\{\mathcal C_x\},\mathsf{Regularity},\mathsf{CausalityCondition}).
}
\]
传播上界只是 Lorentz 锥的线索，不是 Lorentz 结构本身。把速度上界当 Lorentz 锥时输出 \(\mathsf{SpeedBoundAsLorentzConeFail}\)。

内部最大传播量
\[
c_{\rm int}
=
\sup\frac{\text{synchronization distance}}{\text{synchronization time}}
\]
通常是协议单位量。若要解释为物理光速，必须给出
\[
\boxed{
\mathsf{UnitDictionary}:
(c_{\rm int},\ell_{\rm int},t_{\rm int})
\mapsto
(c_{\rm phys},{\rm m},{\rm s}).
}
\]
否则只能称为内部最大传播不变量。误称光速导出时输出 \(\mathsf{InternalSpeedAsPhysicalLightSpeedFail}\)。

\paragraph{支撑几何、holonomy 与曲率。}
共同支撑比较可给出
\[
S(a,b)=\text{minimal common support cost}
\]
以及 residual distance 或 quasi-distance。证书为
\[
\boxed{
\mathsf{SupportGeometryCert}
=(\mathcal E,S,d,\mathsf{Symmetry},\mathsf{TriangleOrQuasiTriangle}).
}
\]
若无三角性或局部 Euclidean 近似，不能称为 Riemannian 空间；否则输出 \(\mathsf{SupportCostAsMetricFail}\)。

给定覆盖 \(\mathcal U=\{U_i\}\)、局部截面 \(s_i\) 与过渡 residual
\[
g_{ij}:U_i\cap U_j\to G,
\]
闭路 holonomy 为
\[
\operatorname{Hol}(\gamma)=g_{i_0i_1}g_{i_1i_2}\cdots g_{i_ki_0}.
\]
证书为
\[
\boxed{
\mathsf{DiscreteHolonomyCert}
=
(\mathcal U,s_i,g_{ij},\gamma,\operatorname{Hol}(\gamma)).
}
\]
若 holonomy 依赖覆盖，应标记 \(\mathsf{CoverDependent}\)。把 cover artifact 当物理曲率时输出 \(\mathsf{CoverHolonomyAsPhysicalCurvatureFail}\)。

离散曲率候选需面积归一化：
\[
K(\gamma)=\frac{\log\operatorname{Hol}(\gamma)}{A(\gamma)}.
\]
必须说明 \(\log:G\to\mathfrak g\) 是否定义、面积如何标定、mesh limit 是否存在。证书为
\[
\boxed{
\mathsf{DiscreteCurvatureCert}
=
(\operatorname{Hol},A,\log,\mathsf{ScaleLimit}).
}
\]
无面积归一化时输出 \(\mathsf{HolonomyAsCurvatureFail}\)。

\paragraph{连续时空、维数与 Lorentz 结构。}
离散结构到连续流形需要
\[
(M_m,d_m,\mu_m)\to(M,d,\mu)
\]
在 GH、measured GH、Cheeger energy、spectral convergence 或其他已声明拓扑下收敛，并给出 compactness、limit 与 uniqueness ledger。证书为
\[
\boxed{
\mathsf{ContinuumLimitCert}
=
(M_m,d_m,\mu_m,\mathsf{Topology},\mathsf{Compactness},\mathsf{Limit},\mathsf{Uniqueness}).
}
\]
尺度细化本身不推出连续流形；否则输出 \(\mathsf{RefinementAsContinuumFail}\)。

若主张四维，必须指定维数口径：
\[
\dim_{\rm top},\quad
\dim_{\rm Hausdorff},\quad
\dim_{\rm spectral},\quad
\dim_{\rm walk},\quad
\dim_{\rm manifold}.
\]
证书为
\[
\boxed{
\mathsf{DimensionCert}
=
(\mathsf{DimensionType},\mathsf{EstimatorOrProof},\dim=4,\mathsf{ScaleRange}).
}
\]
仅凭坐标数或参数数判断维数时输出 \(\mathsf{CoordinateCountAsDimensionFail}\)。

Lorentz 度量要求
\[
\operatorname{sig}(g)=(-,+,+,+)
\]
或相反约定，并记录 smoothness 与 time orientation：
\[
\boxed{
\mathsf{LorentzMetricCert}
=
(M,g,\mathsf{Smoothness},\mathsf{Signature},\mathsf{TimeOrientation}).
}
\]
因果偏序不自动给出 Lorentz 度量；把二者混同输出 \(\mathsf{CausalOrderAsLorentzMetricFail}\)。

\paragraph{Levi--Civita、曲率张量与 Einstein 型闭包。}
Levi--Civita 连接需满足
\[
\nabla g=0,\qquad T^\nabla=0.
\]
证书为
\[
\boxed{
\mathsf{LeviCivitaCert}
=
(M,g,\nabla,\nabla g=0,T^\nabla=0).
}
\]
离散平行运输 \(P_\gamma\) 只有在证明其连续极限为 \(\mathcal P\exp\int_\gamma\nabla\) 后，才可进入该口径。否则输出 \(\mathsf{DiscreteTransportAsLeviCivitaFail}\)。

曲率张量由
\[
R^\nabla(X,Y)Z
=
\nabla_X\nabla_YZ-\nabla_Y\nabla_XZ-\nabla_{[X,Y]}Z
\]
定义，并诱导 Ricci 与 scalar curvature。证书为
\[
\boxed{
\mathsf{CurvatureTensorCert}
=
(\nabla,R^\nabla,\operatorname{Ric},R,\mathsf{Regularity}).
}
\]
标量 residual 不等于完整曲率张量；否则输出 \(\mathsf{ScalarResidualAsCurvatureTensorFail}\)。

Einstein 型闭包写作
\[
G_{\mu\nu}+\Lambda g_{\mu\nu}=8\pi G T_{\mu\nu},
\qquad
G_{\mu\nu}=R_{\mu\nu}-\frac12Rg_{\mu\nu}.
\]
证书为
\[
\boxed{
\mathsf{EinsteinEquationCert}
=
(M,g,T_{\mu\nu},G,\Lambda,\mathsf{Bianchi},
\nabla^\mu T_{\mu\nu}=0,\mathsf{VariationalOrClosurePrinciple}).
}
\]
离散 residual 直接推出 Einstein 方程时输出 \(\mathsf{ResidualAsEinsteinEquationFail}\)。若 \(T_{\mu\nu}\) 不守恒，输出 \(\mathsf{NonconservedStressEnergyFail}\)。

内部张量 \(T^{\rm int}_{\mu\nu}\) 不是物理 stress-energy，除非给出
\[
\boxed{
\mathsf{StressEnergyPhysicalizationCert}
=
(T^{\rm int},\mathsf{UnitMap},\mathsf{OperationalMeaning},\mathsf{Conservation}).
}
\]
缺少单位和操作解释时输出 \(\mathsf{InternalTensorAsStressEnergyFail}\)。

若方程来自作用量 \(S[g,\Psi]\)，需给出
\[
\boxed{
\mathsf{VariationalCert}
=
(S,\delta S,\mathsf{BoundaryTerms},\mathsf{EulerLagrange}).
}
\]
若不是变分来源，则必须给出 \(\mathsf{ClosurePrincipleCert}\)。二者皆缺失时输出 \(\mathsf{EinsteinEquationWithoutDerivationPrincipleFail}\)。

G_{\mu\nu}=8\pi G\langle\hat T_{\mu\nu}\rangle_\rho.

证书为

\begin{theorem}[形式结构不推出物理定律]\label{thm:typed-address-biaxial-completion-formal-structure-not-physical-law}
若只给出 \(\mathsf{FormalObject}\) 与内部一致性证书 \(\mathsf{InternalCert}\)，则最多推出 \(\mathsf{MathematicalStructureCandidate}\)。要输出 \(\mathsf{PhysicalLaw}\)，还需通过 \(\mathsf{PhysicalizationCertificate}\) 与经验审计。
\end{theorem}

\begin{theorem}[Hilbert 表示边界]\label{thm:typed-address-biaxial-completion-hilbert-representation-boundary}
若正定核 \(K\ge0\) 已验证，并构造 \(\mathcal H_K\)，则可输出 \(\mathsf{HilbertRepresentationPass}\)。除非制备、测量、Born 标定和实验协议同时闭合，不能输出 \(\mathsf{EmpiricalQuantumMechanicsPass}\)。
\end{theorem}

\begin{theorem}[Born 配对边界]\label{thm:typed-address-biaxial-completion-born-pairing-boundary}
若 \(\rho\) 是正归一态且 \(E\) 是投影或 effect，则 \(\rho(E)\in[0,1]\)。该命题只给出内部概率规则；\(\rho(E)\) 等于实验频率需要额外 \(\mathsf{EmpiricalFrequencyBridge}\)。缺失该桥时输出 \(\mathsf{BornFormalOnly}\)。
\end{theorem}

\begin{theorem}[Einstein 域条件边界]\label{thm:typed-address-biaxial-completion-einstein-domain-boundary}
