\subsubsection{热核、谱 \(\zeta\)、Seeley--DeWitt 系数与 counterterm 自动分类}\label{subsubsec:typed-address-biaxial-completion-heat-kernel-zeta-counterterm-classification}
有效作用闭包中的 \(\log\det{}'\mathcal O\) 只有在 determinant、zero mode、regularization、counterterm 与 gravity class 都已证书化后，才是可审计对象。Euclidean 口径把 fluctuation operator 改写为 elliptic Laplace-type operator \(\mathcal P\)，并用热核和谱 \(\zeta\) 函数定义 one-loop determinant：
\[
\boxed{
\log\det{}'_\zeta(\mathcal P/\mu^2)
=
-\zeta'_{\mathcal P}(0;\mu).
}
\]
短时热核展开
\[
\boxed{
{\rm Tr}\,e^{-s\mathcal P}
\sim
(4\pi s)^{-d/2}
\sum_{k\ge0}s^kA_{2k}(\mathcal P)
\qquad(s\downarrow0)
}
\]
把 UV divergence 化为有限个 local invariants。该接口的判定目标是区分
\[
\mathsf{MinimalEinsteinRenormalizationPass},
\qquad
\mathsf{HigherCurvatureRenormalizationPass},
\qquad
\mathsf{FormalDeterminantOnlyPass}.
\]

\paragraph{Laplace-type operator 与符号约定。}
固定 Euclidean convention：
\[
\boxed{
\mathcal P=-(g^{\mu\nu}\nabla_\mu\nabla_\nu+\mathcal E)
}
\]
作用在 vector bundle \(E\to M_E\) 的截面上。connection curvature 为
\[
\Omega_{\mu\nu}:=[\nabla_\mu,\nabla_\nu].
\]
若物理 operator 写成 \(\mathcal P_{\rm phys}=-\nabla^2+\mathcal Q\)，则在该 convention 中 \(\mathcal E=-\mathcal Q\)。例如
\[
\mathcal P=-\nabla^2+m^2+\xi R
\]
对应
\[
\mathcal E=-(m^2+\xi R).
\]
该符号 ledger 必须固定，否则 \(a_2\) 与 \(a_4\) 的符号不可审计。

\begin{definition}[Laplace-type operator 证书]\label{def:typed-address-biaxial-completion-laplace-type-operator-cert}
称 \(\mathsf{LaplaceTypeOperatorCert}\) 为 Laplace-type operator 证书，若它包含 \(M_E,g_E,E,\nabla,\mathcal E,\mathcal P\)、principal-symbol ledger、bundle-metric ledger、connection-curvature ledger、sign-convention ledger 与 round hash。验证器检查
\[
\sigma_2(\mathcal P)(x,\xi)
=
g_E^{\mu\nu}(x)\xi_\mu\xi_\nu\,{\rm Id}_E.
\]
若 \(g_E^{\mu\nu}\xi_\mu\xi_\nu>0\) 对 \(\xi\ne0\) 成立，则 principal symbol elliptic。主符号非 elliptic 时输出 \(\mathsf{EllipticityFail}\)。operator 不是 Laplace type 却使用标准 Seeley--DeWitt 系数时输出 \(\mathsf{OperatorNotLaplaceTypeNull}\)。
\end{definition}

\paragraph{热核与短时展开。}
定义
\[
K(s;x,y)=e^{-s\mathcal P}(x,y),\qquad s>0.
\]
若 \(M_E\) compact without boundary，且 \(\mathcal P\) self-adjoint、elliptic、bounded below，则
\[
{\rm Tr}\,e^{-s\mathcal P}
=
\int_M{\rm tr}_E K(s;x,x)\,d{\rm vol}_g
\]
良定，并有短时渐近展开。局部系数记为 \(a_{2k}(x,\mathcal P)\)，全局系数为
\[
A_{2k}(\mathcal P)
=
\int_M{\rm tr}_E a_{2k}(x,\mathcal P)\,d{\rm vol}_g.
\]
若存在边界，展开包含半整数项
\[
{\rm Tr}\,e^{-s\mathcal P}
\sim
(4\pi s)^{-d/2}
\sum_{j\ge0}s^{j/2}A_j(\mathcal P,\mathcal B).
\]

\begin{definition}[热核存在证书]\label{def:typed-address-biaxial-completion-heat-kernel-existence-cert}
称 \(\mathsf{HeatKernelExistenceCert}\) 为热核存在证书，若它包含 \(\mathcal P,K(s;x,y),{\rm Tr}\,e^{-s\mathcal P}\)、ellipticity/self-adjointness cert、lower-bound ledger、trace-class ledger、boundary-condition ledger、zero-mode ledger 与 round hash。验证器检查 \(e^{-s\mathcal P}\) 是否 trace class，或是否在 local trace density sense 下可定义。连续谱情形必须提交 spectral density 或 relative determinant ledger。heat trace 不存在时输出 \(\mathsf{HeatKernelTraceNull}\)。self-adjoint realization 未说明时输出 \(\mathsf{SelfAdjointnessNull}\)。
\end{definition}

在 convention \(\mathcal P=-(\nabla^2+\mathcal E)\) 下，无边界前三个局部系数为

\[
a_2=\mathcal E+\frac16R\,{\rm Id}_E,

以及

\paragraph{谱 \(\zeta\) determinant。}
若 \(\mathcal P\) positive self-adjoint，并已去除 zero modes，则
\[
\zeta_{\mathcal P}(s)={\rm Tr}'\,\mathcal P^{-s}.
\]
引入 scale \(\mu\)：
\[
\zeta_{\mathcal P}(s;\mu)
=
{\rm Tr}'\,(\mathcal P/\mu^2)^{-s}
=
\mu^{2s}\zeta_{\mathcal P}(s).
\]
Mellin 表示为
\[
\zeta_{\mathcal P}(s)
=
\frac1{\Gamma(s)}
\int_0^\infty t^{s-1}{\rm Tr}'\,e^{-t\mathcal P}\,dt.
\]
determinant 定义为
\[
\boxed{
\log\det{}'_\zeta(\mathcal P/\mu^2)
=
-\zeta'_{\mathcal P}(0;\mu).
}
\]
由于
\[
\zeta'_{\mathcal P}(0;\mu)
=
2\log\mu\,\zeta_{\mathcal P}(0)+\zeta'_{\mathcal P}(0),
\]
scale dependence 为
\[
\mu\frac{d}{d\mu}
\log\det{}'_\zeta(\mathcal P/\mu^2)
=
-2\zeta_{\mathcal P}(0).
\]

\end{definition}

\paragraph{四维 counterterm 分类。}
在 \(d=4\) 中，\(A_0,A_2,A_4\) 分别控制不同 counterterm。\(A_0\) 对应
\[
\int\sqrt g
\]
即 cosmological constant / vacuum energy。\(A_2\) 包含
\[
\int\sqrt g\,R
\]
以及 mass-dependent vacuum terms，对应 \(\Lambda\) 与 \(\kappa^{-1}\) 的 renormalization。\(A_4\) 包含
\[
R^2,\quad
R_{\mu\nu}R^{\mu\nu},\quad
R_{\mu\nu\rho\sigma}R^{\mu\nu\rho\sigma},\quad
\Omega_{\mu\nu}\Omega^{\mu\nu},
\quad
\mathcal E^2,
\quad
R\mathcal E.
\]
因此 generic matter one-loop correction 在四维曲背景上通常要求 curvature-squared counterterms。

\begin{definition}[counterterm 分类证书]\label{def:typed-address-biaxial-completion-counterterm-classification-cert}

是否 finite。若 \(A_4\) 需要 curvature-squared counterterms 但仍声称 exact minimal Einstein closure，输出 \(\mathsf{GravityClassMislabelFail}\)。divergence 不能由 local invariants 吸收时输出 \(\mathsf{DivergenceUnabsorbedFail}\)。
\end{definition}

\]
并且 one-loop divergence 完全由 \(2k\le d\) 的 \(A_{2k}\) 中 local invariants 给出。存在 local counterterm action
\[

\]

+S_{\rm hc}^{\rm ren}

\end{theorem}
\begin{proof}
Laplace-type 与 ellipticity/self-adjointness 证书保证 \(\mathcal P\) 具有适用的 heat-kernel calculus。热核存在证书给出 \({\rm Tr}\,e^{-s\mathcal P}\)，热核渐近与 Seeley--DeWitt 系数证书给出 \(A_{2k}\)。proper-time representation 表明 \(s\downarrow0\) divergence 只由 \(2k\le d\) 的热核系数控制。counterterm 分类证书把这些系数分解到 local invariant basis \(\mathcal I_i[g]\)，因此可选择 local \(S_{\rm ct}\) 抵消 \(W_{\rm div}^{(1)}\)。若 \(A_4\) 包含 dynamical curvature-squared invariants，则有效引力类证书必须标记为 higher-curvature EFT；对 renormalized action 做 metric variation 即得相应高阶半经典方程。若 curvature-squared 系数消失、为 topological 或 boundary 项，或被置于声明截断阶以外，则 minimal pass 的判定成立。证毕。
\end{proof}

该接口的边界是：热核 \(\zeta\) 方法是 Euclidean elliptic calculus，不自动适用于 Lorentzian hyperbolic operator；local coefficients 只分类 UV divergence，不给出 full finite nonlocal action；\(\zeta\) 与 proper-time cutoff 的 finite scheme 可不同；zero/negative modes 会改变 determinant 与 saddle stability；gauge theory 必须包含 ghost determinant；有边界时 bulk coefficients 不足以闭合 renormalization；generic 四维 matter loop 通常要求 curvature-squared counterterms，不能无证书声称 exact minimal Einstein closure。
