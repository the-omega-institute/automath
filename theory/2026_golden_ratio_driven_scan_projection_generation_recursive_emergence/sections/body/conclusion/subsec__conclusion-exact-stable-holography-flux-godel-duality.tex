\subsection{精确全息与稳定全息的指数分离及通量--Gödel 维数对偶}\label{subsec:conclusion-exact-stable-holography-flux-godel-duality}

沿用小节 \ref{subsec:conclusion-fixedresolution-screen-artin-dual-obstruction}、定理 \ref{thm:conclusion-boundary-subcritical-perturbation-second-order-rigidity} 与推论 \ref{cor:spg-stokes-godel-dimension-chain} 的记号。前述结果已经分别给出：fixed-resolution boundary Gödel code 与有限矩盒的双向完备等价、single-layer 线性左逆的指数病态，以及次临界边界扰动下的 dyadic Minkowski 维数与含量刚性。把这些接口合并后，还可进一步抽出一条更清晰的双层分裂：几何 bulk 的精确全息与线性稳定全息并不相容，但同一边界码对体积、维数与 primorial 双对数常数等热力学标量却保留严格鲁棒的极限读出。

\begin{theorem}[精确全息与稳定全息的指数分离]\label{thm:conclusion-exact-vs-stable-holography-exponential-separation}
固定 \(m,n\ge 1\)。在 fixed-resolution dyadic polycube 类上，boundary 数据一方面唯一决定 bulk，另一方面任意线性左逆却都具有指数恶化的稳定常数。更精确地：
\begin{enumerate}
  \item boundary Gödel code 与有限矩盒在固定分辨率上给出可计算双向完备坐标，故 bulk 可以被算术地精确反演；
  \item 若
  \[
  L_{m,n}:C_{n-1}(\mathcal Q_m;\RR)\to C_n(\mathcal Q_m;\RR)
  \]
  满足
  \[
  L_{m,n}\partial_n=\mathrm{id},
  \]
  并且对某个 \(\delta>0\) 的统一重构误差要求存在噪声半径 \(\varepsilon_{m,n}(\delta)\)，则必有
  \[
  \varepsilon_{m,n}(\delta)\le \sqrt{2n}\,\delta\,2^{-m/2}.
  \]
\end{enumerate}
因此，exact holography 在 fixed-resolution 口径下为真，而 uniformly stable holography 则以 \(2^{-m/2}\) 速率指数失效。
\end{theorem}
\begin{proof}
第一条由定理 \ref{thm:conclusion-fixedresolution-boundary-godel-momentbox-equivalence} 给出。第二条正是定理 \ref{thm:conclusion-fixedresolution-exact-completeness-exponential-instability} 的噪声阈值结论。于是，单层边界既可算术地精确编码 bulk，又不可能在统一误差容忍下保持分辨率无关的线性稳定性。证毕。
\end{proof}

\begin{theorem}[算术完备性与数值奇异性的不可兼容]\label{thm:conclusion-boundary-godel-arithmetic-completeness-numerical-singularity}
对每个固定分辨率 \(m\)，dyadic 顶维边界 Gödel 数据同时满足如下两条互不相容的性质：
\begin{enumerate}
  \item 它是算术完备的：经素因子分解与线性时间 bulk 译码后，可唯一回放原始 dyadic polycube；
  \item 它又是数值奇异的：任意线性左逆
  \[
  L_m:C_{n-1}(\mathcal Q_m;\RR)\to C_n(\mathcal Q_m;\RR)
  \]
  都满足
  \[
  \|L_m\|_{2\to 2}\ge \frac{2^{m/2}}{\sqrt{2n}}.
  \]
\end{enumerate}
因此，对任意固定 bulk 误差阈值 \(\delta>0\)，若边界观测写作
\[
b_m=\partial_n x_m+\eta_m,
\]
并希望统一保证
\[
\|L_m b_m-x_m\|_2\le \delta,
\]
则必要条件是噪声预算满足
\[
\|\eta_m\|_2\le \sqrt{2n}\,\delta\,2^{-m/2}.
\]
换言之，boundary Gödel holography 可以做到完全可逆，却不可能同时做到分辨率无关的稳定反演。
\end{theorem}
\begin{proof}
由定理 \ref{thm:conclusion-fixedresolution-boundary-godel-momentbox-equivalence}，boundary Gödel code 经素因子分解可唯一恢复边界链；再结合正文中的线性时间 bulk 译码，便得到算术型 exact reconstruction。

另一方面，定理 \ref{thm:conclusion-fixedresolution-exact-completeness-exponential-instability} 已给出
\[
\|L_m\|_{2\to 2}\ge \frac{2^{m/2}}{\sqrt{2n}}
\]
对任意线性左逆都成立。若要求
\[
\|L_m(\partial_n x_m+\eta_m)-x_m\|_2=\|L_m\eta_m\|_2\le \delta
\]
统一成立，则必要有
\[
\|\eta_m\|_2\le \delta\,\|L_m\|_{2\to 2}^{-1}\le \sqrt{2n}\,\delta\,2^{-m/2}.
\]
故 exact invertibility 与 uniform numerical robustness 不能兼容。证毕。
\end{proof}

\begin{corollary}[有限精度全息的三性不可同得]\label{cor:conclusion-finite-precision-holography-triple-incompatibility}
\leanverified{paper\_conclusion\_finite\_precision\_holography\_triple\_incompatibility}
不存在一族与分辨率 \(m\) 无关的有限精度边界协议，能够同时满足下列三项：
\begin{enumerate}
  \item 对全部 dyadic polycube 精确恢复；
  \item 对固定幅度噪声一致稳定；
  \item 反演复杂度在分辨率上仅作多项式增长。
\end{enumerate}
换言之，在 boundary holography 中，exactness、uniform stability 与 finite-precision polynomial-time inversion 不能三者兼得。
\end{corollary}
\begin{proof}
若前两条同时成立，则与定理 \ref{thm:conclusion-boundary-godel-arithmetic-completeness-numerical-singularity} 的指数病态下界矛盾，因为该定理已表明任何精确线性左逆的噪声容忍半径都必须按
\[
\sqrt{2n}\,\delta\,2^{-m/2}
\]
指数收缩。若再加入第三条，则意味着试图在统一精度层上实现稳定且精确的分辨率族反演，矛盾只会更强。故三者不可兼得。证毕。
\end{proof}

\begin{theorem}[有限精确几何与渐近标量充分性的严格分层]\label{thm:conclusion-boundary-finite-exactness-vs-asymptotic-scalar-sufficiency}
设 \(A\subset I^n\) 为紧集，\(U_m(A)\) 为其 dyadic 外逼近。则存在如下严格分层：
\begin{enumerate}
  \item 全边界 Gödel 码 \(H_m(U_m(A))\) 唯一决定每一个有限尺度近似 \(U_m(A)\)；
  \item 单一 Stokes 探针
  \[
  \int_{\partial U_m(A)}\omega_0
  \]
  只在 \(m\to\infty\) 的极限中恢复 dyadic Minkowski 维数；
  \item 若进一步存在
  \[
  \operatorname{Vol}(U_m(A))=M\,2^{-m(n-d)}(1+o(1)),
  \]
  则同一单一探针还恢复 \(d\) 与 \(M\)。
\end{enumerate}
因此，有限精确几何与渐近维--含量几何不是同一充分统计；前者要求完整 boundary code，后者只要求单一标量通量的极限行为。
\end{theorem}
\begin{proof}
第一条由定理 \ref{thm:conclusion-boundary-godel-arithmetic-completeness-numerical-singularity} 的算术完备性部分立即给出：在每个固定分辨率 \(m\)，boundary Gödel 码经素因子分解与 bulk 译码可唯一恢复 \(U_m(A)\)。

第二条由定理 \ref{thm:conclusion-flux-godel-dimension-duality} 给出：单一 Stokes 探针只通过
\[
n+\limsup_{m\to\infty}\frac{\log\!\left(\int_{\partial U_m(A)}\omega_0\right)}{m\log 2}
\]
读取 dyadic Minkowski 维数，因此本质上是渐近标量读出。

第三条则来自体积渐近与 Stokes 恒等式
\[
\int_{\partial U_m(A)}\omega_0=\operatorname{Vol}(U_m(A)).
\]
一旦
\[
\operatorname{Vol}(U_m(A))=M\,2^{-m(n-d)}(1+o(1)),
\]
成立，其对数斜率恢复 \(d\)，二阶常数恢复 \(M\)。故完整 finite-scale 几何与 asymptotic scalar geometry 严格分层。证毕。
\end{proof}

\begin{theorem}[几何全息病态与热力学标量刚性的分离]\label{thm:conclusion-geometric-holography-instability-vs-scalar-rigidity}
设 \(A\subset I^n\) 满足
\[
N_m(A)=M\,2^{md}+o(2^{md}),
\qquad
M\in(0,\infty),\ d\in(0,n],
\]
并令 \(U_m(A)\) 为其 dyadic 外逼近。若另一列同分辨率 dyadic polycube \(V_m\) 满足
\[
\operatorname{Dist}_m\bigl(U_m(A),V_m\bigr)=o\!\bigl(2^{m(d-1)}\bigr),
\]
则尽管完整 bulk 几何的 single-layer 重构对边界噪声指数不稳定，以下热力学标量仍保持刚性：
\begin{enumerate}
  \item \(V_m\) 与 \(U_m(A)\) 具有相同的 dyadic Minkowski 维数 \(d\) 与相同的 dyadic Minkowski 含量 \(M\)；
  \item 通用 Stokes 探针的通量满足
  \[
  \frac{\int_{\partial V_m}\omega_0}{\int_{\partial U_m(A)}\omega_0}\longrightarrow 1;
  \]
  \item 相应 primorial Gödel 双对数的一阶斜率与二阶正规化常数保持不变。
\end{enumerate}
因此，同一边界码本在几何重构上指数病态，却在体积、维数与二阶 Gödel 常数等热力学标量上具有鲁棒极限。
\end{theorem}
\begin{proof}
由定理 \ref{thm:conclusion-boundary-subcritical-perturbation-second-order-rigidity}，题设的次临界距离假设推出
\[
N_m(V_m)=M\,2^{md}+o(2^{md}),
\]
故 \(V_m\) 与 \(U_m(A)\) 具有相同的 dyadic Minkowski 维数与含量，并且由同一定理，相关 primorial 双对数的一阶斜率与二阶正规化常数完全一致。

再由
\[
\operatorname{Vol}(U_m(A))=M\,2^{-m(n-d)}+o\!\bigl(2^{-m(n-d)}\bigr),
\qquad
\operatorname{Vol}(V_m)=M\,2^{-m(n-d)}+o\!\bigl(2^{-m(n-d)}\bigr),
\]
可得
\[
\frac{\operatorname{Vol}(V_m)}{\operatorname{Vol}(U_m(A))}\to 1.
\]
最后利用通用 Stokes 探针的体积恒等式
\[
\operatorname{Vol}(W)=\int_{\partial W}\omega_0,
\]
便得到
\[
\frac{\int_{\partial V_m}\omega_0}{\int_{\partial U_m(A)}\omega_0}\to 1.
\]
与此同时，定理 \ref{thm:conclusion-exact-vs-stable-holography-exponential-separation} 已说明完整 bulk 的线性重构噪声预算必须以 \(2^{-m/2}\) 指数收缩，故几何层病态与标量层刚性严格并存。证毕。
\end{proof}

\begin{theorem}[边界 Gödel 全息的闭包刚性与指数刚性分叉]\label{thm:conclusion-boundary-godel-holography-closure-vs-exponent-bifurcation}
设 \(A\subset I^n\) 为紧集，\(U_m(A)\) 为其 dyadic 外逼近，\(H_m(\cdot)\) 为边界 Gödel 码。对任意 dyadic polycube 序列 \(V_m\)，记
\[
d_{\omega_0}(V_\bullet)
:=
n+\limsup_{m\to\infty}
\frac{\log\!\left(\int_{\partial V_m}\omega_0\right)}{m\log 2}.
\]
则存在严格的双层分叉：
\begin{enumerate}
  \item 若
  \[
  H_m(V_m)=H_m(U_m(A))
  \qquad (m\gg 1),
  \]
  则
  \[
  V_m=U_m(A)
  \qquad (m\gg 1),
  \]
  从而
  \[
  \bigcap_{m\gg 1}V_m=\overline A.
  \]
  这给出闭包刚性。
  \item 若
  \[
  N_m(A)=M\,2^{md}+o(2^{md}),
  \qquad
  M\in(0,\infty),\ d\in(0,n],
  \]
  且
  \[
  \operatorname{Dist}_m\bigl(U_m(A),V_m\bigr)=o\!\bigl(2^{m(d-1)}\bigr),
  \]
  则
  \[
  d_{\omega_0}(V_\bullet)=d_{\omega_0}(A).
  \]
  这只给出由单一 Stokes 探针读出的指数刚性。
\end{enumerate}
因此，精确 boundary equality 强制 bulk 闭包刚性，而次临界 \(\operatorname{Dist}_m\) 扰动只控制通量指数；两种 rigid regime 并不坍缩为同一恢复口径。
\end{theorem}
\begin{proof}
若 \(H_m(V_m)=H_m(U_m(A))\) 对充分大 \(m\) 成立，则由定理 \ref{thm:conclusion-fixedresolution-boundary-godel-momentbox-equivalence} 的单射性，立得
\[
V_m=U_m(A)
\qquad (m\gg 1).
\]
再由 dyadic 外逼近的定义
\[
\overline A=\bigcap_{m\ge 1}U_m(A)
\]
得到
\[
\bigcap_{m\gg 1}V_m=\overline A.
\]

另一方面，若
\[
\operatorname{Dist}_m\bigl(U_m(A),V_m\bigr)=o\!\bigl(2^{m(d-1)}\bigr),
\]
则定理 \ref{thm:conclusion-geometric-holography-instability-vs-scalar-rigidity} 已经给出
\[
\frac{\int_{\partial V_m}\omega_0}{\int_{\partial U_m(A)}\omega_0}\to 1,
\]
从而
\[
\limsup_{m\to\infty}
\frac{\log\!\left(\int_{\partial V_m}\omega_0\right)}{m\log 2}
=
\limsup_{m\to\infty}
\frac{\log\!\left(\int_{\partial U_m(A)}\omega_0\right)}{m\log 2}.
\]
也即
\[
d_{\omega_0}(V_\bullet)=d_{\omega_0}(A).
\]
故第二种假设只强制通量指数不变，而第一种假设则强制 eventual equality 与闭包刚性。证毕。
\end{proof}

\begin{theorem}[共尾分辨率子序列上的边界 Gödel 码已经决定拓扑闭包]\label{thm:conclusion-boundary-godel-cofinal-subsequence-determines-closure}
设 \(A\subset [0,1]^n\) 任意，\(U_m(A)\) 为其尺度 \(2^{-m}\) 的 dyadic 外逼近，\(H_m(A)\) 为相应边界 Gödel 整数。
取任意严格递增且满足 \(m_j\to\infty\) 的子序列 \((m_j)_{j\ge 1}\)。
则仅凭子序列
\[
\{H_{m_j}(A)\}_{j\ge 1}
\]
就已唯一决定 \(\overline A\)。
\end{theorem}
\begin{proof}
对每个 \(j\)，\(H_{m_j}(A)\) 经素因子分解唯一决定边界链，从而唯一决定 dyadic polycube \(U_{m_j}(A)\) 本身。
又由于 \(m_j\to\infty\)，有嵌套
\[
U_{m_{j+1}}(A)\subseteq U_{m_j}(A).
\]
显然
\[
\overline A\subseteq \bigcap_{j\ge1}U_{m_j}(A).
\]
反之，若 \(x\notin \overline A\)，取
\[
\varepsilon:=\operatorname{dist}(x,\overline A)>0.
\]
选 \(j\) 足够大使 \(\sqrt n\,2^{-m_j}<\varepsilon\)。
则包含 \(x\) 的边长 \(2^{-m_j}\) dyadic 立方体不可能与 \(A\) 相交，故 \(x\notin U_{m_j}(A)\)。
于是
\[
\bigcap_{j\ge1}U_{m_j}(A)=\overline A.
\]
因此，共尾子序列 \(\{H_{m_j}(A)\}\) 已唯一决定 \(\overline A\)。证毕。
\end{proof}

\begin{theorem}[边界 Gödel 塔是闭包与 dyadic 几何的完备算术表示]\label{thm:conclusion-boundary-godel-tower-complete-arithmetic-representation}

d_m^{\mathrm{arith}}\bigl(H_m(U),H_m(V)\bigr)

并满足体积读出的严格 Lipschitz 控制

故一条平方自由整数塔已经足以承载闭包、体积、边界通量与 dyadic 维数的全部可见信息。
\end{theorem}
\begin{proof}
对每个固定尺度 \(m\)，定理 \ref{thm:conclusion-boundary-godel-syndrome-completeness-linear-decode} 说明：边界 Gödel 码 \(H_m(U_m(A))\) 经素因子分解唯一恢复边界链，进而唯一恢复 dyadic polycube \(U_m(A)\)。
故整个塔 \(\mathcal H(A)\) 唯一决定全部 \(\{U_m(A)\}_{m\ge1}\)。

再由
\[
\overline A=\bigcap_{m\ge1}U_m(A)

与定理 \ref{thm:conclusion-boundary-godel-cofinal-subsequence-determines-closure} 的同类论证，\(\mathcal H(A)\) 唯一决定 \(\overline A\)。
而每个 \(U_m(A)\) 一旦确定，其体积 \(\operatorname{Vol}(U_m(A))\) 也随之确定；由通用 Stokes 探针恒等式

\operatorname{Vol}(U_m(A))=\int_{\partial U_m(A)}\omega_0

可知边界通量同样由 \(\mathcal H(A)\) 唯一决定。
最后由定理 \ref{thm:conclusion-flux-godel-dimension-duality}，

d_G(A)=d_V(A)
=
n+\limsup_{m\to\infty}\frac{\log\!\left(\int_{\partial U_m(A)}\omega_0\right)}{m\log 2},

故 dyadic 维数也由 \(\mathcal H(A)\) 唯一恢复；若 dyadic Minkowski 维数存在，则该共同值就是 \(\dim_M(A)\)。

对同尺度 \(U,V\)，定理 \ref{thm:conclusion-dyadic-boundary-code-dual-realization-parameters} 已给出
\[
d_m^{\mathrm{arith}}\bigl(H_m(U),H_m(V)\bigr)
=
\operatorname{Dist}_m(U,V)
=
\|\partial_n[U]-\partial_n[V]\|_{\ell^1}.
\]
再由定理 \ref{thm:conclusion-strong-holography-lipschitz-stability}，

\operatorname{Vol}(U\triangle V)\le 2^{-m(n-1)}\,\operatorname{Dist}_m(U,V).
\]
由于

\le

\]
便得到所示 Lipschitz 控制。证毕。
\end{proof}

\begin{theorem}[通量--Gödel 的 dyadic 维数对偶]\label{thm:conclusion-flux-godel-dimension-duality}
\leanverified{paper\_conclusion\_flux\_godel\_dimension\_duality}
对任意紧集 \(A\subset I^n\)，记 \(U_m(A)\) 为其 dyadic 外逼近，\(G_m^\ast(A)\) 为相应 primorial Gödel 码，并记 \(\omega_0\) 为通用 Stokes 探针。则有严格等式
\[
d_G(A)=d_V(A)
=
n+\limsup_{m\to\infty}\frac{\log\!\left(\int_{\partial U_m(A)}\omega_0\right)}{m\log 2}
=
\limsup_{m\to\infty}\frac{\log\log G_m^\ast(A)}{m\log 2}.
\]
若 \(A\) 的 dyadic Minkowski 维数存在，则上述共同值即等于该维数。因而，同一个分形指数可由两类完全不同的边界对象恢复：一个单一的 Stokes 通量标量，以及一个单一算术 Gödel 整数的双对数增长率。
\end{theorem}
\begin{proof}
这正是推论 \ref{cor:spg-stokes-godel-dimension-chain} 的直接改写。该推论已经把单一 Stokes 探针的通量读出、体积指数以及 primorial Gödel 双对数增长率统一到同一 dyadic 维数 \(d_G(A)=d_V(A)\) 上。若 dyadic Minkowski 维数存在，则正文关于 dyadic 外逼近体积指数的定义说明该共同值即为该维数本身。证毕。
\end{proof}

\begin{corollary}[边界全息的临界误差阈值]\label{cor:conclusion-boundary-holography-critical-error-threshold}
设 \(A\subset I^n\) 的 dyadic Minkowski 维数为 \(d\in(0,n]\)，并满足
\[
\operatorname{Vol}(U_m(A))\ge C_-\,2^{-m(n-d)}
\]
对充分大 \(m\) 成立。则边界误差的真正临界尺度是
\[
2^{m(d-1)}.
\]
更精确地，若一列 dyadic polycube \(V_m\) 满足
\[
\operatorname{Dist}_m\bigl(U_m(A),V_m\bigr)=o\!\bigl(2^{m(d-1)}\bigr),
\]
则对 Stokes--Gödel 观察者而言，这种扰动在维数层面不可见：\(V_m\) 与 \(U_m(A)\) 给出相同的 dyadic 维数读出。
\end{corollary}
\begin{proof}
这正是定理 \ref{thm:conclusion-geometric-holography-instability-vs-scalar-rigidity} 与定理 \ref{thm:conclusion-flux-godel-dimension-duality} 的合并重述。前者说明在
\[
\operatorname{Dist}_m=o\!\bigl(2^{m(d-1)}\bigr)
\]
的次临界 regime 中，通量与 primorial Gödel 双对数的一阶指数保持刚性；后者把这两个读出统一识别为同一个 dyadic 维数。故 \(2^{m(d-1)}\) 正是 boundary holography 在维数层面的内生临界误差尺度。证毕。
\end{proof}

\begin{theorem}[边界全息的精确、标量与稳定三层分岔]\label{thm:conclusion-boundary-holography-three-layer-splitting}
在 dyadic body holography 中，存在三种严格不同且不可互相压缩的边界读出层：
\begin{enumerate}
  \item 全 combinatorial 精确层：boundary Gödel 码唯一恢复整个 polycube；
  \item 标量几何层：单一 Stokes 探针 \(\omega_0\) 与 primorial Gödel 双对数只恢复体积、Minkowski 维与含量；
  \item 解析线性层：任何在全体上精确的线性左逆都具有至少 \(2^{m/2}\) 级的条件数发散。
\end{enumerate}
更形式地，不存在一族线性算子
\[
L_m:C_{n-1}(\mathcal Q_m;\RR)\to C_n(\mathcal Q_m;\RR)
\]
能够同时满足
\[
L_m\partial_n=I\quad\text{在全部 dyadic polycube 上},
\qquad
\sup_m\|L_m\|_{2\to2}<\infty.
\]
然而，与此相对，exact arithmetic holography 与 scalar asymptotic holography 都确实存在。因而在本文框架中，
\[
\text{全精确恢复可行},
\qquad
\text{标量几何恢复可行},
\qquad
\text{uniformly stable 的线性全恢复不可行}.
\]
\end{theorem}
\begin{proof}
定理 \ref{thm:conclusion-boundary-godel-arithmetic-completeness-numerical-singularity} 已经说明：boundary Gödel 码经素因子分解与线性时间 bulk 译码可精确恢复整个 polycube，这给出第一层。

定理 \ref{thm:conclusion-geometric-holography-instability-vs-scalar-rigidity} 与定理 \ref{thm:conclusion-flux-godel-dimension-duality} 则说明：同一边界数据仍能稳定恢复单一 Stokes 通量、dyadic Minkowski 维数及其二阶含量常数，这给出第二层。

另一方面，由定理 \ref{thm:conclusion-boundary-godel-arithmetic-completeness-numerical-singularity}，
\[
\|L_m\|_{2\to2}\ge \frac{2^{m/2}}{\sqrt{2n}}
\]
对任意精确线性左逆都成立，故 uniformly bounded 的 linear holography 不可能存在。这就是第三层的否定结论。证毕。
\end{proof}

\begin{theorem}[exact screen 的拟阵交换刚性与余维一全息饱和]\label{thm:conclusion-exact-screen-exchange-rigidity-codim1-saturation}
在 dyadic body holography 的 fixed-resolution 口径下，exact screen 与边界双对数维数同时满足以下两组刚性：
\begin{enumerate}
  \item 对每个固定 \(m,n\)，任一极小精确屏幕都具有基数
        \[
        2^{mn}=\dim_{\QQ}C_n(\mathcal Q_m;\QQ),
        \]
        即恰等于 ambient dyadic complex 的顶维胞腔数；
  \item 达到该极小值的全部屏幕，恰为边界行拟阵的全部基；
  \item 任意两组极小精确屏幕之间都可由单次基交换连通；等价地，存在链
        \[
        B=B_0,B_1,\dots,B_\ell=B'
        \]
        使每一步都满足
        \[
        |B_j\triangle B_{j+1}|=2;
        \]
  \item 审计缺口 \(a(S)\) 在边界算术超立方体上满足尖锐的 \(1\)-Lipschitz 刚性：
        \[
        |a(S)-a(T)|
        \le
        d_m^{\mathrm{arith}}\bigl(\mathcal G(S),\mathcal G(T)\bigr);
        \]
  \item 对任意一族 \(A_n\subset[0,1]^n\) 若
        \[
        \sup_n d_G(A_n)<\infty,
        \]
        则
        \[
        0\le d_G(A_n)-d_{\partial G}(A_n)\le \frac{1}{n}d_G(A_n)\xrightarrow[n\to\infty]{}0.
        \]
\end{enumerate}
因此，exact screen 不是任意的“最小边界子集”，而是一个受拟阵交换律控制的刚性对象；与此同时，在有界复杂度的高维族上，codimension-one 边界双对数损失消失，从而出现严格的余维一全息饱和。
\end{theorem}
\begin{proof}
第 (1) 与第 (2) 条由定理 \ref{thm:xi-time-part9fa-exact-screen-minimal-size-matroid-basis} 直接给出：极小精确屏幕的最小规模恰为
\(
2^{mn}
\)，并且达到该极小值者恰为边界行拟阵的全部基。

第 (3) 与第 (4) 条由定理 \ref{thm:conclusion-screen-audit-gap-arithmetic-lipschitz} 给出：其一证明了极小精确屏幕之间由单次基交换连通，其二证明了审计缺口在边界算术超立方体上的 \(1\)-Lipschitz 刚性且常数 sharp。

第 (5) 条正是命题 \ref{prop:xi-time-part9f-boundary-holographic-saturation} 的内容。故在 \(d_G(A_n)\) 有界的高维族上，
\(
d_G(A_n)-d_{\partial G}(A_n)\to 0
\)，
也即边界双对数维数渐近饱和体维。

将这五条并读，即得 exact screen 的拟阵交换刚性与余维一全息饱和。证毕。
\end{proof}

\begin{conjecture}[非线性全息稳定性同样指数受阻]\label{conj:conclusion-boundary-holography-nonlinear-lipschitz-instability}
固定 \(m,n\ge 1\)。设
\[
\mathcal D_m:\operatorname{Im}\partial_n\to C_n(\mathcal Q_m;\RR)
\]
是一族在所有 dyadic polycube 上都精确的解码器，即
\[
\mathcal D_m(\partial_n[U])=[U].
\]
则应存在仅依赖于 \(n\) 的常数 \(c_n>0\)，使得 \(\mathcal D_m\) 的局部 Lipschitz 常数在某些边界点附近满足
\[
\operatorname{Lip}_{\mathrm{loc}}(\mathcal D_m)\ge c_n\,2^{m/2}.
\]
换言之，定理 \ref{thm:conclusion-boundary-godel-arithmetic-completeness-numerical-singularity} 中的 \(2^{m/2}\)-级病态性并非线性代数偶然，而是 codimension-one body holography 本身的非线性障碍。若该猜想成立，则 exact Gödel decoder 之所以成功，正因为它放弃了 uniformly stable 的几何连续性，而转向算术离散性与因子分解。
\end{conjecture}

\paragraph{统一读法}
这一组结果把 boundary holography 的三种看似矛盾的性质压成一条一致链：单层上，boundary code 的确足以精确恢复 bulk；但只要坚持线性左逆，噪声容忍半径就必须按 \(2^{-m/2}\) 指数塌缩。与此同时，若只要求读取体积、维数与双对数常数这类热力学标量，则同一边界码在次临界 \(\operatorname{Dist}_m\) 扰动下又保持严格刚性。最终，单一 Stokes 通量与单一 primorial Gödel 整数的双对数增长率还恢复出同一个 dyadic 维数指数。故本文的 boundary semantics 并不把“全息”简单分成可行或不可行两类；更精确的事实是：几何重构在单层上精确但不稳，标量读出则在极限上稳而且可由通量与算术两种互异坐标等价表达。
