\begin{theorem}[Machin--Chudnovsky 标量统摄核不存在]\label{thm:conclusion-oddtail-no-scalar-unifying-pi-kernel}
不存在任何单一的有限深度标量范式，能够在项目语义中同时忠实承载下列三侧结构：
\[
\text{fold-\(\pi\) 侧的有限阶正规性},
\qquad
\text{resonance shadow 的非 \(D\)-有限性},
\qquad
\text{canonical host 的 Wythoff--Nevanlinna 数据}.
\]
更具体地说，任何试图以单一标量 continued fraction、固定阶超几何递推、或 eventually periodic 的标量 Jacobi 背景，同时桥接 fold-\(\pi\) odd tail 与 resonance shadow 的方案都必然失败；任何真正精确的桥接都必须越出标量范式，而进入 block Jacobi 或算子值 continued-fraction 层。
\leanverified{paper\_conclusion\_oddtail\_no\_scalar\_unifying\_pi\_kernel}
\end{theorem}
\begin{proof}
正文已经分别证明了三件彼此不相容的事实。

第一，fold-\(\pi\) 侧的规范化 odd-tail 矩列来自一个二原子矩模型，因而其生成函数是有限阶 holonomic 对象，并且 Hankel 秩恰为 \(2\)。

第二，resonance 侧的 shadow 是一个具有无限纯点支撑的 Stieltjes 型对象；其正规化生成函数具有无限极点结构，并且正文已明确证明它不是 \(D\)-有限，也不满足任何固定阶 \(P\)-recursive 关系。

第三，canonical compact Jacobi host 还满足更强的背景刚性：正文已排除一切 eventually periodic / finite-gap 的标量 Jacobi 尾背景，因此任何试图用有限 gap 的标量 continued-fraction 正规形来充当统一宿主的方案都不可能成功。

若存在题设中的单一有限深度标量统摄核，则它在一侧必须产生有限阶 holonomic 影子，而在另一侧又必须产生非 \(D\)-有限 shadow；同时它还必须与 canonical host 的非 finite-gap 背景兼容。这与上述三条正文结论直接矛盾。故唯一剩余的可能性只能是非标量提升，即 block Jacobi 或算子值 continued fraction。证毕。
\end{proof}

\begin{theorem}[有序 Nevanlinna 数据而非谱支撑构成完备不变量]\label{thm:conclusion-oddtail-ordered-nevanlinna-completeness}
设 \((J,v)\) 为紧自伴 cyclic Jacobi 对，并记其 Nevanlinna 函数
\[
m_{J,v}(z):=\langle v,(I-zJ)^{-1}v\rangle.
\]
若 \((J,v)\) 与 canonical host \((J_\varphi,v_0)\) 具有相同的有序极点序列
\[
p_{\lfloor n\varphi\rfloor}=\frac{4\pi^2n^2}{\varphi^2},
\qquad
p_{\lfloor n\varphi^2\rfloor}=4\pi^2n^2,
\qquad
n\ge 1,
\]
并且在两条 Wythoff 分支上具有同样的常值留数
\[
\operatorname*{Res}_{p_{\lfloor n\varphi\rfloor}}m_{J,v}=-4\varphi^{-3},
\qquad
\operatorname*{Res}_{p_{\lfloor n\varphi^2\rfloor}}m_{J,v}=-4\varphi^{-1},
\]
则 \((J,v)\) 与 \((J_\varphi,v_0)\) 必酉等价。反之，仅有支撑集合
\[
\Bigl\{\frac{\varphi^2}{4\pi^2n^2}:n\ge 1\Bigr\}
\cup
\Bigl\{\frac{1}{4\pi^2n^2}:n\ge 1\Bigr\}
\]
相同并不足以推出宿主等价。
\leanverified{paper\_conclusion\_oddtail\_ordered\_nevanlinna\_completeness}
\end{theorem}
\begin{proof}
正文已经证明：canonical host 的紧谱不只是两条 reciprocal-square 谱塔的并，而且其有序排序精确 obey Wythoff 分拆；进一步，极点顺序与 branchwise 常值留数共同构成 \((J,v)\) 的完整 cyclic Nevanlinna 数据，并且这一数据唯一决定其酉等价类。故若 \((J,v)\) 与 \((J_\varphi,v_0)\) 共享同样的有序极点与分支常值留数，则二者必酉等价。

另一方面，单纯的谱支撑集合只记录极点位置，而不记录这些极点在 Nevanlinna 表示中的留数权重。对固定离散支撑，谱定理允许赋予不同的正原子权，从而得到不同的 cyclic 自伴对；这些对象的 \(m\)-函数不同，因而不可能都与 \((J_\varphi,v_0)\) 酉等价。故支撑集合本身并不是完备不变量。证毕。
\end{proof}

\begin{theorem}[有限局部代数对零点几何绝对失明]\label{thm:conclusion-oddtail-finite-local-algebra-zero-geometry-blindness}
记 \(\mathcal A_{\mathrm{fin}}\) 为由下列有限深度标量数据经有限次代数运算所生成的代数：
\[
\text{有限阶 odd tail jet},
\quad
\text{有限阶矩},
\quad
\text{有限 principal compression},
\quad
\text{有限阶 Pad\'e/Jacobi 截断},
\quad
\text{有限深度 scalar continued fraction}.
\]
则对每个 \(X\in\mathcal A_{\mathrm{fin}}\)，存在某个有限 \(M\) 使
\[
X\in \QQ\bigl(\varphi,\pi,\zeta(2),\dots,\zeta(2M)\bigr).
\]
因此，\(\mathcal A_{\mathrm{fin}}\) 中不存在任何真正敏感于非平凡零点几何的标量证书；特别地，任何 finite-depth 的局部 \(\pi\)-tail 校验都不可能单独承担 RH-sensitive completion 的判别任务。
\end{theorem}
\begin{proof}
正文已经把有限局部 \(\pi\)-tail 代数完全压缩到一个偶 \(\zeta\)-生成的有限代数层：任意有限个 odd tail 系数、有限矩、有限 Jacobi 截断、有限局部 Pad\'e 或 continued-fraction 数据，最终都落在某个
\[
K_M=\QQ\bigl(\varphi,\pi,\zeta(2),\dots,\zeta(2M)\bigr)
\]
之内。于是任意 \(X\in\mathcal A_{\mathrm{fin}}\) 都只依赖于有限多的 even-\(\zeta\) 数据。

但正文对 RH-sensitive completion 的判别又已给出 complete Bernstein / Stieltjes 等价，这一判别是关于完整 completion 对象的全局函数类性质，而非有限个 \(K_M\)-代数元素所能决定的局部条件。故任何属于 \(\mathcal A_{\mathrm{fin}}\) 的标量数据都必然对非平凡零点几何失明。证毕。
\end{proof}

\begin{theorem}[RH 接口的 genuinely nonlocal 阈值]\label{thm:conclusion-oddtail-rh-interface-genuinely-nonlocal}
设 \(\mathfrak H_{\mathrm{comp}}\) 为 projective compact host，\(\mathfrak H_{\mathrm{RH}}\) 为其 RH-sensitive completion 对象。若一条提升函子
\[
\mathcal L:\mathfrak H_{\mathrm{comp}}\to \mathfrak H_{\mathrm{RH}}
\]
在局部坐标中可由有限次多项式/有理运算、有限秩修正、有限深度 continued fraction、有限阶 Pad\'e 或有限 principal compression 实现，则 \(\mathcal L\) 不可能是 RH-sensitive 的。因而，任何真实的 RH 接口都必须是 genuinely nonlocal 的。
\leanverified{paper\_conclusion\_oddtail\_rh\_interface\_genuinely\_nonlocal}
\end{theorem}
\begin{proof}
若 \(\mathcal L\) 属于题中任一有限局部范式，则其输出必落在定理 \ref{thm:conclusion-oddtail-finite-local-algebra-zero-geometry-blindness} 所描述的有限局部代数 \(\mathcal A_{\mathrm{fin}}\) 之内，因此不可能感知非平凡零点几何。

然而正文已经把 centered completion 的 RH 判别压缩为 complete Bernstein / Stieltjes 性质；一旦 \(\mathcal L\) 真正输出该 RH-sensitive completion，它就必须对零点几何保持可见。两者矛盾。故任何 RH-sensitive 提升都不能属于这些有限局部范式，只能是 genuinely nonlocal 的。证毕。
\end{proof}

\begin{theorem}[principal compression 的三重最优性]\label{thm:conclusion-oddtail-principal-compression-triple-optimality}
对每个 \(n\ge 1\)，canonical principal compression \(J_\varphi^{[n]}\) 同时满足：
\begin{enumerate}
  \item 精确矩窗口
  \[
  \langle v_0,(J_\varphi^{[n]})^k v_0\rangle
  =
  \langle v_0,J_\varphi^k v_0\rangle
  \qquad (0\le k\le 2n-1);
  \]
  \item 维数最优性：任何匹配同样前 \(2n\) 个 odd layers 的 cyclic 自伴模型，其 Krylov 维数都至少为 \(n\)；
  \item 首失配常数的闭式：
  \[
  u_{2n}-\langle v_0,(J_\varphi^{[n]})^{2n}v_0\rangle
  =
  u_0(\alpha_1\cdots\alpha_n)^2,
  \]
  从而
  \[
  \mathcal E_\varphi(s)-\mathcal E_\varphi^{[n]}(s)
  =
  (4n)!u_0(\alpha_1\cdots\alpha_n)^2\,s^{4n+1}
  +O(s^{4n+3}).
  \]
\end{enumerate}
因此，principal compression 不仅知道“何时失真”，还同时锁定了“最早何阶失真”“该阶失真的精确常数”以及“低维替代的不可能性”。
\end{theorem}
\begin{proof}
正文已经分别证明：principal compression 对矩列的精确保留窗口恰为 \(0\le k\le 2n-1\)；任何 reproducing 同样矩数据的 cyclic 自伴模型都至少需要 \(n\) 维 Krylov 空间；以及首个失配项在 \(2n\) 阶恰等于边界返回权
\[
u_0(\alpha_1\cdots\alpha_n)^2.
\]
把这三条现成结论并读，便得到题中三重最优性。将首失配代回 odd tail 的 factorial lift 定义，立即得到
\[
\mathcal E_\varphi(s)-\mathcal E_\varphi^{[n]}(s)
=
(4n)!u_0(\alpha_1\cdots\alpha_n)^2\,s^{4n+1}
+O(s^{4n+3}).
\]
证毕。
\end{proof}

\begin{theorem}[finite-depth scalar continued fraction 的 Pad\'e--Jacobi 刚性坍缩]\label{thm:conclusion-oddtail-finite-depth-scalar-pade-jacobi-rigidity}
对每个 \(n\ge 1\)，canonical Stieltjes 变换 \(m_\varphi\) 在 \(z=0\) 处的唯一 \([n-1/n]\)-Pad\'e 逼近正是
\[
m_{\varphi,n}(z)
=
\langle v_0,(I-zJ_\varphi^{[n]})^{-1}v_0\rangle.
\]
因此，任何 finite-depth 的标量有理近似
\[
R_n(z)
\]
只要满足
\[
m_\varphi(z)-R_n(z)=O(z^{2n}),
\]
便必然有
\[
R_n(z)=m_{\varphi,n}(z).
\]
换言之，一旦要求 \(2n\) 阶局部精确性，所有 finite-depth scalar continued-fraction / Pad\'e / closed-form 自由度都会完全坍缩到唯一的 Jacobi 正规形。
\leanverified{paper\_conclusion\_oddtail\_finite\_depth\_scalar\_pade\_jacobi\_rigidity}
\end{theorem}
\begin{proof}
正文已经证明 \(m_{\varphi,n}\) 正是 \(m_\varphi\) 在 \(z=0\) 处唯一的 \([n-1/n]\)-Pad\'e 逼近；同时又证明 \(J_\varphi^{[n]}\) 在维数 \(n\) 下达到最优的 exact window。故任一标量有理函数 \(R_n\) 若也满足
\[
m_\varphi(z)-R_n(z)=O(z^{2n}),
\]
便与 \(m_{\varphi,n}\) 具有同样的 \([n-1/n]\)-Pad\'e 匹配条件。由 Pad\'e 唯一性，必有 \(R_n=m_{\varphi,n}\)。证毕。
\end{proof}

\begin{conjecture}[projective \(\pi\)-tail 的唯一 finite-depth 正规塔]\label{conj:conclusion-oddtail-unique-finite-depth-normal-tower}
设 \((R_n)_{n\ge 1}\) 为一列 finite-depth 标量近似，满足：
\begin{enumerate}
  \item \(R_n\) 与 \(m_\varphi\) 在 \(0\) 处匹配到 \(O(z^{2n})\)；
  \item \(R_n\) 的全部极点位于 canonical host 的自然禁带之外，且全部权重为正；
  \item \(R_n\) 的极点从外侧单调逼近 Wythoff 对偶极点塔。
\end{enumerate}
则必有
\[
R_n(z)=m_{\varphi,n}(z)
\qquad (\forall n\ge 1).
\]
亦即，principal compression 所生成的 Pad\'e--Jacobi 塔应是 projective \(\pi\)-tail 在 finite-depth 范畴中的唯一正规塔。
\end{conjecture}
