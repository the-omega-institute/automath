\subsection{固定分辨率 boundary Gödel--moment 全息、屏幕余秩审计与 screen--Artin 双障碍}\label{subsec:conclusion-fixedresolution-screen-artin-dual-obstruction}

在 fixed-resolution dyadic 全息、边界 Gödel 整数化、有限矩盒完备性、部分屏幕的相对同调核以及 Artin-visible hull 的最小忠实商都已经建立之后，还可把这些原本分散的接口重新压成一条更紧的结论链。其核心并不在于 bulk 不能被整数化或固定层不能被完全编码；恰恰相反，boundary Gödel code 与有限矩盒在固定分辨率上给出两套可计算双向等价的完备坐标。真正拒绝被统一有限压缩的，是两类彼此独立的缺失方向：一类来自 partial screen 留下的 relative-screen homology corank，另一类来自群商擦除的 non-inflation Artin determinant 通道。

\begin{theorem}[boundary Gödel code 与有限矩盒的双向完备等价]\label{thm:conclusion-fixedresolution-boundary-godel-momentbox-equivalence}
固定 \(m,n\ge 1\)，令 \(\mathcal U_{m,n}\) 为 \(I^n=[0,1]^n\) 上全部分辨率 \(m\) 的 dyadic polycube。对每个 \(U\in\mathcal U_{m,n}\)，记
\[
\Phi(U):=(G\circ \partial_n)(u_U),
\]
其中 \(u_U\in C_n(\mathcal Q_m;\FF_2)\) 为占据链，而 \(G\) 是像上的任意可计算可逆平方自由 Gödel 整数化。再定义有限矩盒
\[
\mathcal M_m(U):=
\bigl(\mu_\alpha(U)\bigr)_{\alpha\in\{0,\dots,2^m-1\}^n}.
\]
则对任意 \(U,V\in\mathcal U_{m,n}\)，成立严格等价
\[
\Phi(U)=\Phi(V)
\iff
\mathcal M_m(U)=\mathcal M_m(V).
\]
更强地，二者是可计算双向完备坐标，并满足
\[
K(U\mid \Phi(U),m,n)=O(1),
\qquad
K(U\mid \mathcal M_m(U),m,n)=O(1),
\]
\[
K(\Phi(U)\mid \mathcal M_m(U),m,n)=O(1),
\qquad
K(\mathcal M_m(U)\mid \Phi(U),m,n)=O(1).
\]
因此，在固定分辨率上，boundary Gödel code 与有限 tensor moment box 不是两种不同的全息，而是同一 bulk 对象的两套复杂度等价坐标。
\end{theorem}

\begin{proof}
由推论 \ref{cor:spg-boundary-godel-finite-moment-completeness}，\(\Phi(U)\) 唯一决定 \(U\)；由定理 \ref{thm:spg-dyadic-finite-moment-completeness}，\(\mathcal M_m(U)\) 也唯一决定 \(U\)。故
\[
\Phi(U)=\Phi(V)
\iff
U=V
\iff
\mathcal M_m(U)=\mathcal M_m(V).
\]

复杂度方面，第一式正是定理 \ref{thm:spg-stokes-godel-algorithmic-holographic-completeness} 的固定分辨率结论。对第二式，由定理 \ref{thm:conclusion-boundary-stokes-strict-linear-holography}，完整矩盒映射在固定 \((m,n)\) 下是一个可计算线性同构；结合 dyadic polycube 与其顶维 \(0/1\)-链之间的可计算双向编码，即得
\[
K(U\mid \mathcal M_m(U),m,n)=O(1).
\]
再由 \(U\mapsto \Phi(U)\) 与 \(U\mapsto \mathcal M_m(U)\) 的前向计算都是统一可计算过程，复合上述两条 \(O(1)\)-反演便得到
\[
K(\Phi(U)\mid \mathcal M_m(U),m,n)=O(1),
\qquad
K(\mathcal M_m(U)\mid \Phi(U),m,n)=O(1).
\]
证毕。
\end{proof}

\begin{theorem}[固定分辨率精确全息的完备性与指数不稳定性的分裂]\label{thm:conclusion-fixedresolution-exact-completeness-exponential-instability}
固定 \(m,n\ge 1\)。在定理 \ref{thm:conclusion-fixedresolution-boundary-godel-momentbox-equivalence} 的设定下，fixed-resolution bulk 既 admit 精确算术重构，也同时满足 sharp 的线性噪声阈值崩塌：
\begin{enumerate}
  \item 边界 Gödel 整数化存在非线性、算术型、精确译码器，可由边界整数唯一恢复原始 bulk；
  \item 若实线性算子
        \[
        L_{m,n}:C_{n-1}(\mathcal Q_m;\RR)\to C_n(\mathcal Q_m;\RR)
        \]
        满足
        \[
        L_{m,n}\partial_n=\mathrm{id},
        \]
        且对某个 \(\delta>0\) 与全部真值 \(x\in C_n(\mathcal Q_m;\RR)\)、全部噪声 \(\eta\in C_{n-1}(\mathcal Q_m;\RR)\) 都有
        \[
        \|\eta\|_2\le \varepsilon_{m,n}(\delta)
        \Longrightarrow
        \|L_{m,n}(\partial_nx+\eta)-x\|_2\le \delta,
        \]
        则必有 sharp 上界
        \[
        \varepsilon_{m,n}(\delta)\le \sqrt{2n}\,\delta\,2^{-m/2}.
        \]
\end{enumerate}
因此，单层 fixed-resolution holography 的真实分裂不是“是否可逆”，而是“算术上可精确反演”与“线性上不可能保持分辨率无关稳定性”。
\end{theorem}

\begin{proof}
第一条正是定理 \ref{thm:conclusion-godel-arithmetic-exact-vs-linear-stable-separation} 的第一部分。第二条则由推论 \ref{cor:spg-dyadic-holographic-reconstruction-noise-budget} 直接给出：任一实线性左逆若想在统一噪声球上把重构误差压到 \(\delta\)，其允许噪声半径至多为
\[
\sqrt{2n}\,\delta\,2^{-m/2}.
\]
故随着 \(m\to\infty\)，所有 fixed-\(\delta\) 的统一噪声预算都必须指数收缩。证毕。
\end{proof}

\begin{theorem}[inverse-system 完备与 single-layer 病态的范畴分裂]\label{thm:conclusion-projective-holography-crossresolution-vs-singlelayer}
设 \(A\subset I^n\) 为紧集，\(U_m(A)\) 为其 \(2^{-m}\)-dyadic 外逼近，并记 boundary Gödel 整数
\[
H_m(A):=H_m\!\bigl(U_m(A)\bigr).
\]
则同时成立：
\begin{enumerate}
  \item 逆系统
        \[
        \{H_m(A)\}_{m\ge 1}
        \]
        唯一决定 \(\overline A\)；
  \item 若
        \[
        N_m(A)=M\,2^{md}+o(2^{md}),
        \qquad
        M\in(0,\infty),\ d\in(0,n],
        \]
        并令
        \[
        G_m^\ast(A):=\prod_{k=1}^{N_m(A)}p_k,
        \]
        则有
        \[
        \lim_{m\to\infty}
        \frac{\log\log G_m^\ast(A)}{m\log 2}
        =d,
        \qquad
        M=
        \lim_{m\to\infty}
        \frac{
        \exp\!\bigl(\log\log G_m^\ast(A)-md\log 2\bigr)
        }{md\log 2}.
        \]
\end{enumerate}
换言之，projective holography 的完备性并不栖居于任何单个 fixed-resolution layer，而栖居于整个 inverse system：跨层序列恢复闭包、维数与含量，而每一固定层仍按定理 \ref{thm:conclusion-fixedresolution-exact-completeness-exponential-instability} 受到指数级稳定性崩塌的约束。
\end{theorem}

\begin{proof}
第一条由定理 \ref{thm:spg-boundary-godel-sequence-determines-closure} 直接给出。第二条分别由定理 \ref{thm:spg-stokes-godel-doublelog-dimension} 与定理 \ref{thm:spg-godel-doublelog-second-order-minkowski-content} 给出。最后，由定理 \ref{thm:conclusion-fixedresolution-exact-completeness-exponential-instability}，每个固定层 \(m\) 上的精确线性反演都仍需支付 \(2^{-m/2}\) 量级的噪声阈值收缩。故跨层完备与单层稳定分属不同范畴。证毕。
\end{proof}

\begin{theorem}[屏幕补全成本的相对同调余秩闭式与超模律]\label{thm:conclusion-fixedresolution-screen-corank-audit-cost-law}
固定 \(m,n\ge 1\)。对任意部分屏幕
\[
S\subset \overrightarrow{\mathcal F}^{\,n-1}_m,
\]
记部分边界观测
\[
f_S:=\Pi_S\circ\partial_n:C_n(\mathcal Q_m;\ZZ)\to \ZZ^S,
\]
并定义最小审计补全成本
\[
\operatorname{Cost}(S):=\min_{(R,r_S)}\cdim(R).
\]
则有 canonical 闭式
\[
\operatorname{Cost}(S)
=
\rank_\ZZ \ker f_S
=
\rank_\ZZ H_n(\mathcal Q_m,A_{\neg S};\ZZ)
=
c(\Gamma_S)-1.
\]
此外，\(\operatorname{Cost}\) 还满足：
\[
S\subseteq T
\Longrightarrow
\operatorname{Cost}(T)\le \operatorname{Cost}(S),
\]
\[
\operatorname{Cost}(S)+\operatorname{Cost}(T)
\le
\operatorname{Cost}(S\cup T)+\operatorname{Cost}(S\cap T),
\]
并且对任意单面增广 \(\tilde f\notin S\)，有 unit-step law
\[
\operatorname{Cost}(S)-1
\le
\operatorname{Cost}(S\cup\{\tilde f\})
\le
\operatorname{Cost}(S).
\]
因此，屏幕补全成本并不是启发式外置账本，而正是 relative-screen homology 的自由余秩；其边际削减律由 graphic matroid 的闭包结构完全控制。
\end{theorem}

\begin{proof}
由定理 \ref{thm:spg-partial-boundary-screen-kernel-relative-homology} 与定理 \ref{thm:spg-screen-kernel-connected-components}，
\[
\ker f_S
\cong
H_n(\mathcal Q_m,A_{\neg S};\ZZ)
\cong
\ZZ^{\,c(\Gamma_S)-1}.
\]
再由推论 \ref{cor:spg-screen-injectivity-and-audit-cost-components}，\(\operatorname{Cost}(S)\) 恰等于该自由秩，故得到闭式
\[
\operatorname{Cost}(S)=c(\Gamma_S)-1.
\]

单调性与超模性由定理 \ref{thm:conclusion-screen-graphic-matroid-corank-supermodularity} 直接给出。最后，对单面增广 \(\tilde f\)，定理 \ref{thm:conclusion-screen-rank-gain-closure-diminishing-returns} 表明其边际收益只能取 \(0\) 或 \(1\)，故
\[
\operatorname{Cost}(S\cup\{\tilde f\})
\in
\{\operatorname{Cost}(S),\ \operatorname{Cost}(S)-1\},
\]
等价地得到所示 unit-step law。证毕。
\end{proof}

\begin{corollary}[轴向屏幕余秩的 sharp 面积律]\label{cor:conclusion-fixedresolution-axial-screen-corank-area-law}
令 \(S^{(1)}\) 为只保留与 \(x_1\)-轴正交的内部 \((n-1)\)-面且不含外边界面的轴向屏幕。则

因此，在 codimension-one 可见性而外边界全失的制度下，最小审计补全成本不是多项式量级，而是精确的面积指数律。
\end{corollary}

\begin{proof}

\begin{theorem}[fixed-resolution 边界码族的渐近劣性]\label{thm:conclusion-fixedresolution-boundary-code-asymptotic-badness}
令
\[
\mathcal C_{m,n}
:=
\operatorname{im}\bigl(
\partial_n:C_n(\mathcal Q_m;\FF_2)\to C_{n-1}(\mathcal Q_m;\FF_2)
\bigr).
\]
则 \(\mathcal C_{m,n}\) 构成长度
\[
N_{m,n}=n(2^m+1)\,2^{m(n-1)}
\]
的线性码，满足
\[
\dim \mathcal C_{m,n}=2^{mn},
\qquad
R_{m,n}:=\frac{\dim \mathcal C_{m,n}}{N_{m,n}}
\longrightarrow
\frac1n,
\]
并且最小距离具有精确公式
\[
d_{\min}(\mathcal C_{m,n})=2n.
\]
因此其相对距离
\[
\delta_{m,n}
:=
\frac{d_{\min}(\mathcal C_{m,n})}{N_{m,n}}
=
\frac{2n}{n(2^m+1)\,2^{m(n-1)}}
\longrightarrow
0.
\]
于是，尽管 \(\mathcal C_{m,n}\) 在 fixed-resolution holography 的意义下是信息完备阴影，它在编码论意义下却是渐近劣码。换言之，完备性与稳健纠错并不共居于同一边界码族。
\end{theorem}

\begin{proof}
由定理 \ref{thm:spg-dyadic-boundary-image-ldpc}，\(\mathcal C_{m,n}\) 的长度、维数与码率极限如上；由推论 \ref{cor:spg-dyadic-boundary-image-min-distance}，
\[
d_{\min}(\mathcal C_{m,n})=2n.
\]
两式相除即得
\[
\delta_{m,n}
=
\frac{2n}{n(2^m+1)\,2^{m(n-1)}}
\to 0.
\]
这正是渐近劣码的定义。证毕。
\end{proof}

\begin{theorem}[partial screen 与 Artin quotient 的双重不可压缩核]\label{thm:conclusion-fixedresolution-screen-artin-dual-obstruction}
设 \(S\) 为部分屏幕，\(\pi:G\twoheadrightarrow H\) 为有限群满同态，并考虑压缩证书
\[
\mathcal C_{S,\pi}(x,\mathcal K)
:=
\bigl(f_S(x),\ \pi_\ast\mathcal K\bigr),
\]
其中
\[
x\in C_n(\mathcal Q_m;\ZZ),
\qquad
\mathcal K\in \CC[G]^{V\times V}.
\]
并按 Peter--Weyl 分块记 \(\mathcal K\) 的 \(\rho\)-通道为 \(\mathcal K_\rho\)。
若
\[
\operatorname{Cost}(S)>0,
\]
且存在某个
\[
\rho\in \widehat G\setminus \mathrm{Inf}_\pi(\widehat H)
\]
满足
\[
\det(I-z\mathcal K_\rho)\not\equiv 1,
\]
则任何仅经由 \(\mathcal C_{S,\pi}\) 的证书都不可能同时成为 bulk-faithful 与 determinant-faithful compression。更精确地，它至少遗漏两类彼此独立的方向：
\begin{enumerate}
  \item 一个自由阿贝尔的 topological hidden sector
        \[
        \ker f_S\cong \ZZ^{\,\operatorname{Cost}(S)};
        \]
  \item 一个由全部 determinant-活跃的 non-inflation 通道所承载的 Artin hidden sector。
\end{enumerate}
因此，任何欲同时保留 bulk 回放与完整 Artin determinant law 的 completion-faithful 证书，都至少必须外置：
\[
\text{一个秩为 }\operatorname{Cost}(S)\text{ 的自由阿贝尔补全寄存器},
\]
以及
\[
\text{全部 determinant-活跃的 non-inflation Artin 通道，或与之等价的 visible hull}.
\]
\end{theorem}

\begin{proof}
由定理 \ref{thm:conclusion-fixedresolution-screen-corank-audit-cost-law}，
\[
\operatorname{Cost}(S)>0
\Longrightarrow
\rank_\ZZ\ker f_S>0,
\]
故存在非零
\[
k\in\ker f_S
\]
使
\[
f_S(x+k)=f_S(x).
\]
这说明若不额外记录一个秩为 \(\operatorname{Cost}(S)\) 的自由阿贝尔补全寄存器，部分屏幕数据本身不可能 faithful 地恢复 bulk。

另一方面，由定理 \ref{thm:conclusion-artin-determinant-minimal-visible-quotient}，群商 \(\pi\) 保持完整 determinant 当且仅当全部 determinant-活跃通道都 inflation 到 \(H\)。现在存在
\[
\rho\in \widehat G\setminus \mathrm{Inf}_\pi(\widehat H)
\]
且
\[
\det(I-z\mathcal K_\rho)\not\equiv 1,
\]
故 \(\pi_\ast\mathcal K\) 不可能保持 \(\mathcal K\) 的完整 Artin determinant law。于是，即便 bulk 侧另行补全，商后的标签证书仍会丢失全部 determinant-活跃的 non-inflation 通道。

因此，\(\mathcal C_{S,\pi}\) 的缺失并非单轴的，而是分裂为一个 topological free-abelian kernel 与一个 non-Abelian Artin kernel。两者分别来自 screen corank 与 quotient erasure，必须分别补足。证毕。
\end{proof}

\begin{conjecture}[completion-faithful \texorpdfstring{\(\pi\)}{pi}-证书的 screen--Artin 三歧]\label{conj:conclusion-fixedresolution-screen-artin-trilemma}
设 \(\{\mathfrak C_m\}_{m\ge 1}\) 为一族试图在本文语义中忠实承载 completion-side \(\pi\)-证书的对象，并假定每一级都因子化于
\[
\bigl(f_{S_m},\ \pi_{m\ast}\mathcal K_m\bigr).
\]
则以下三项不应能够同时成立：
\begin{enumerate}
  \item
        \[
        \sup_m \operatorname{Cost}(S_m)<\infty;
        \]
  \item \(\pi_m\) 最终退化为阿贝尔化、或最终退化为固定有限商压缩；
  \item \(\mathfrak C_m\) 仍保持完整的 completion fidelity、determinant law 与 primitive-orbit fidelity。
\end{enumerate}
换言之，任何最终只保留有限 screen 与阿贝尔化标签的 classical scalar \(\pi\)-formula，至多是值层算法，而不可能是 completion-faithful 证书。前述诸定理所给出的证据链正是：fixed-resolution bulk 本身并不抗拒被整数化，真正不可有限化的是 screen homology corank 与 non-Abelian Artin loss 的同时存在。
\end{conjecture}

\noindent
综上，fixed-resolution holography 的核心障碍并不在于“边界不足以编码体”。在单层上，boundary Gödel code 与有限矩盒已经给出复杂度等价的精确全息；在跨层上，inverse system 甚至恢复闭包、维数与含量。真正抗拒统一有限压缩的，是 partial screen 留下的 free-abelian relative homology corank，以及群商擦除的 determinant-活跃 non-inflation Artin 扇区。因而，对本文对象更合适的终端障碍语言，不应再只是“有限证书不足”，而应是一个同时记录 screen completion cost 与 non-Abelian channel loss 的 screen--Artin obstruction theory。
