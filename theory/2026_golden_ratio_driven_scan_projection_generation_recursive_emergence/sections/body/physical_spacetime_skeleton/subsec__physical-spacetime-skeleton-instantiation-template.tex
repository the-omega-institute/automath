\subsection{数学实例化模板：哪些对象代入后才可能产生“物理型时空”}\label{subsec:physical-spacetime-skeleton-instantiation-template}

沿 \S\ref{subsec:cdim-cech-projective-phase-interference} 与 \S\ref{subsec:cdim-sync-causal-holonomy-spacetime} 的口径，量子结构已被压回到局部--全局粘接失配在统计表示层上的后果，而时空结构则被压回到同步、传播与闭路 residual 在几何表示层上的组织。
这条导出链本身并不选定唯一的后端对象；若在此处直接把某种具体表示写成起始设定，便会把表示选择误写成新的公理。
因此，本小节不再推进新的物理主张，而只回答一个技术问题：什么样的数学数据，才足以承载上述骨架；以及，一旦这些数据已经到位，哪些现有模块可以充当其离散后台、量子后台与时空后台。

\subsubsection{可接受实例化的数据接口}

\begin{definition}[物理型实例化]\label{def:physical-spacetime-acceptable-instantiation}
称一组数据
\[
\mathfrak I=
\bigl(
\mathcal X,\mathcal U,\mathrm{Read},\mathsf{NULL},
\mathcal G,\mathcal O,\mathbb E,
\mathcal R,\tau,d,
A,\Omega,\{\mathcal U_h\}_{h\downarrow 0}
\bigr)
\]
为一个\textbf{物理型实例化}，若其中：
\begin{enumerate}
  \item \((\mathcal X,\mathcal U)\) 给出局部状态与覆盖体系；
  \item \(\mathrm{Read}\) 与 \(\mathsf{NULL}\) 给出局部可读性与失败见证；
  \item \(\mathcal G\) 给出局部重叠上的过渡数据、粘接差或等价的规范残差；
  \item \((\mathcal O,\mathbb E)\) 给出可观测量代数与条件期望，使局部读出能够推送到全局统计；
  \item \((\mathcal R,\tau,d)\) 给出资源账本、同步延迟与空间代价，使局部比较能够几何化；
  \item \(A\) 与 \(\Omega\) 给出离散联络及其曲率证书；
  \item \(\{\mathcal U_h\}_{h\downarrow 0}\) 给出细化族，使离散对象能够谈论连续极限。
\end{enumerate}
若这些数据还满足下列相容性条件：
\begin{enumerate}
  \item 局部读出在重叠上可比较；
  \item 全局化失败有可审计的 \v{C}ech 型障碍证书；
  \item 条件期望与 coarse-graining 相容；
  \item \(\tau\) 与 \(d\) 对允许链次可加；
  \item \(\Omega\) 满足离散 Bianchi 型关系；
  \item 细化极限与前述结构兼容，
\end{enumerate}
则称 \(\mathfrak I\) 为一个\textbf{可接受实例化}。
\end{definition}

\begin{proposition}[实例化准则]\label{prop:physical-spacetime-instantiation-criterion}
任一可接受实例化 \(\mathfrak I\) 都足以承载本文前述骨架构造。
特别地，它同时给出：
\begin{enumerate}
  \item 局部可读而全局受阻的 \v{C}ech 障碍；
  \item 可见态的射影类与干涉项候选；
  \item 由同步延迟与空间代价定义的传播上界 \(c_*\)；
  \item 由离散曲率在细化极限中导出的连续曲率候选。
\end{enumerate}
\leanverified{paper\_physical\_spacetime\_instantiation\_criterion}
\end{proposition}
\begin{proof}
\S\ref{subsec:physical-spacetime-skeleton-local-global-obstruction} 与 \S\ref{subsec:cdim-cech-projective-phase-interference} 只要求局部读出、重叠比较、可见投影与障碍证书；
\S\ref{subsec:cdim-sync-causal-holonomy-spacetime} 进一步要求同步延迟、空间代价、离散联络与细化族。
定义 \ref{def:physical-spacetime-acceptable-instantiation} 的数据恰好覆盖了这些输入。
因此，前述构造均可在 \(\mathfrak I\) 上逐项实现。
\end{proof}

上述命题的作用不是再给出一类新对象，而是明确：真正被固定的是接口，而不是具体表示。
只要接口被满足，后端可以变化；相反，若后端试图越过接口自行引入“额外物理内容”，它就不再是实例化，而是另起公理。

\subsubsection{离散后台：局部截面、\texorpdfstring{$\mathsf{NULL}$}{NULL} 与离散联络}
最基础的后端是离散后台。
它承担三件不可替代的职责：给出局部读出，记录全局化失败，并把复合失配组织为曲率证书。
只要这三件事完成，\S\ref{subsec:physical-spacetime-skeleton-local-global-obstruction} 与 \S\ref{subsec:cdim-sync-causal-holonomy-spacetime} 中的相位残差、holonomy 与曲率候选就已经在离散层面全部出现。

从结构上看，一个离散后台至少应当提供：
\begin{enumerate}
  \item 局部对象及其重叠关系；
  \item 局部 lift 或局部截面；
  \item 粘接失败的证书，尤其是 \v{C}ech 型二阶障碍；
  \item 定义在基本 \(2\)-胞腔或最小闭路上的复合缺陷。
\end{enumerate}

在这一层上，\(\mathsf{NULL}\) 也应被完全重写。
它不再只是“没有输出”的技术标志，而是“局部可读对象在给定覆盖下无法被全局化”的可审计见证。
于是，\(\mathsf{NULL}\)、\v{C}ech 障碍与异常证书不再是彼此独立的失败语言，而变成同一局部--全局失配机制在不同分辨率下的三种表述。

离散联络在这里同样不需要被预设为连续纤维丛上的光滑联络。
只要存在沿重叠边的平行移动数据 \(A\)，以及定义在 \(2\)-胞腔上的复合残差
\[
\Omega:=-\delta A,
\]
就已经足以谈论离散 holonomy、离散 Bianchi 关系以及“曲率是闭路复合的首个不可消去项”这一核心思想。
连续几何不是离散曲率的前提，而只是它在细化极限中的可能表示。

\subsubsection{量子后台：扭曲谱、核化与射影表示}
量子后台的职责与离散后台不同。
它不需要重新提供局部--全局障碍，而需要把该障碍在可见统计层上的后果实现出来。
换言之，它的任务是把相对过渡类、射影态类与干涉型修正，落实为某种可计算、可比较、可正化的表示。

一个足够的量子后台，至少应满足三条原则：
\begin{enumerate}
  \item 它能够把相对过渡类编码为可见相位、谱相位或等价的扭曲参数；
  \item 它能够给出一个对 coarse-graining 稳定的正核或半正核；
  \item 它允许把可见统计写成该后端表示上的正二次型或等价对象。
\end{enumerate}

满足这些条件的后端可以有多种形式：扭曲行列式、单位根谱模态、有限秩核化、再现核空间，乃至更抽象的射影表示空间，都属于同一接口族。
因此，与其先验地说“量子态就是 Hilbert 空间向量”，不如说：
\[
\text{量子后台必须把局部相对过渡类的可见投影，组织成一个足以承载射影态类和干涉项的正表示。}
\]
本文的 \S\ref{subsec:cdim-hilbert-quantum-closure} 已经说明：在可读时间字、mixed-collision 正定核与附录算子代数接口齐备时，这一后验闭包确实导出 Hilbert 载体、Born 规则、量子信道以及连续时间分支。
也正因此，Hilbert 空间在本文中仍然不是起始对象，而是接口满足后的表示结果。

\begin{proposition}[核化充分模板]\label{prop:physical-spacetime-kernelization-template}
设一个量子后台为每个可见态类 \([\ell]\) 赋予一个核表示
\[
K([\ell_1],[\ell_2]),
\]
并满足：
\begin{enumerate}
  \item \(K\) 对局部规范选择不敏感；
  \item \(K\) 对 coarse-graining 稳定；
  \item \(K\) 在张量复合下保持正性与封闭性；
  \item 交叉项仅由相对过渡类决定。
\end{enumerate}
则 \S\ref{subsec:cdim-cech-projective-phase-interference} 中的干涉项与 \S\ref{subsec:cdim-hilbert-quantum-closure} 中的 Born 型读出，都可被重写为该核表示上的正二次型一致性问题。
\leanverified{paper\_physical\_spacetime\_kernelization\_template}
\leanverified{paper\_physical\_spacetime\_kernelization\_template\_seeds}
\end{proposition}
\begin{proof}
在核表示下，自交项给出各态类的可见权重，交叉项由非对角核值提供。
局部规范不变性与 coarse-graining 稳定性保证这些量只依赖于态类而不依赖于代表元。
因此，干涉修正与 Hilbert 读出可统一降为：该正核如何决定可见统计与其后端表示的一致性问题。
\end{proof}

\subsubsection{时空后台：资源几何、传播锥与尺度极限}
时空后台的职责也与量子后台不同。
它要做的不是提供统计读法，而是把同步延迟、空间代价与闭路残差在尺度上组织成时间、空间、因果与曲率的表示。

一个足够的时空后台至少需要三类对象。
第一类，是几何化的比较量，例如同步延迟 \(\tau\)、空间代价 \(d\)、共同支撑或共同 forcing 的比较结构；这些量给出 \S\ref{subsec:cdim-sync-causal-holonomy-spacetime} 中的时间、空间与因果前序。
第二类，是细化尺度上的离散联络与离散曲率 \((A_h,\Omega_h)\)；它们给出传播锥边界与曲率候选的离散来源。
第三类，是控制连续极限的稳定性条件，例如统一有界性、弱紧性、次可加性以及观察者变换下的协变控制；没有这些条件，时空只能停留在离散草图，而不能过渡到连续骨架。

应当强调的是，时空后台不必从一开始就表现为光滑流形。
资源几何、在线延迟、时间--空间交换、prime-register 或 solenoid 型全局因子，都可以被看作这一后台中的不同层。
它们各自负责不同职责：局部比较层负责因果结构，资源层负责传播上界，全局紧化层负责 holonomy 的拓扑残余，而细化极限才负责把离散曲率压成连续曲率候选。

\begin{proposition}[连续极限的充分模板]\label{prop:physical-spacetime-continuum-limit-template}
设
\[
\{(\tau_h,d_h,A_h,\Omega_h)\}_{h\downarrow 0}
\]
是一族时空后台数据，并满足：
\begin{enumerate}
  \item \(\tau_h\) 与 \(d_h\) 对允许链次可加，且在紧集上统一受控；
  \item 面积归一化后的离散曲率场 \(\Omega_h/|\sigma_h|\) 在基本 \(2\)-胞腔上统一有界或弱紧；
  \item 观察者变换在每个尺度上保持这些量的等价类；
  \item 细化映射与局部--全局障碍相容。
\end{enumerate}
则存在子列使传播锥候选与归一化离散曲率候选在弱意义下收敛到连续极限对象。
\leanverified{paper\_physical\_spacetime\_continuum\_limit\_template}
\end{proposition}

\begin{proposition}[connected grain quotient and slab--sector holonomy exclusion]\label{distill:wang_zahl-grain_prism_structure_dichotomy-physical-connected-grain-quotient}
令 \(N\) 为可接受实例化中某有限局部覆盖的 nerve，令 \(M\) 为有限阿贝尔系数群。设
\[
q:N\longrightarrow Q
\]
是一个 fold-aware grain quotient map，满足：\(q\) 在顶点与 \(1\)-单形上满射，且每个顶点纤维 \(q^{-1}(v)\) 的诱导 \(1\)-骨架连通。对 \(\bar g\in Z^1(Q;M)\)，记 \(g:=q^\ast\bar g\in Z^1(N;M)\)，其中落在同一 quotient 顶点内的边被拉回为零。

则
\[
[g]=0\in H^1(N;M)\qquad\Longleftrightarrow\qquad [\bar g]=0\in H^1(Q;M).
\]

进一步，令 \(E_{\rm ss}\subset Q_1\) 为 quotient 上由 slab 或 sector audit 标出的有向边集，并定义
\[
Z^1_{E_{\rm ss}}(Q;M)
:=
\{c\in Z^1(Q;M):c(e)=0\text{ for every oriented edge }e\notin E_{\rm ss}\}.
\]
若该 slab--sector quotient audit 满足有限线性条件
\[
Z^1_{E_{\rm ss}}(Q;M)\subset B^1(Q;M),
\]
则任何 grain-factorable \v Cech residual \(g=q^\ast\bar g\) whose quotient support is contained in \(E_{\rm ss}\) has trivial global holonomy class:
\[
[g]=0\in H^1(N;M).
\]
因此，在该接口下，一个非平凡局部可解但全局不可粘接的 residual 不能被隐藏在 connected grains 内，也不能完全支撑在已通过 slab--sector relative audit 的 quotient edges 上；它必须在 quotient 中占据至少一条未被该 audit 消去的 transverse holonomy edge。
\end{proposition}

\begin{proposition}[原始语法限定下的曲率证书二通道正规形]\label{distill:euclid_elements-finite_construction_certificate_families-curvature-certificate-two-channel-normal-form}
令 \(N\) 为定义 \ref{def:physical-spacetime-acceptable-instantiation} 中某个有限局部覆盖的 nerve，并令 \(M\) 为有限阿贝尔系数群。固定一个 primitive-admissibility 语法，规定尺度 \(N\) 上的曲率证书只能通过下列两类有限链产生：
\begin{enumerate}
  \item \textbf{生成通道：} 先构造一个离散联络 \(A\in C^1(N;M)\)，再定义
  \[
  \Omega_A:=-\delta A\in C^2(N;M);
  \]
  \item \textbf{供给通道：} 直接引用可接受实例化中已经作为接口数据供给的有限证书
  \[
  \Omega\in Z^2(N;M),
  \]
  并同时核验其 Bianchi 型闭合条件与细化相容记录。
\end{enumerate}
则：
\begin{enumerate}
  \item 生成通道产生的每个曲率证书都满足
  \[
  [\Omega_A]=0\in H^2(N;M);
  \]
  \item 若供给通道给出 \(\Omega\) 且 \([\Omega]\ne0\in H^2(N;M)\)，则 \(\Omega\) 不能由当前生成通道中的任何 \(1\)-cochain 联络产生；
  \item 若某个证书链没有使用供给通道，却声称构造出 \([\Omega]\ne0\) 的曲率类，则该链在当前 primitive 语法下必须被拒绝或返回 \(\mathsf{NULL}\)。
\end{enumerate}
本命题不声称可接受实例化中的全部 \(\Omega\) 都必须 exact；它只说明：在把曲率 primitive 限定为 \(1\)-cochain 联络生成时，非零全局曲率类只能作为已供给且已审计的接口数据进入，而不能被图示闭路复合免费制造。
\end{proposition}

\begin{proposition}[物理型实例化中的 reduced edge-holonomy 树门控]
\label{distill:euclid_elements-diagrammatic_compatibility_audit_families-physical-reduced-holonomy-tree-gate}
令 \(N\) 为定义 \ref{def:physical-spacetime-acceptable-instantiation} 中某有限局部覆盖 nerve 的连通一骨架。设 \(G\) 为边过渡残差所在的群，并给定一组有向边数据
\[
g_{ij}\in G,\qquad g_{ji}=g_{ij}^{-1}.
\]
固定一个群同态
\[
\kappa:G\longrightarrow \overline G
\]
并定义 reduced edge system
\[
\bar g_{ij}:=\kappa(g_{ij})\in\overline G.
\]
取 \(N\) 的一棵生成树 \(T\) 与根顶点 \(o\)。对每条不属于 \(T\) 的有向边 \(e=(u,v)\)，令 \(\ell_e\) 为闭路：先沿 \(T\) 从 \(o\) 到 \(u\)，再走边 \(e\)，最后沿 \(T\) 从 \(v\) 回到 \(o\)。若所有基本闭路满足
\[
H_{\bar g}(\ell_e):=\prod_{ab\in\ell_e}\bar g_{ab}=1_{\overline G},
\qquad e\notin T,
\]
则存在顶点规范 \(a_x\in\overline G\) 使得
\[
\bar g_{xy}=a_x^{-1}a_y
\qquad\text{for every oriented edge }(x,y)\in N_1^{\rm or}.
\]
因此 reduced 通道中的每条闭路 holonomy 均为平凡，且该有限图解的 reduced 局部过渡数据可被同时平坦化。反之，若存在这样的顶点规范，则所有基本闭路的 reduced holonomy 均为平凡。非连通 nerve 的情形逐连通分支适用同一判据。
\end{proposition}

\begin{proposition}[柱面与环面闭合的有限 holonomy 生成元门控]\label{distill:cech_cohomology-holonomy_residue_classification-physical-cylinder-torus-generator-gate}
设 \(\mathfrak I\) 为定义 \ref{def:physical-spacetime-acceptable-instantiation} 意义下的可接受实例化，并令 \(K\) 为其中某个有限事件--区域覆盖经过协议 quotient 后得到的连通 nerve 复形。令 \(M\) 为有限阿贝尔 reduced 系数群，并设
\[
h\in Z^1(K;M)
\]
为 reduced overlap 残差；也就是说，对每条取向边 \(e\) 有 \(h(\bar e)=-h(e)\)，并且在每个二维面上满足 \(\delta h=0\)。对任意整数 \(1\)-循环

定义闭路 holonomy
\[
\operatorname{Hol}_h(z):=\sum_e n_e h(e)\in M.
\]
假设一组已登记的柱面或环面闭路

\Gamma=\{\gamma_a:a\in A\}\subset Z_1(K;\ZZ)

的同调类生成 \(H_1(K;\ZZ)\)。则下列条件等价：

\end{proposition}

\begin{proposition}[物理型实例化中离散曲率账本的基变函子性与闭合残差]\label{distill:grothendieck-relative_certificate_base_change_stability-physical-curvature-ledger-functoriality}
令 \(\mathfrak I_b\) 与 \(\mathfrak I_{b'}\) 为定义 \ref{def:physical-spacetime-acceptable-instantiation} 意义下的两份物理型实例化，并设允许基变 \(\nu:b'\to b\) 在某有限覆盖 nerve 上给出单纯映射
\[
\rho_\nu:N_{b'}\longrightarrow N_b
\]
以及有限阿贝尔系数群同态 \(\phi_\nu:M_b\to M_{b'}\)。对 \(c\in C^k(N_b;M_b)\) 定义
\[
(\nu^\flat c)(\sigma'):=\phi_\nu(c(\rho_\nu(\sigma'))),
\qquad \sigma'\in N_{b',k}.
\]
则
\[
\delta(\nu^\flat c)=\nu^\flat(\delta c).
\]
因此：
\begin{enumerate}
  \item 若曲率证书由生成通道给出，即 \(\Omega_b=-\delta A_b\)，则
  \[
  \nu^\flat\Omega_b=-\delta(\nu^\flat A_b),
  \]
  故其上同调类仍为零；
  \item 若曲率证书由供给通道给出，即 \(\Omega_b\in Z^2(N_b;M_b)\)，则 \(\nu^\flat\Omega_b\in Z^2(N_{b'};M_{b'})\)，且其类为拉回类；
  \item 若拉回后的离散 holonomy 或曲率类非零，并且没有登记边界 strata、顶点势或尺度相容 untwisting potential 杀死该类，则 \(\mathfrak I_{b'}\) 只给出带闭合残差的离散后台，不能作为命题 \ref{prop:physical-spacetime-continuum-limit-template} 中闭合连续曲率候选的输入。
\end{enumerate}

\begin{proof}

\subsubsection{本节结论}
本小节并未引入任何新的物理对象；它只把“什么样的后端足以承载前述抽象骨架”说明清楚。
离散后台负责局部可读、全局受阻与曲率证书；量子后台负责射影态类、相位与干涉的正表示；时空后台负责同步、传播、因果与连续极限。
三者并不彼此独立，而是同一抽象接口在不同表示层上的分工。

因此，本小节真正固定下来的不是某一种特定后端，而是实例化接口本身。
后文任何更具体的谱表示、核表示或几何极限表示，只能被当作这组接口的后端实现，而不能再回填为新的起始公理。
